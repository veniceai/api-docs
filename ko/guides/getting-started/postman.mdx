## 개요

Venice는 개발자가 API의 전체 기능을 탐색하고 테스트할 수 있도록 포괄적인 Postman 컬렉션을 제공합니다. 이 컬렉션에는 사전 구성된 요청, 예제 및 환경 변수가 포함되어 있어 Venice의 AI 서비스를 빠르게 시작할 수 있습니다.

## 컬렉션 접근

공식 Postman 컬렉션은 Venice AI 워크스페이스에서 사용할 수 있습니다.

* [Venice AI Postman 워크스페이스](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman 예제](https://postman.venice.ai/)

## 컬렉션 기능

- **즉시 사용 가능한 요청**: 모든 Venice 엔드포인트에 대한 사전 구성된 API 호출
- **환경 템플릿**: 올바르게 구조화된 환경 변수
- **요청 예제**: 각 엔드포인트에 대한 실제 사용 사례 예제
- **응답 샘플**: API 출력을 이해하는 데 도움이 되는 예제 응답
- **문서**: 각 요청에 대한 인라인 문서

## 시작하기

<Steps>
   <Step title="컬렉션 포크">
      - Venice AI 워크스페이스로 이동합니다
      - "Fork"를 클릭하여 컬렉션의 사본을 만듭니다
      - 워크스페이스 대상을 선택합니다
   </Step>

   <Step title="환경 설정">
      - Postman에서 새 환경을 생성합니다
      - Venice API 키를 추가합니다
      - 베이스 URL을 구성합니다: `https://api.venice.ai/api/v1`
   </Step>

   <Step title="첫 번째 요청 보내기">
      - 컬렉션에서 요청을 선택합니다
      - 환경이 선택되어 있는지 확인합니다
      - "Send"를 클릭하여 API를 테스트합니다
   </Step>
</Steps>

## 사용 가능한 엔드포인트

컬렉션에는 모든 Venice API 엔드포인트에 대한 예제가 포함되어 있습니다.

- 텍스트 생성
- 이미지 생성
- 모델 정보
- 이미지 업스케일링
- 시스템 프롬프트 구성

## 모범 사례

- API 키를 안전하게 보관하고 절대 공유하지 마세요
- 민감한 정보에는 환경 변수를 사용하세요
- 구현 전에 Postman 콘솔에서 응답을 테스트하세요
- 예상되는 데이터 구조에 대해 예제 응답을 검토하세요

<Note>*참고: Postman 컬렉션은 최신 API 변경 사항과 기능을 반영하기 위해 정기적으로 업데이트됩니다.*</Note>
