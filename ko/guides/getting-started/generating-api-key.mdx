Venice의 API는 API 키로 보호됩니다. Venice API 사용을 시작하려면 먼저 새 키를 생성해야 합니다. 시작하려면 다음 단계를 따르세요.

<Steps>
  <Step title="API 설정 페이지 방문">
    API 설정 페이지로 이동하려면 https://venice.ai/settings/api 를 방문하세요. 이 페이지는 왼쪽 도구 모음에서 "API"를 클릭하거나 사용자 설정 내에서 "API"를 클릭하여 액세스할 수 있습니다.

    이 대시보드에서 Diem 및 USD 잔액, API 등급, API 사용량 및 API 키를 확인할 수 있습니다.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API Overview" />
    </Frame>
  </Step>

  <Step title="새 API 키 생성 클릭">
    대시보드를 아래로 스크롤하고 "Generate New API Key"를 선택하세요. 옵션 목록이 표시됩니다.

    * **Description:** API 키의 이름을 지정하는 데 사용됩니다

    * **API Key Type:**

      * "Admin" 키는 프로그래밍 방식으로 추가 API 키를 삭제하거나 생성할 수 있는 기능을 가집니다.

      * "Inference Only" 키는 추론 실행만 허용됩니다.

    * **Expires at:** API 키에 만료 날짜를 설정하여 그 이후에는 작동하지 않도록 선택할 수 있습니다. 기본적으로 날짜가 설정되지 않으며 키는 영구적으로 작동합니다.

    * **Epoch Consumption Limits:** 이를 통해 개별 API 키에서 API 사용에 대한 제한을 만들 수 있습니다. 주어진 epoch(24시간) 내에 허용되는 Diem 또는 USD 금액을 제한하도록 선택할 수 있습니다.

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Generate New API Key" />
    </Frame>
  </Step>

  <Step title="키 생성">
    Generate를 클릭하면 API 키가 표시됩니다.

    <Warning>
      **중요:** 이 키는 한 번만 표시됩니다. 반드시 복사하여 안전한 곳에 보관하세요. 분실한 경우, 삭제하고 새 키를 생성해야 합니다.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Your API Key" />
    </Frame>
  </Step>
</Steps>
