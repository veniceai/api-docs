[OpenClaw](https://openclaw.ai)는 메시징 플랫폼(WhatsApp, Telegram, Discord, iMessage, Slack)을 AI 모델에 연결하는 오픈 소스, 자체 호스팅 AI 게이트웨이입니다. Venice AI는 내장 제공자로 사용할 수 있어, 연결된 모든 채널에서 비공개 및 검열되지 않은 모델에 액세스할 수 있습니다.

## 설정

### 1. OpenClaw 설치

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. 온보딩 마법사 실행

```bash
openclaw onboard
```

마법사가 설정을 안내합니다. 메시지가 표시되면 목록에서 제공자로 **Venice AI**를 선택한 다음 API 키를 붙여넣으세요. 아직 키가 없다면 [venice.ai/settings/api](https://venice.ai/settings/api)에서 받으세요.

### 3. 모델 선택

온보딩 중에 OpenClaw는 사용 가능한 모든 Venice 모델을 표시합니다. 일부 권장 사항:

| 사용 사례 | 모델 | 개인정보 보호 |
|---|---|---|
| 일반 | `venice/zai-org-glm-5` | Private |
| 추론 | `venice/kimi-k2-5` | Private |
| 코딩 | `venice/claude-opus-4-6` | Anonymized |
| 비전 | `venice/qwen3-vl-235b-a22b` | Private |
| 검열되지 않음 | `venice/venice-uncensored` | Private |

언제든지 기본 모델을 변경할 수 있습니다:

```bash
openclaw models set venice/zai-org-glm-5
```

사용 가능한 모든 모델 나열:

```bash
openclaw models list | grep venice
```

### 4. 채팅 시작

터미널 UI를 엽니다:

```bash
openclaw tui
```

또는 웹 대시보드:

```bash
openclaw dashboard
```

또는 메시징 채널(WhatsApp, Telegram, Discord 등)을 연결:

```bash
openclaw channels login
openclaw gateway
```

## 개인정보 보호 모드

OpenClaw의 Venice 모델은 Venice API와 동일한 [개인정보 보호 등급](/overview/privacy)을 따릅니다:

- **Private** 모델(GLM, Qwen, DeepSeek, Llama, Venice Uncensored)은 Venice의 GPU 플릿에서 실행됩니다. 프롬프트는 저장되거나 기록되지 않습니다.
- **Anonymized** 모델(Claude, GPT, Gemini, Grok)은 모든 식별 정보가 제거된 채로 Venice를 통해 프록시됩니다. 서드파티 제공자는 사용자가 아닌 Venice를 고객으로 봅니다.

## 이미지 및 비디오 생성

이미지 및 비디오 생성을 위한 Venice AI Media 스킬을 설치합니다:

```bash
openclaw skills install nhannah/venice-ai-media
```
