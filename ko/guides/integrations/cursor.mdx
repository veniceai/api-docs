[Cursor](https://www.cursor.com/)는 AI 기반 코드 에디터입니다. Venice AI와 함께 사용하면 다양한 모델에 대해 private하고 검열되지 않은 접근을 할 수 있습니다.

## 설정

<Steps>
  <Step title="API 키 받기">
    [venice.ai/settings/api](https://venice.ai/settings/api)에서 키를 생성하세요.
  </Step>

  <Step title="Cursor 구성">
    **Cursor Settings**(톱니바퀴 아이콘)를 열고 **Models**로 스크롤한 다음 **Add Model**을 클릭하세요.

    **OpenAI Compatible** 아래에 다음을 입력하세요.
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: Venice API 키

  </Step>

  <Step title="모델 추가">
    모델 이름 필드에 Venice 모델 ID를 입력하고 Enter를 누르세요. 사용하려는 각 모델을 추가하세요. 예:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` ([아래 접두사 참고](#the-venice--model-prefix) 참조)

    사용 가능한 모든 모델 ID는 [모델 카탈로그](/ko/models/text)를 참고하세요.
  </Step>

  <Step title="선택 및 확인">
    새 채팅을 열고 **채팅 하단의 모델 선택기**를 사용하여 방금 추가한 Venice 모델 중 하나를 선택하세요. 테스트 메시지를 보내세요. 응답을 받으면 모든 설정이 완료된 것입니다.
  </Step>
</Steps>

## `venice-` 모델 접두사

<Warning>
**Claude 모델은 Cursor에서 사용할 때 `venice-` 접두사가 필요합니다.**

Cursor는 `claude-*` 모델에 대한 요청을 Anthropic의 네이티브 형식으로 재작성하는데, 이는 Venice와 호환되지 않습니다. 모델 ID에 `venice-` 접두사를 붙이면 이 재작성을 방지합니다. Venice는 접두사를 자동으로 제거합니다.
</Warning>

| 모델 | 표준 ID | Cursor ID |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
Claude가 아닌 모델(예: `minimax-m25`, `kimi-k2-5`, `zai-org-glm-5`)은 **영향을 받지 않으며** 접두사 없이 작동합니다. Venice는 항상 접두사를 제거하므로 모든 모델에 `venice-` 접두사를 사용해도 안전하지만, Cursor에서는 Claude 모델에만 필수입니다.
</Info>
