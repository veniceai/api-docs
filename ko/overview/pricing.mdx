### 무료 API 티어가 있나요?

Venice API에는 비용이 0인 모델이 없습니다 — 모든 모델은 토큰당, 이미지당, 초당 또는 요청당 가격이 책정되어 있습니다. Venice.ai Pro 구독의 일일 이미지 및 메시지 한도는 웹 앱과 모바일 앱에만 적용되며, API 호출에는 적용되지 않습니다.

요청당 USD 과금 없이 API를 사용하는 두 가지 방법은 다음과 같습니다:

- **VVV를 스테이킹해 일일 DIEM 크레딧 받기.** VVV 스테이커는 스테이킹 풀 내 자신의 비중에 비례해 매일 DIEM을 배정받습니다. 1 DIEM = 하루 $1 상당의 추론 크레딧이며, 매일 00:00 UTC에 갱신됩니다. 아래 [VVV 스테이킹으로 일일 크레딧 받기](#earn-daily-credits-by-staking-vvv) 섹션을 참조하세요.
- **Pro 구독 크레딧.** Pro 구독자는 업그레이드 시 일회성 $10 API 크레딧을 받습니다. 테스트와 소규모 앱을 위한 것이며, 지속적인 무료 사용을 위한 것이 아닙니다.

`stt-xai-v1`의 Speech-to-text는 오디오 초당 $0.0000로 표시되지만, 이는 반올림 표시상의 아티팩트입니다 — 사용량은 여전히 잔액에서 차감되며 상위 제공업체의 가격으로 과금됩니다.

### VVV 스테이킹으로 일일 크레딧 받기 [#earn-daily-credits-by-staking-vvv]

Base에서 VVV를 스테이킹하면 지갑에 일일 DIEM 한도가 부여되며, 이는 USD 잔액보다 먼저 자동으로 소진됩니다.

1. **Base에서 VVV 획득.** Aerodrome 또는 Uniswap 같은 DEX에서 VVV(컨트랙트 `0xacfE6019Ed1A7Dc6f7B508C02d1b04ec88cC21bf`)를 구매하거나 스왑하세요. 가스비를 위해 Base에 소량의 ETH도 필요합니다.
2. **[venice.ai/token](https://venice.ai/token)에서 스테이킹.** 스테이킹 버튼은 해당 페이지에 있습니다. 내부적으로는 [Venice 스테이킹 컨트랙트](https://basescan.org/address/0x321b7ff75154472B18EDb199033fF4D116F340Ff)에 대한 두 개의 온체인 트랜잭션 — VVV 토큰의 `approve` 그리고 스테이킹 컨트랙트의 `stake` — 로 이루어집니다. VVV 잔액이 줄고 같은 양만큼 sVVV 잔액이 늘어납니다.
3. **매일 DIEM 획득.** 지갑의 일일 DIEM 배정량은 스테이킹 풀 내 비중에 비례하며, 매 에포크마다 00:00 UTC에 갱신됩니다. DIEM을 사용하려면 계정에 최소 **0.1 DIEM 이상**이 누적되어야 합니다.
4. **API에서 사용.** API를 호출하면 발급된 키는 DIEM → 번들 크레딧 → USD 순으로 차감됩니다. DIEM으로 지출이 커버되는 동안에는 요청당 별도 과금이 적용되지 않습니다.

사람 개입 없이 스테이킹 지갑에서 자체 API 키를 발급받는 에이전트에 대해서는 [자율 에이전트 API 키 생성](/guides/getting-started/generating-api-key-agent)을 참조하세요.

## 무검열(uncensored) 및 관대한(permissive) 모델

Venice의 콘텐츠 정책은 대부분의 상위 제공업체보다 관대하며, 위 카탈로그의 여러 모델은 상위 제공업체의 안전 계층 없이 제공됩니다. 모델의 ID 또는 표시 이름으로 이를 구분할 수 있습니다:

- **`*-uncensored*`** — 제공업체의 기본 거부(refusal) 학습을 생략한, Venice가 튜닝하거나 선정한 체크포인트입니다. 예: `venice-uncensored-1-2`, `gemma-4-uncensored`, `qwen-edit-uncensored`, `wan-2-7-uncensored-image-to-video`.
- **`*-heretic*`** — Venice에 제공되기 전에 검열 해제(decensoring) 파인튜닝을 거친 커뮤니티 튜닝 체크포인트입니다(예: `olafangensan-glm-4.7-flash-heretic`). 기본 가중치는 모델 ID에 명시된 상위 제공업체에서 가져오며, `heretic` 접미사는 기본 모델이 아니라 파인튜닝을 식별합니다.
- **롤플레이(Role-play) 변형** — 예: `venice-uncensored-role-play`. 장문의 캐릭터·창작 글쓰기에 맞게 튜닝되었습니다.

Venice에서 uncensored가 아닌 모델도 상위 제공업체가 거부하는 프롬프트에 대해서는 여전히 `CONTENT_POLICY_VIOLATION`(HTTP 422)을 반환합니다. 전체 목록은 [오류 코드](/api-reference/error-codes)를 참조하세요. 가장 관대한 동작을 원한다면 `uncensored` 또는 `heretic` 변형을 선택하고 `venice_parameters.include_venice_system_prompt: false`로 Venice의 기본 시스템 프롬프트를 비활성화하세요 — 파라미터 참조는 [API 명세](/api-reference/api-spec)를 참조하세요.
