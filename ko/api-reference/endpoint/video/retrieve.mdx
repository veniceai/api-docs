비공개 모델의 경우, 완성된 파일은 (이 엔드포인트에서 바이너리로 반환되는 것이 아니라) `/video/queue`가 반환한 `download_url`에서 다운로드합니다. 링크 동작 방식, 재시도, 선택적 `DELETE`에 대해서는 [비공개 다운로드 링크](/ko/guides/media/video-generation#private-download-links)를 참고하세요.
