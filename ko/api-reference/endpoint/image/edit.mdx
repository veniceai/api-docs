## 품질 등급

일부 편집 모델은 시각적 품질과 비용을 절충하는 선택적 `quality` 파라미터(`low`, `medium`, `high`)를 받습니다. 현재 `gpt-image-2-edit`에서 지원되며, 다른 모델은 해당 파라미터를 무시합니다.

```json
{
  "model": "gpt-image-2-edit",
  "image": "iVBORw0KGgo...",
  "prompt": "change the sky to a sunrise",
  "resolution": "2K",
  "quality": "medium"
}
```

`quality`를 생략하면 모델은 기본 등급(`gpt-image-2-edit`의 경우 `high`)을 사용합니다. 등급별 가격(1K/2K/4K × low/medium/high)은 [Models 엔드포인트](/ko/api-reference/endpoint/models/list)의 `model_spec.pricing.quality` 아래에 있으며 [가격 개요](/ko/overview/pricing)에도 나열되어 있습니다.

<Info>
품질 기반 가격 책정은 모든 API 및 SDK 호출자에 대해 기본값입니다 — 별도의 opt-in 헤더가 필요하지 않습니다. `quality`를 생략한 요청은 모델의 기본 등급(`gpt-image-2-edit`의 경우 `high`)으로 청구됩니다.
</Info>
