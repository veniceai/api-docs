Venice crypto RPC 프록시가 지원하는 네트워크 slug의 알파벳 정렬 목록을 반환합니다. 이를 사용해 `POST /crypto/rpc/{network}`에서 유효한 `:network` 값을 확인하세요.

이 엔드포인트는 **공개**이며, API 키나 지갑 인증이 필요하지 않습니다.

## 응답 형태

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

이 목록은 권위 있는 기준입니다 — 여기에 없는 slug라면 프록시 엔드포인트는 `400 Unsupported RPC network`를 반환합니다.
