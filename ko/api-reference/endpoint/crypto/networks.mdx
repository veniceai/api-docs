"og:title": "Crypto Networks | Venice API Docs"
"og:description": "Returns the sorted list of blockchain network slugs supported by the Venice crypto RPC proxy."
---

Venice crypto RPC 프록시에서 지원하는 네트워크 슬러그의 알파벳순으로 정렬된 목록을 반환합니다. 이를 사용하여 `POST /crypto/rpc/{network}`에 유효한 `:network` 값을 찾으세요.

이 엔드포인트는 **공개**이며 — API 키나 지갑 인증이 필요하지 않습니다.

## 응답 형태

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "robinhood-mainnet",
    "robinhood-testnet",
    "solana-devnet",
    "solana-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

이 목록은 결정적입니다 — 슬러그가 여기에 없으면 프록시 엔드포인트는 `400 Unsupported RPC network`를 반환합니다.
