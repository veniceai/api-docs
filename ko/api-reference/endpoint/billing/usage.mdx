사용자의 사용량 데이터를 내보냅니다. 응답 필드 설명은 아래와 같습니다.

- **timestamp**: 결제 사용량 항목이 생성된 타임스탬프
- **sku**: 결제 사용량 항목과 연관된 제품
- **pricePerUnitUsd**: USD 기준 단위당 가격
- **unit**: 사용된 단위 수
- **amount**: 결제 사용량 항목에 청구된 총액
- **currency**: 결제 사용량 항목에 청구된 통화
- **notes**: 결제 사용량 항목에 대한 메모
- **inferenceDetails.requestId**: 추론과 연관된 요청 ID
- **inferenceDetails.inferenceExecutionTime**: 추론 실행에 소요된 시간(밀리초)
- **inferenceDetails.promptTokens**: 프롬프트에서 요청된 토큰 수. LLM 사용에서만 표시됩니다.
- **inferenceDetails.completionTokens**: completion에 사용된 토큰 수. LLM 사용에서만 표시됩니다.
