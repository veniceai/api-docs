Venice는 chat completions 엔드포인트의 `venice_parameters` 입력으로 구동될 수 있는 추가 모델 기능을 지원합니다.

특정 상황에서는 요청 본문을 수정할 수 없는 클라이언트를 사용할 수도 있습니다. 그런 플랫폼에서는 Venice의 모델 기능 접미사 기능을 활용해 모델 ID를 통해 플래그를 전달할 수 있습니다.

## 문법

모델 기능 접미사는 다음 패턴을 따릅니다.

```
<model_id>:<parameter>=<value>
```

여러 파라미터의 경우 `&`로 연결합니다.

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## 예제

### 웹 검색을 Auto로 설정
```
default:enable_web_search=auto
```

### 웹 검색을 활성화하고 시스템 프롬프트 비활성화
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### 웹 검색을 활성화하고 응답에 인용 추가
```
default:enable_web_search=on&enable_web_citations=true
```

### 전체 페이지 스크레이핑을 포함한 웹 검색 활성화
```
default:enable_web_search=on&enable_web_scraping=true
```

### 캐릭터 사용
```
default:character_slug=alan-watts
```

### 추론 모델 응답에서 thinking 블록 숨기기
```
qwen3-4b:strip_thinking_response=true
```

### 지원되는 추론 모델에서 thinking 비활성화
일부 추론 모델(예: Qwen 3)은 thinking 프로세스 비활성화를 지원합니다. 아래 접미사를 사용해 활성화할 수 있습니다.
```
qwen3-4b:disable_thinking=true
```

### 스트리밍 응답에 웹 검색 결과 추가
이 옵션은 웹 검색을 활성화하고, 응답 본문에 인용을 추가하며, 스트림의 최종 응답 메시지로 검색 결과를 포함시킵니다.

[Postman 컬렉션](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041)에서 예제를 확인할 수 있습니다.

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
