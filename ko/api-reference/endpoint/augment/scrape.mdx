<Warning>이는 실험적 API입니다. 요청 및 응답 형식은 사전 공지 없이 변경될 수 있습니다.</Warning>

`url` 필드에 공개적으로 접근 가능한 URL을 보내세요. API는 페이지 콘텐츠를 **마크다운**으로 반환합니다.

스크레이퍼는 먼저 대상 사이트의 네이티브 마크다운 지원([Cloudflare Markdown for Agents](https://blog.cloudflare.com/markdown-for-agents/) 경유)을 시도한 다음, 헤드리스 브라우저 추출로 폴백합니다. 자동화된 접근을 차단하는 일부 사이트(예: X/Twitter, Reddit)는 즉시 `400` 오류로 거부됩니다.

**가격:** 요청당 $0.01.

### 예제 (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/scrape \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```
