## 음성 카탈로그

음성은 **모델별**입니다. 전달하는 `voice`는 선택한 `model`의 카탈로그에서
가져온 것이어야 합니다. 아래에서 모델을 선택하여 음성을 둘러보세요.

<div id="tts-voice-picker-placeholder">음성 로딩 중...</div>

<Note>
음성 ID는 대소문자를 구분하며 **일치하는 `model`에서만 유효**합니다. 요청 페이로드에서
두 필드를 함께 전달하세요. 예제는
[Audio Speech API](/ko/api-reference/endpoint/audio/speech)를 참고하세요.
</Note>

### 예제 요청

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

모델을 전환하려면 **둘 다** `model`과 `voice`를 위에서 선택한 모델의 쌍으로 변경하세요.
