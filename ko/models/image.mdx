## 모델 유형

- **Generation:** 텍스트 프롬프트에서 이미지 생성
- **Upscale:** 이미지 해상도와 품질 향상
- **Edit:** 인페인팅으로 기존 이미지 수정

<Note>
text-to-image는 [Image Generate API](/ko/api-reference/endpoint/image/generate), 향상은 [Upscale API](/ko/api-reference/endpoint/image/upscale), 인페인팅은 [Edit API](/ko/api-reference/endpoint/image/edit)를 참고하세요.
</Note>

<Tip>
이미지 생성 크기는 모델별로 다릅니다. `venice-sd35`, `qwen-image`와 같은 픽셀 기반 모델은 `width`와 `height`를 사용합니다. `qwen-image-2`와 같은 종횡비 모델은 `aspect_ratio`를 사용합니다. `1K`, `2K`, `4K`와 같은 해상도 옵션을 나열하는 모델은 `resolution`과 `aspect_ratio` 모두를 사용합니다.
</Tip>

<Tip>
**품질 등급.** `gpt-image-2`(생성) 및 `gpt-image-2-edit`(편집/멀티 편집)은 선택적 `quality` 파라미터 — `low`, `medium` 또는 `high`(기본값) — 를 받습니다. 낮은 등급은 비용이 적습니다. 전체 매트릭스는 [가격 개요](/ko/overview/pricing#quality-tier-pricing-gpt-image-2)를 참고하거나 [Models 엔드포인트](/ko/api-reference/endpoint/models/list)에서 `model_spec.pricing.quality`를 가져오세요.
</Tip>
