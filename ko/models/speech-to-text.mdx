## 사용

음성 텍스트 변환 모델은 말한 오디오를 작성된 텍스트로 변환합니다. [Audio Transcriptions API](/ko/api-reference/endpoint/audio/transcriptions)를 통해 접근됩니다.

### 지원되는 오디오 형식

`mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `wav`, `webm`, `flac`, `ogg`

### 응답 형식

| 형식 | 설명 |
|--------|-------------|
| `json` | 기본값. `{ "text": "..." }`를 반환합니다. |
| `text` | 일반 변환 텍스트. |
| `srt` | 타임스탬프가 있는 SubRip 자막 형식. |
| `vtt` | 타임스탬프가 있는 WebVTT 자막 형식. |
| `verbose_json` | 세그먼트 수준 타임스탬프 및 메타데이터가 있는 전체 응답. |

<Note>
가격은 입력 오디오의 초 단위로 청구됩니다. 요청 예제와 파라미터 세부 사항은 [Audio Transcriptions API](/ko/api-reference/endpoint/audio/transcriptions)를 참고하세요.
</Note>
