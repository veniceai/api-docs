## Model Traits

| Trait | Model | Description |
|-------|-------|-------------|
| `default` | Llama 3.3 70B | Default model for general chat |
| `default_vision` | Venice Medium | Default model for vision tasks |
| `default_code` | Qwen 3 Coder 480B | Default model for code generation |
| `fastest` | Llama 3.2 3B | Fastest response times |
| `most_uncensored` | Venice Uncensored 1.1 | Least content restrictions |
| `function_calling_default` | Llama 3.3 70B | Default for function calling |
