[Clawdbot](https://docs.clawd.bot/) is an agentic AI assistant that runs in your terminal and connects to messaging platforms. Venice AI is available as a built-in provider, giving you access to privacy-focused, uncensored models.

## Quick Start

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://clawd.bot/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://clawd.bot/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm / pnpm">
    ```bash
    npm install -g clawdbot
    ```
  </Tab>
</Tabs>

```bash
clawdbot onboard
```

## Onboarding

The wizard prompts you through setup. Select these options for Venice:

<Steps>
  <Step title="Provider">
    Select **Venice AI (Privacy-focused (uncensored models))**
  </Step>
  <Step title="API Key">
    Paste your key from [venice.ai/settings/api](https://venice.ai/settings/api)
  </Step>
  <Step title="Model">
    Choose `venice/zai-org-glm-4.7` or another Venice model
  </Step>
  <Step title="Channel">
    Connect Discord, Telegram, Slack, or skip for terminal-only
  </Step>
  <Step title="Skills">
    Select **No** to skip default skills
  </Step>
  <Step title="Hooks">
    Select **Skip for now**
  </Step>
</Steps>

## Usage

Launch the terminal UI:

```bash
clawdbot tui
```

Or open the web dashboard:

```bash
clawdbot dashboard
```

---

## Image and Video Generation

Install the [Venice AI Media skill](https://clawdhub.com/nhannah/venice-ai-media) for image and video generation:

```bash
clawdbot skills install nhannah/venice-ai-media
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Venice AI not appearing as provider">
    ```bash
    npm update -g clawdbot
    clawdbot onboard
    ```
  </Accordion>

  <Accordion title="Reconfigure API key">
    ```bash
    clawdbot configure
    ```
  </Accordion>

  <Accordion title="Skill installation fails">
    ```bash
    npm update -g clawdbot
    clawdbot skills install nhannah/venice-ai-media
    ```
