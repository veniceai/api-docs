[Molt Bot](https://molt.bot) is a Discord bot that enables AI-powered conversations in your server using the Venice API.

## Quick Start

1. Invite Molt Bot to your Discord server from [molt.bot](https://molt.bot)
2. Get your Venice API Key by following our [API key generation guide](/overview/guides/generating-api-key)
3. Configure Venice as your provider using the `/provider` command
4. Start chatting with the bot

For detailed instructions, visit the [official Molt Bot documentation](https://docs.molt.bot/providers/venice).
