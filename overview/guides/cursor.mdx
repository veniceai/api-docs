## Setup

<Steps>
  <Step title="Get Your API Key">
    Generate a key from [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="Configure Cursor">
    Open **Cursor Settings** (gear icon), scroll to **Models** and click **Add Model**.

    Under **OpenAI Compatible**, enter:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: Your Venice API key
  </Step>

  <Step title="Add Models">
    In the model name field, type the Venice model ID and press Enter. For example:
    - `minimax`
    - `kimi-k2.5`
    - `venice-claude-opus-4-6` (see [prefix note below](#the-venice--model-prefix))

    See the [model catalog](/models/text) for all available model IDs.
  </Step>

  <Step title="Verify">
    Open a file, select some code, and press **Ctrl+K** (or **Cmd+K** on Mac) to invoke Cursor's inline edit. Pick your Venice model from the dropdown. If you get a response, you're all set.
  </Step>
</Steps>

## The `venice-` Model Prefix

<Warning>
**Claude models require the `venice-` prefix when used in Cursor.**

Cursor automatically detects model IDs beginning with `claude-` and converts the request to Anthropic's native API format. Since Venice exclusively accepts OpenAI-compatible requests, this conversion causes the request to fail.

To resolve this, prefix the model ID with `venice-` (for example, `venice-claude-opus-4-6`). Venice automatically strips this prefix during model resolution, allowing the request to reach the correct model while remaining in OpenAI-compatible format.
</Warning>

| Model | Breaks in Cursor | Use in Cursor |
|-------|-----------------|---------------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-45` | `venice-claude-opus-45` |
| Claude Sonnet 4.5 | `claude-sonnet-45` | `venice-claude-sonnet-45` |

<Info>
Non-Claude models (e.g. `minimax`, `kimi-k2.5`, `gpt-4o`) are **not affected** and work without the prefix. The `venice-` prefix is safe to use on any model since Venice always strips it, but it is only required for Claude models in Cursor.
</Info>
