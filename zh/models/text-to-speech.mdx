## 语音目录

语音是**与模型相关**的。你传入的 `voice` 必须来自所选 `model` 的语音目录。在下方选择一个模型以浏览其可用语音。

<div id="tts-voice-picker-placeholder">Loading voices...</div>

<Note>
语音 ID 区分大小写，并且**仅对匹配的 `model` 有效**。请在请求体中同时传入两个字段。示例请参见
[Audio Speech API](/zh/api-reference/endpoint/audio/speech)。
</Note>

### 示例请求

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

要切换模型，请将 `model` 和 `voice` **同时**改为上方所选模型的一对组合。
