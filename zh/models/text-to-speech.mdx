## 语音目录

语音是**模型专属的**。您传入的 `voice` 必须来自所选 `model`
的语音目录。请在下面选择一个模型以浏览其语音。

<div id="tts-voice-picker-placeholder">Loading voices...</div>

<Note>
语音 ID 区分大小写，并且**仅对匹配的 `model` 有效**。请在请求载荷中
同时传入这两个字段。请参阅
[Audio Speech API](/api-reference/endpoint/audio/speech) 查看示例。
</Note>

### 请求示例

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

如需切换模型，请将 `model` 和 `voice` **同时**改为上方所选模型中的一对组合。
