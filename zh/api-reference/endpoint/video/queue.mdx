### Seedance 2.0

对于 `seedance-2-0-*` 系列模型（text-to-video、image-to-video、reference-to-video，以及 `-fast-*` 变体），请参阅 [Seedance 2.0 指南](/zh/guides/media/seedance-2-0)，了解四种工作流模型（Reference / Edit / Extend / Stitch）、规范化提示模式、多模态输入限制以及定价详情。

### Seedance 同意

当 Seedance 的 image-to-video 或 reference-to-video 请求包含人脸时，API 会返回 `409 needs_consent` 以及策略文本，你需要在请求中加上 `consents.seedance` 证明并重新提交。完整流程、同意对象、去重与撤销方式请参阅 [Seedance 人脸媒体同意指南](/zh/guides/media/seedance-face-consent)。

### 视频放大

对于 `topaz-video-upscale` 模型，使用 `upscale_factor`（1、2 或 4）代替 `resolution`，并提供 `video_url`。时长和 FPS 会从视频文件自动检测。完整细节和示例请参阅[视频放大指南](/zh/guides/media/video-upscaling)。
