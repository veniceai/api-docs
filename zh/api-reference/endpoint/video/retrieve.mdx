对于私有模型，已完成的文件通过 `/video/queue` 返回的 `download_url` 下载（而非通过此端点以二进制方式返回）。有关下载链接的工作方式、重试机制及可选的 `DELETE`，请参阅[私有下载链接](/guides/media/video-generation#private-download-links)。
