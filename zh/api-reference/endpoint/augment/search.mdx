<Warning>这是一个实验性 API。请求和响应格式可能在不另行通知的情况下发生变更。</Warning>

在 `query` 字段中发送搜索查询。该 API 会返回结构化结果，包括标题、URL、内容片段和日期。

**搜索提供商：**
- `brave`（默认）—— 采用零数据留存（ZDR）的 Brave Search。搜索查询不会被搜索提供商存储或记录。
- `google` —— 使用匿名化查询的 Google Search。搜索通过 Venice 基础设施进行代理，因此发送给 Google 的搜索请求不会与你的身份相关联。Venice 不存储或记录搜索查询。

**定价：** 每次请求 0.01 美元。

### 示例（cURL）

```bash
curl -X POST https://api.venice.ai/api/v1/augment/search \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "latest news about AI"}'
```
