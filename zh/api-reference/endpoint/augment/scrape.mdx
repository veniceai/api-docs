<Warning>这是一个实验性 API。请求和响应格式可能在不另行通知的情况下发生变更。</Warning>

在 `url` 字段中发送可公开访问的 URL。该 API 会以 **markdown** 格式返回页面内容。

抓取器会先尝试目标站点的原生 markdown 支持（通过 [Cloudflare Markdown for Agents](https://blog.cloudflare.com/markdown-for-agents/)），然后回退到无头浏览器抽取。某些屏蔽自动化访问的站点（例如 X/Twitter、Reddit）会立即被拒绝并返回 `400` 错误。

**定价：** 每次请求 0.01 美元。

### 示例（cURL）

```bash
curl -X POST https://api.venice.ai/api/v1/augment/scrape \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```
