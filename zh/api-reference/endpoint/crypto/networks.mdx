返回 Venice 加密 RPC 代理支持的按字母顺序排序的网络 slug 列表。可使用此端点来发现哪些 `:network` 值对 `POST /crypto/rpc/{network}` 有效。

此端点为**公开**接口 —— 无需 API 密钥或钱包认证。

## 响应结构

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

该列表具有权威性 —— 如果某个 slug 不在其中，代理端点会返回 `400 Unsupported RPC network`。
