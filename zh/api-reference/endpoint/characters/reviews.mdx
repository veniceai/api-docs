<Warning>
这是一个实验性端点，可能会发生变更。
</Warning>

## 此端点的返回内容

此端点返回单个角色的分页公开评价。

- 使用 `slug` 路径参数标识角色。
- 使用 `page` 和 `pageSize` 查询参数对评价进行分页。
- 分页元数据会同时返回在响应正文和 `x-pagination-*` 响应头中。
