## 此端点的返回内容

此端点返回单个角色的分页公开评论。

- 使用 `slug` 路径参数标识角色。
- 使用 `page` 和 `pageSize` 查询参数分页浏览评论。
- 分页元数据同时返回在响应正文和 `x-pagination-*` 响应头中。
