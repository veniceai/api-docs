导出用户的使用数据。响应字段说明如下：

- **timestamp**：账单使用条目创建的时间戳
- **sku**：与该账单使用条目关联的产品
- **pricePerUnitUsd**：每单位的美元价格
- **unit**：消耗的单位数量
- **amount**：该账单使用条目的总收费金额
- **currency**：账单使用条目收费的货币
- **notes**：关于账单使用条目的备注
- **inferenceDetails.requestId**：与推理关联的请求 ID
- **inferenceDetails.inferenceExecutionTime**：推理执行所用时间（毫秒）
- **inferenceDetails.promptTokens**：提示中请求的 token 数量。仅在 LLM 使用时出现。
- **inferenceDetails.completionTokens**：补全中使用的 token 数量。仅在 LLM 使用时出现。
