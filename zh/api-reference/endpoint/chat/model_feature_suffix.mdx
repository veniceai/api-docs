Venice 在其模型中支持一些额外功能，可通过聊天补全端点上的 `venice_parameters` 输入来启用。

在某些情况下，你使用的客户端可能不允许修改请求正文。对于这些平台，你可以利用 Venice 的模型特性后缀（Model Feature Suffix）能力，通过模型 ID 传入标志。

## 语法

模型特性后缀遵循以下格式：

```
<model_id>:<parameter>=<value>
```

如需传入多个参数，可使用 `&` 拼接：

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## 示例

### 将网页搜索设置为自动
```
default:enable_web_search=auto
```

### 启用网页搜索并禁用系统提示
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### 启用网页搜索并在响应中添加引用
```
default:enable_web_search=on&enable_web_citations=true
```

### 启用网页搜索并启用整页抓取
```
default:enable_web_search=on&enable_web_scraping=true
```

### 使用某个角色
```
default:character_slug=alan-watts
```

### 在推理模型响应中隐藏思考块
```
qwen3-4b:strip_thinking_response=true
```

### 在支持的推理模型上禁用思考
某些推理模型（如 Qwen 3）支持禁用思考过程。你可以使用下面的后缀来启用此功能：
```
qwen3-4b:disable_thinking=true
```

### 将网页搜索结果加入流式响应
这将启用网页搜索、在响应正文中添加引用，并将搜索结果作为最终响应消息包含在流中。

你可以在我们的 [Postman 集合中查看示例](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041)。

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
