## 质量层级

部分多图编辑模型接受可选的 `quality` 参数（`low`、`medium`、`high`），用于在视觉保真度和成本之间权衡。当前由 `gpt-image-2-edit` 支持；其他模型会忽略该参数。

```json
{
  "modelId": "gpt-image-2-edit",
  "images": ["iVBORw0KGgo...", "iVBORw0KGgo..."],
  "prompt": "blend the two scenes into one cinematic frame",
  "resolution": "2K",
  "quality": "medium"
}
```

当你省略 `quality` 时，模型使用其默认层级（`gpt-image-2-edit` 为 `high`）。各层级价格（1K/2K/4K × low/medium/high）位于 [Models 端点](/zh/api-reference/endpoint/models/list) 中 `model_spec.pricing.quality` 字段下，并在[定价概览](/zh/overview/pricing)中列出。

<Info>
质量感知的定价对所有 API 和 SDK 调用方都是默认行为 —— 无需选择加入的请求头。未传 `quality` 的请求按模型的默认层级计费（`gpt-image-2-edit` 为 `high`）。
</Info>
