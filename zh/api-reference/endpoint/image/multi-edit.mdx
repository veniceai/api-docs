## 质量等级

部分多图像编辑模型接受可选的 `quality` 参数（`low`、`medium`、`high`），用于在视觉保真度和成本之间取得平衡。当前由 `gpt-image-2-edit` 支持；其他模型会忽略该参数。

```json
{
  "modelId": "gpt-image-2-edit",
  "images": ["iVBORw0KGgo...", "iVBORw0KGgo..."],
  "prompt": "blend the two scenes into one cinematic frame",
  "resolution": "2K",
  "quality": "medium"
}
```

当您省略 `quality` 时，模型将使用其默认等级（`gpt-image-2-edit` 为 `high`）。各等级价格（1K/2K/4K × low/medium/high）位于 [Models 端点](/api-reference/endpoint/models/list)中 `model_spec.pricing.quality` 下，也在[定价概览](/overview/pricing)中列出。

<Info>
质量感知定价是所有 API 和 SDK 调用方的默认行为 —— 无需显式启用的请求头。省略 `quality` 的请求按模型的默认等级（`gpt-image-2-edit` 为 `high`）计费。
</Info>
