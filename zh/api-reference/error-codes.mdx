---
title: 错误码
---

当发生错误时，Venice API 会返回 HTTP 状态码和描述性消息。部分错误还包括额外字段，例如退款状态、建议的重试模型、校验详情或请求 ID。

## 错误响应格式

大多数错误返回带有人类可读消息的 `error` 字段：

```json
{
  "error": "Invalid request parameters"
}
```

某些端点会包含端点特有的额外字段。例如，视频内容审核错误可能包含是否退还了 credit：

```json
{
  "error": "Your generation was blocked due to the model provider's content policies. Credits have been refunded.",
  "credits_refunded": true
}
```

校验错误可能包含 `details` 和 `issues` 字段，提供字段级别的校验信息：

```json
{
  "error": "Invalid request parameters",
  "details": {
    "_errors": [],
    "field": {
      "_errors": ["Field is required"]
    }
  },
  "issues": []
}
```

部分 OpenAI 兼容端点在特定情况下（例如上下文长度错误）可能会返回 OpenAI 风格的错误对象：

```json
{
  "error": {
    "message": "Your request exceeds the model's maximum context. Please reduce your prompt or completion length.",
    "type": "invalid_request_error",
    "param": "messages",
    "code": "context_length_exceeded"
  }
}
```

## 错误码

| 错误码 | HTTP 状态 | 消息 | 日志级别 |
| --- | --- | --- | --- |
| `AUTHENTICATION_FAILED` | 401 | 认证失败 | - |
| `AUTHENTICATION_FAILED_INACTIVE_KEY` | 401 | 认证失败 —— Pro 订阅未激活。请升级订阅以继续使用 API | - |
| `X402_INVALID_SIGN_IN` | 401 | Sign-in-with-x 签名无效 | - |
| `PRO_ONLY_MODEL` | 401 | 此模型仅向 Pro 用户开放 | - |
| `INSUFFICIENT_BALANCE` | 402 | USD 或 Diem 余额不足以完成请求。请访问 https://venice.ai/settings/api 添加 credit。 | - |
| `API_KEY_DIEM_SPEND_LIMIT_EXCEEDED` | 402 | API 密钥 DIEM 花费上限已超出。你的账户可能仍有 DIEM 余额，但该 API 密钥已达到其配置的 DIEM 花费上限。 | - |
| `API_KEY_USD_SPEND_LIMIT_EXCEEDED` | 402 | API 密钥 USD 花费上限已超出。你的账户可能仍有 USD 余额，但该 API 密钥已达到其配置的 USD 花费上限。 | - |
| `UNAUTHORIZED` | 403 | 未授权访问 | - |
| `API_ACCESS_DISABLED` | 403 | 该账户的 API 访问已被禁用。请联系 support@venice.ai | - |
| `X402_WALLET_MISMATCH` | 403 | 你只能访问属于自己钱包的资源 | - |
| `INVALID_REQUEST` | 400 | 请求参数无效 | - |
| `INVALID_MODEL` | 400 | 指定的模型无效 | - |
| `REQUEST_ID_NOT_FOUND` | 400 | 请求 ID 无效。 | - |
| `INVALID_AUDIO_FORMAT` | 400 | 音频格式无效。支持的格式为 WAV 和 MP3。 | - |
| `INVALID_VIDEO_FORMAT` | 400 | 视频格式无效。支持的格式包括 MP4、MOV、WebM、MKV、AVI 等。 | - |
| `CORRUPTED_IMAGE` | 400 | 图像文件已损坏或无法读取 | - |
| `IMAGE_TOO_SMALL` | 400 | 图像尺寸过小 | - |
| `TOO_MANY_TOKENS` | 400 | 请求超出了模型的最大上下文。请缩短提示或补全长度。 | - |
| `CHARACTER_NOT_FOUND` | 404 | 根据所提供的 character_slug 找不到角色 | - |
| `MODEL_NOT_FOUND` | 404 | 找不到指定的模型 | - |
| `MEDIA_NOT_FOUND` | 404 | 找不到媒体。请求可能无效、过期或已被删除。 | - |
| `PAYLOAD_TOO_LARGE` | 413 | 请求负载过大。请减小请求体积。 | - |
| `INVALID_CONTENT_TYPE` | 415 | 请求 content-type 无效 | - |
| `VIDEO_DURATION_TOO_LONG` | 422 | 视频时长超过允许的最大值。 | - |
| `VIDEO_DURATION_TOO_SHORT` | 422 | 视频时长过短。 | - |
| `IMAGE_TOO_LARGE` | 422 | 图像尺寸超过允许的最大值。 | - |
| `CONTENT_POLICY_VIOLATION` | 422 | 你的提示违反了 Venice.ai 或模型提供商的内容策略 | - |
| `ASR_UPSTREAM_VALIDATION_FAILED` | 422 | 无法处理该音频进行转录。常见原因：零长度、静音、损坏或语言不受支持的音频。请检查文件并重试。 | warn |
| `RATE_LIMIT_EXCEEDED` | 429 | 速率限制已超出 | - |
| `MODEL_OVERLOADED` | 429 | 模型当前过载。请稍后再试。 | - |
| `INFERENCE_FAILED` | 500 | 推理处理失败 | error |
| `UPSCALE_FAILED` | 500 | 图像放大失败 | error |
| `IMAGE_EDIT_ERROR` | 500 | 图像编辑失败 | error |
| `UNKNOWN_ERROR` | 500 | 发生未知错误 | error |
| `TEE_ATTESTATION_FAILED` | 502 | TEE 证明请求失败。可信执行环境提供商可能暂时不可用。 | error |
| `TEE_SIGNATURE_FAILED` | 502 | TEE 签名请求失败。可信执行环境提供商可能暂时不可用。 | error |
| `ASR_UPSTREAM_FAILED` | 502 | 由于临时上游错误，音频转录失败。请重试。 | warn |
| `MODEL_OFFLINE` | 503 | 模型暂时离线。请稍后再试。 | - |
| `MODEL_AT_CAPACITY` | 503 | 模型达到容量上限。请稍后再试。 | - |
| `REQUEST_TIMEOUT` | 504 | 请求耗时过长已超时。对于长时间运行的推理请求，请在请求中设置 `stream=true` 使用流式 API。 | - |
