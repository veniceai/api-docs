我们有时会以 Beta 形式发布模型，以收集反馈并在全面投入生产前确认其表现。Beta 模型对所有用户开放，但**不推荐用于生产环境**。

Beta 状态并不保证最终晋升到生产环境。如果某个 Beta 模型运行成本过高、在大规模下表现不佳，或带来安全顾虑，可能会被移除。Beta 模型可能在不另行通知的情况下发生变更，文档或支持也可能有限。被证明稳定、广泛实用并符合我们标准的模型会升级到正式可用（GA）。

## 重要注意事项

使用 Beta 模型时，请注意：

- 可能在任意时间被更改或移除，不享有标准弃用通知期
- 不适用于生产应用或关键业务流程
- 性能、可用性或行为可能不稳定
- 被移除时迁移支持有限或没有
- 最适合用于测试、评估和实验性项目

对于生产应用，我们建议使用[主模型阵容](/zh/models/overview)中的稳定模型。

### 通过 API 检查 Beta 状态

你可以通过调用 [List Models](/zh/api-reference/endpoint/models/list) 端点来检查模型是否处于 Beta 状态。Beta 模型在其 `model_spec` 中包含一个值为 `true` 的 `betaModel` 字段：

```json
{
  "id": "some-beta-model",
  "model_spec": {
    "name": "Some Beta Model",
    "betaModel": true,
    "privacy": "private"
  },
  "type": "text",
  "object": "model",
  "owned_by": "venice.ai"
}
```

你可以通过判断 `if (model.model_spec.betaModel)` 来识别 Beta 模型，并在应用中给出警告或采用不同的处理方式。
