我们有时会以 beta 形式发布模型，以便在全面投入生产前收集反馈并确认其表现。Beta 模型对所有用户开放，但**不建议用于生产环境**。

Beta 状态并不保证最终晋升为正式版。如果某个 beta 模型运行成本过高、规模化下表现不佳，或引发安全担忧，可能会被下线。Beta 模型可能在不通知的情况下变更，文档或支持可能有限。被证明稳定、广泛实用且符合我们标准的模型将晋升为正式可用。

## 重要注意事项

使用 beta 模型时，请注意：

- 可能在没有标准弃用通知期的情况下随时被更改或移除
- 不适用于生产应用或关键工作流
- 可能存在性能、可用性或行为不一致的情况
- 移除时迁移支持有限或无支持
- 最适合用于测试、评估和实验性项目

对于生产应用，我们建议使用[主模型阵容](/models/overview)中的稳定模型。

### 通过 API 检查 Beta 状态

您可以通过调用 [List Models](/api-reference/endpoint/models/list) 端点来检查某个模型是否处于 beta 状态。Beta 模型在其 `model_spec` 中包含 `betaModel` 字段且值为 `true`：

```json
{
  "id": "some-beta-model",
  "model_spec": {
    "name": "Some Beta Model",
    "betaModel": true,
    "privacy": "private"
  },
  "type": "text",
  "object": "model",
  "owned_by": "venice.ai"
}
```

您可以通过 `if (model.model_spec.betaModel)` 来识别 beta 模型，并在应用中提醒用户或以不同方式处理。
