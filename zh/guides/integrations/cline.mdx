[Cline](https://cline.bot/) 是 VS Code 的 AI 编码助手，可以读取和编辑文件、运行命令，并完成多步骤的开发任务。Venice 通过 Cline 的兼容 OpenAI 的提供商设置工作。

  <Card title="OpenAI 兼容" icon="plug">

  <Card title="模型灵活性" icon="microchip">
    选择任意兼容的 Venice 文本模型
  </Card>

## 前置条件

- 来自 [venice.ai/settings/api](https://venice.ai/settings/api) 的 Venice API 密钥
- 已安装 [Visual Studio Code](https://code.visualstudio.com/)
- 在 VS Code 中安装了 [Cline 扩展](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev)

---

## 设置

<Steps>
  <Step title="打开 Cline 设置">
    在 VS Code 活动栏中选择 Cline 图标，然后选择齿轮图标以打开 Cline 的设置。
  </Step>

  <Step title="选择 API 提供商">
    将 **API Provider** 设置为 **OpenAI Compatible**。
  </Step>

  <Step title="配置 Venice">
    输入以下值：

    | 设置 | 值 |
    | --- | --- |
    | Base URL | `https://api.venice.ai/api/v1` |
    | API Key | 您的 Venice API 密钥 |
    | Model ID | 来自[文本模型目录](/models/text)的模型 ID |

    例如，使用 `zai-org-glm-5-1` 作为 Model ID。

    请直接输入 API 密钥本身，不要带 `Bearer` 前缀。将 Base URL 保持为 `/api/v1`；Cline 会自行添加 `/chat/completions` 路径。
  </Step>

  <Step title="配置模型能力">
    展开 **Model Configuration**，将上下文窗口和最大输出 token 数设置为与所选模型匹配。启用该模型支持的能力，包括适用时的图像输入或计算机使用（computer use）。

    在启用可选能力之前，请在[文本模型目录](/models/text)中查看所选模型。
  </Step>

  <Step title="保存并测试">
    保存提供商设置，打开一个项目，交给 Cline 一个小任务，例如：

    ```txt
    Review the current file and suggest one safe improvement.
    ```

    在批准每个文件或命令权限之前请仔细审查。
  </Step>
</Steps>

## 验证 Venice API

如果 Cline 无法连接，请独立确认 API 密钥和可用的模型 ID：

```bash
curl https://api.venice.ai/api/v1/models \
  -H "Authorization: Bearer $VENICE_API_KEY"
```

将模型的 `id` 完整复制到 Cline 的 **Model ID** 字段。

---

## 配置注意事项

- 使用 `https://api.venice.ai/api/v1` 作为 Base URL，而不是完整的 `/chat/completions` 端点。
- 模型 ID 可能会随时间变化。请使用[文本模型目录](/models/text)或 `GET /models`，而不是依赖过时的示例。
- Cline 的上下文窗口和输出 token 设置会影响其对请求的预算分配。为获得最佳结果，请将其与所选模型匹配。
- 请妥善保管您的 Venice API 密钥。不要将其粘贴到源文件或提交到 git。

## 疑难排查

<AccordionGroup>
  <Accordion title="401 或身份验证错误">
    确认 API 密钥来自 [venice.ai/settings/api](https://venice.ai/settings/api)。将原始密钥粘贴到 Cline 中，不要带 `Bearer` 或多余的空白字符。
  </Accordion>
  <Accordion title="找不到模型">
    确认该 Model ID 出现在[文本模型目录](/models/text)或 `GET /models` 的响应中，然后完整复制。
  </Accordion>
  <Accordion title="连接或端点错误">
    确认 **API Provider** 为 **OpenAI Compatible**，且 Base URL 严格为 `https://api.venice.ai/api/v1`。
  </Accordion>
  <Accordion title="Cline 无法使用工具或编辑文件">
    打开 **Model Configuration**，确认所选模型支持所需的能力，如果 Cline 为该模型暴露了 computer use 选项，请予以启用。
  </Accordion>
</AccordionGroup>
