## 设置

<Steps>
  <Step title="获取 API 密钥">
    在 [venice.ai/settings/api](https://venice.ai/settings/api) 生成密钥。
  </Step>

  <Step title="配置 Cursor">
    打开 **Cursor Settings**（齿轮图标），滚动到 **Models**，点击 **Add Model**。

    在 **OpenAI Compatible** 下，填入：
    - **Override OpenAI Base URL**：`https://api.venice.ai/api/v1`
    - **OpenAI API Key**：你的 Venice API 密钥
  </Step>

  <Step title="添加模型">
    在模型名称字段中输入 Venice 模型 ID 并按回车。逐个添加你希望使用的模型。例如：
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6`（见[下方的前缀说明](#the-venice--model-prefix)）

    所有可用模型 ID 详见[模型目录](/zh/models/text)。
  </Step>

  <Step title="选择并验证">
    打开一个新对话，使用**聊天底部的模型选择器**选择刚刚添加的某个 Venice 模型。发送一条测试消息。如果收到响应，就一切就绪。
  </Step>
</Steps>

## `venice-` 模型前缀

<Warning>
**在 Cursor 中使用 Claude 模型时需要使用 `venice-` 前缀。**

Cursor 会把 `claude-*` 模型的请求重写为 Anthropic 的原生格式，这与 Venice 不兼容。在模型 ID 前加上 `venice-` 前缀可以避免该重写。Venice 会自动去掉前缀。
</Warning>

| 模型 | 标准 ID | Cursor 中的 ID |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
非 Claude 模型（如 `minimax-m25`、`kimi-k2-5`、`zai-org-glm-5`）**不受影响**，无需前缀即可使用。`venice-` 前缀在任何模型上都安全，因为 Venice 总是会去除它，但在 Cursor 中仅 Claude 模型需要使用。
</Info>
