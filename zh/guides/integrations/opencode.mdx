[OpenCode](https://opencode.ai/) 是一款面向终端的 AI 编码代理。通过 OpenCode 的自定义提供者配置，使用 OpenAI 兼容适配器和自定义 base URL，即可让 Venice 工作。

## 前置条件

- 在 [venice.ai/settings/api](https://venice.ai/settings/api) 获取 Venice API 密钥
- 已在本机安装并能正常运行 OpenCode

---

## 设置

<Steps>
  <Step title="设置你的 Venice API 密钥">
    在运行 OpenCode 的 shell 中导出 Venice API 密钥：

    ```bash
    export VENICE_API_KEY="your-api-key"
    ```
  </Step>

  <Step title="创建 OpenCode 配置文件">
    针对项目级配置，在项目根目录创建 `opencode.json`。

    全局配置则改为使用 `~/.config/opencode/opencode.json`。
  </Step>

  <Step title="添加 Venice 提供者">
    将以下配置粘贴到 `opencode.json`：

    ```json
    {
      "$schema": "https://opencode.ai/config.json",
      "model": "venice/zai-org-glm-5-1",
      "small_model": "venice/zai-org-glm-5-1",
      "provider": {
        "venice": {
          "npm": "@ai-sdk/openai-compatible",
          "name": "Venice AI",
          "options": {
            "baseURL": "https://api.venice.ai/api/v1",
            "apiKey": "{env:VENICE_API_KEY}"
          },
          "models": {
            "zai-org-glm-5-1": {
              "name": "GLM 5.1"
            },
            "kimi-k2-5": {
              "name": "Kimi K2.5"
            },
            "minimax-m25": {
              "name": "MiniMax M2.5"
            }
          }
        }
      }
    }
    ```
  </Step>

  <Step title="运行 OpenCode">
    在同一个项目目录启动 OpenCode：

    ```bash
    opencode
    ```

    OpenCode 会加载 `opencode.json`、使用 `VENICE_API_KEY`，并通过 `https://api.venice.ai/api/v1` 路由所选 Venice 模型。
  </Step>
</Steps>

---

## 验证设置

在 OpenCode 中运行：

```txt
/models
```

你应当能看到配置中的 Venice 模型。选择其中之一，然后发送一条简短测试提示。

如果按上文设置了 `"model": "venice/zai-org-glm-5-1"`，OpenCode 应默认使用该 Venice 模型。要更改默认模型，将 `zai-org-glm-5-1` 替换为[文本模型目录](/zh/models/text)中的任意模型 ID，并把它加入 `models` 映射。

---

## 配置说明

- 保持 `baseURL` 为 `https://api.venice.ai/api/v1`。OpenCode 的 OpenAI 兼容适配器以该 base URL 进行聊天补全。
- 保持 `npm` 为 `@ai-sdk/openai-compatible`，以使用 Venice 的 OpenAI 兼容聊天端点。
- 提供者 ID 为 `venice`，因此模型引用形如 `venice/<model-id>`。
- 如果你偏好 OpenCode 的 `/connect` 流程，选择 **Other**，用 `venice` 作为提供者 ID，粘贴你的 Venice API 密钥，并从配置中移除 `options.apiKey`。

<Info>
OpenCode 已通过 `@ai-sdk/openai-compatible` 支持自定义提供者 `baseURL` 与 OpenAI 兼容提供者。为接入 Venice，无需修改 OpenCode 核心提供者代码。
</Info>
