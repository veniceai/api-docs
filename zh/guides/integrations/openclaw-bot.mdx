[OpenClaw](https://openclaw.ai) 是一个开源、自托管的 AI 网关，可将消息平台（WhatsApp、Telegram、Discord、iMessage、Slack）连接到 AI 模型。Venice AI 作为内置提供商可用，让您可以从任何连接的渠道访问私有且无审查的模型。

## 设置

### 1. 安装 OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. 运行入门向导

```bash
openclaw onboard
```

向导将引导您完成设置。出现提示时，从列表中选择 **Venice AI** 作为您的提供商，然后粘贴您的 API 密钥。如果您还没有密钥，可从 [venice.ai/settings/api](https://venice.ai/settings/api) 获取。

### 3. 选择模型

在入门过程中，OpenClaw 会显示所有可用的 Venice 模型。一些推荐：

| 用例 | 模型 | 隐私 |
|---|---|---|
| 通用 | `venice/zai-org-glm-5` | Private |
| 推理 | `venice/kimi-k2-5` | Private |
| 编码 | `venice/claude-opus-4-6` | Anonymized |
| 视觉 | `venice/qwen3-vl-235b-a22b` | Private |
| 无审查 | `venice/venice-uncensored` | Private |

随时更改默认模型：

```bash
openclaw models set venice/zai-org-glm-5
```

列出所有可用模型：

```bash
openclaw models list | grep venice
```

### 4. 开始聊天

打开终端 UI：

```bash
openclaw tui
```

或 Web 仪表板：

```bash
openclaw dashboard
```

或连接消息渠道（WhatsApp、Telegram、Discord 等）：

```bash
openclaw channels login
openclaw gateway
```

## 隐私模式

OpenClaw 中的 Venice 模型遵循与 Venice API 相同的[隐私等级](/overview/privacy)：

- **Private** 模型（GLM、Qwen、DeepSeek、Llama、Venice Uncensored）在 Venice 的 GPU 集群上运行。Prompt 永远不会被存储或记录。
- **Anonymized** 模型（Claude、GPT、Gemini、Grok）通过 Venice 代理，并剥离所有身份信息。第三方提供商看到的客户是 Venice，而不是您。

## 图像和视频生成

安装 Venice AI Media skill 以进行图像和视频生成：

```bash
openclaw skills install nhannah/venice-ai-media
```
