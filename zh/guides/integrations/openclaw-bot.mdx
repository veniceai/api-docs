[OpenClaw](https://openclaw.ai) 是一个开源、自托管的 AI 网关，可将消息平台（WhatsApp、Telegram、Discord、iMessage、Slack）连接到 AI 模型。Venice AI 作为内置提供者可用，让你从任意接入渠道访问私有与未审查模型。

## 设置

### 1. 安装 OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. 运行引导向导

```bash
openclaw onboard
```

向导会带你完成设置。当出现选择时，从列表中选择 **Venice AI** 作为提供者，然后粘贴你的 API 密钥。如果尚未获取，请前往 [venice.ai/settings/api](https://venice.ai/settings/api)。

### 3. 选择模型

在引导过程中，OpenClaw 会显示所有可用 Venice 模型。以下是一些推荐：

| 用例 | 模型 | 隐私 |
|---|---|---|
| 通用 | `venice/zai-org-glm-5` | Private |
| 推理 | `venice/kimi-k2-5` | Private |
| 编码 | `venice/claude-opus-4-6` | Anonymized |
| 视觉 | `venice/qwen3-vl-235b-a22b` | Private |
| 未审查 | `venice/venice-uncensored` | Private |

随时更改默认模型：

```bash
openclaw models set venice/zai-org-glm-5
```

列出所有可用模型：

```bash
openclaw models list | grep venice
```

### 4. 开始聊天

打开终端 UI：

```bash
openclaw tui
```

或打开网页控制台：

```bash
openclaw dashboard
```

或接入一个消息渠道（WhatsApp、Telegram、Discord 等）：

```bash
openclaw channels login
openclaw gateway
```

## 隐私模式

OpenClaw 中的 Venice 模型遵循与 Venice API 相同的[隐私层级](/zh/overview/privacy)：

- **Private** 模型（GLM、Qwen、DeepSeek、Llama、Venice Uncensored）运行在 Venice 的 GPU 集群上。提示永不存储或记录。
- **Anonymized** 模型（Claude、GPT、Gemini、Grok）通过 Venice 进行代理，所有可识别信息均被去除。第三方提供商看到的是 Venice 作为客户，而不是你。

## 图像与视频生成

安装 Venice AI Media 技能以进行图像与视频生成：

```bash
openclaw skills install nhannah/venice-ai-media
```
