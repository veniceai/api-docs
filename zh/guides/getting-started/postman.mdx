## 概述

Venice 提供了一个全面的 Postman 集合，允许开发者探索和测试我们 API 的完整功能。该集合包含预配置的请求、示例和环境变量，帮助您快速上手 Venice 的 AI 服务。

## 访问集合

我们的官方 Postman 集合可在 Venice AI Workspace 中获取：

* [Venice AI Postman Workspace](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman Examples](https://postman.venice.ai/)

## 集合功能

- **即用即调用的请求**：为所有 Venice 端点预配置的 API 调用
- **环境模板**：结构完善的环境变量
- **请求示例**：每个端点的真实使用场景示例
- **响应样例**：示例响应，帮助您理解 API 的输出
- **文档**：每个请求的内嵌文档

## 入门

<Steps>
   <Step title="Fork 集合">
      - 前往 Venice AI Workspace
      - 点击 "Fork" 创建集合的副本
      - 选择您的目标工作区
   </Step>

   <Step title="设置环境">
      - 在 Postman 中创建一个新环境
      - 添加您的 Venice API 密钥
      - 配置 base URL：`https://api.venice.ai/api/v1`
   </Step>

   <Step title="发起首个请求">
      - 从集合中选择任意请求
      - 确保已选择您的环境
      - 点击 "Send" 测试 API
   </Step>
</Steps>

## 可用端点

该集合包含所有 Venice API 端点的示例：

- 文本生成
- 图像生成
- 模型信息
- 图像放大
- 系统 prompt 配置

## 最佳实践

- 妥善保管您的 API 密钥，切勿分享
- 对敏感信息使用环境变量
- 在实际实现前，在 Postman 控制台中测试响应
- 查看示例响应了解期望的数据结构

<Note>*注意：Postman 集合会定期更新，以反映最新的 API 变更和功能。*</Note>
