## 概述

Venice 提供了一个完整的 Postman 集合，方便开发者探索和测试我们 API 的全部能力。该集合包含预配置的请求、示例与环境变量，帮助你快速上手 Venice 的 AI 服务。

## 访问集合

我们的官方 Postman 集合位于 Venice AI 工作区：

* [Venice AI Postman 工作区](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman 示例](https://postman.venice.ai/)

## 集合特性

- **开箱即用的请求**：覆盖所有 Venice 端点的预配置 API 调用
- **环境模板**：结构良好的环境变量
- **请求示例**：每个端点的实际使用示例
- **响应样例**：示例响应，帮助你理解 API 的输出
- **文档**：每个请求都附带内联文档

## 快速开始

<Steps>
   <Step title="Fork 集合">
      - 进入 Venice AI 工作区
      - 点击 "Fork" 创建你自己的集合副本
      - 选择目标工作区
   </Step>

   <Step title="设置环境">
      - 在 Postman 中创建一个新环境
      - 添加你的 Venice API 密钥
      - 配置 base URL：`https://api.venice.ai/api/v1`
   </Step>

   <Step title="发出你的第一个请求">
      - 从集合中选择任意请求
      - 确认已选择正确的环境
      - 点击 "Send" 测试该 API
   </Step>
</Steps>

## 可用端点

集合中包含所有 Venice API 端点的示例：

- 文本生成
- 图像生成
- 模型信息
- 图像放大
- 系统提示配置

## 最佳实践

- 妥善保管 API 密钥，切勿分享
- 使用环境变量存放敏感信息
- 在实施前先在 Postman 控制台测试响应
- 查看示例响应以了解预期的数据结构

<Note>*注意：Postman 集合会定期更新，以反映最新的 API 变更和功能。*</Note>
