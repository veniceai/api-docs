Venice 的 API 通过 API 密钥进行保护。要开始使用 Venice API，您需要先生成一个新密钥。请按以下步骤开始。

<Steps>
  <Step title="访问 API 设置页面">
    访问 https://venice.ai/settings/api 进入 API 设置页面。您也可以点击左侧工具栏的 "API"，或在用户设置中点击 "API" 进入此页面。

    在该仪表板中，您可以查看 Diem 和 USD 余额、API 等级、API 用量以及您的 API 密钥。

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API Overview" />
    </Frame>
  </Step>

  <Step title="点击 Generate New API Key">
    向下滚动仪表板，选择 "Generate New API Key"。您将看到一系列选项。

    * **Description：** 用于命名您的 API 密钥

    * **API Key Type：**

      * "Admin" 密钥具有以编程方式删除或生成其他 API 密钥的能力。

      * "Inference Only" 密钥仅允许运行推理。

    * **Expires at：** 您可以为 API 密钥设置过期时间，过期后该密钥将无法使用。默认情况下不设置日期，密钥将永久有效。

    * **Epoch Consumption Limits：** 允许您为单个 API 密钥的使用设置限额。您可以选择限制在给定 epoch（24 小时）内可用的 Diem 或 USD 数量。

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Generate New API Key" />
    </Frame>
  </Step>

  <Step title="生成密钥">
    点击 Generate 将显示 API 密钥。

    <Warning>
      **重要：** 此密钥仅显示一次。请务必复制并保存在安全的地方。如果丢失，您需要删除它并重新创建。
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Your API Key" />
    </Frame>
  </Step>
</Steps>
