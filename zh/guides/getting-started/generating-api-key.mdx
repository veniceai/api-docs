Venice 的 API 通过 API 密钥进行保护。要开始使用 Venice API，你首先需要生成一个新的密钥。按照以下步骤开始。

<Steps>
  <Step title="访问 API 设置页面">
    访问 https://venice.ai/settings/api 进入 API 设置页面。也可以通过点击左侧工具栏中的 "API"，或在用户设置中点击 "API" 进入此页面。

    在此控制台中，你可以查看自己的 Diem 与 USD 余额、API 等级、API 使用情况以及 API 密钥。

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API 概览" />
    </Frame>
  </Step>

  <Step title="点击生成新的 API 密钥">
    向下滚动控制台并选择 "Generate New API Key"。系统会展示一系列选项。

    * **描述（Description）：** 用于为 API 密钥命名。

    * **API 密钥类型（API Key Type）：**

      * "Admin" 密钥具备以编程方式删除或生成其他 API 密钥的权限。

      * "Inference Only" 密钥仅允许执行推理。

    * **过期时间（Expires at）：** 你可以选择为 API 密钥设置过期时间，到期后密钥将停止工作。默认不设置日期，密钥永久有效。

    * **周期消费上限（Epoch Consumption Limits）：** 你可以为单个 API 密钥设置 API 使用上限。可在给定周期（24 小时）内限制可使用的 Diem 或 USD 金额。

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="生成新的 API 密钥" />
    </Frame>
  </Step>

  <Step title="生成密钥">
    点击 Generate 后将显示 API 密钥。

    <Warning>
      **重要：** 此密钥仅显示一次。请务必复制并保存在安全位置。如果丢失，你需要删除该密钥并创建一个新的。
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="你的 API 密钥" />
    </Frame>
  </Step>
</Steps>
