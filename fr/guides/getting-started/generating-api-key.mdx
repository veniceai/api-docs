L'API de Venice est protégée par des clés API. Pour commencer à utiliser l'API Venice, vous devez d'abord générer une nouvelle clé. Suivez ces étapes pour démarrer.

<Steps>
  <Step title="Accédez à la page des paramètres d'API">
    Pour atteindre la page des paramètres d'API, rendez-vous sur https://venice.ai/settings/api. Cette page est accessible en cliquant sur "API" dans la barre d'outils de gauche, ou en cliquant sur "API" dans les paramètres utilisateur.

    Dans ce tableau de bord, vous pouvez consulter vos soldes Diem et USD, votre palier d'API, votre utilisation d'API et vos clés API.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="Aperçu de l'API" />
    </Frame>
  </Step>

  <Step title="Cliquez sur Generate New API Key">
    Faites défiler le tableau de bord et sélectionnez "Generate New API Key". Une liste d'options vous sera présentée.

    * **Description :** sert à nommer votre clé API

    * **Type de clé API :**

      * Les clés "Admin" ont la capacité de supprimer ou de générer des clés API supplémentaires de manière programmatique.

      * Les clés "Inference Only" ne sont autorisées qu'à exécuter de l'inférence.

    * **Expires at :** vous pouvez choisir de définir une date d'expiration pour la clé API, après laquelle elle cessera de fonctionner. Par défaut, aucune date n'est définie, et la clé fonctionnera indéfiniment.

    * **Limites de consommation par epoch :** ceci vous permet de définir des limites d'utilisation d'API pour chaque clé. Vous pouvez choisir de limiter le montant Diem ou USD autorisé pendant une epoch (24 h) donnée.

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Générer une nouvelle clé API" />
    </Frame>
  </Step>

  <Step title="Générez la clé">
    Cliquer sur Generate affichera la clé API.

    <Warning>
      **Important :** cette clé n'est affichée qu'une seule fois. Assurez-vous de la copier et de la stocker dans un endroit sûr. Si vous la perdez, vous devrez la supprimer et en créer une nouvelle.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Votre clé API" />
    </Frame>
  </Step>
</Steps>
