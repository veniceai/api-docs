L'API de Venice est protégée par des clés API. Pour commencer à utiliser l'API Venice, vous devez d'abord générer une nouvelle clé. Suivez ces étapes pour démarrer.

<Steps>
  <Step title="Visitez la page des paramètres API">
    Pour vous rendre sur la page des paramètres API, visitez https://venice.ai/settings/api. Cette page est accessible en cliquant sur « API » dans la barre d'outils de gauche, ou en cliquant sur « API » dans vos paramètres utilisateur.

    Dans ce tableau de bord, vous pouvez consulter vos soldes Diem et USD, votre niveau d'API, votre utilisation de l'API et vos clés API.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="Vue d'ensemble de l'API" />
    </Frame>
  </Step>

  <Step title="Cliquez sur Générer une nouvelle clé API">
    Faites défiler le tableau de bord et sélectionnez « Generate New API Key ». Une liste d'options vous sera présentée.

    * **Description :** Sert à nommer votre clé API

    * **Type de clé API :**

      * Les clés « Admin » ont la capacité de supprimer ou générer des clés API supplémentaires de manière programmatique.

      * Les clés « Inference Only » ne sont autorisées qu'à exécuter de l'inférence.

    * **Expire le :** Vous pouvez choisir de définir une date d'expiration pour la clé API après laquelle elle cessera de fonctionner. Par défaut, aucune date n'est définie et la clé fonctionnera indéfiniment.

    * **Limites de consommation par epoch :** Cela vous permet de créer des limites pour l'utilisation de l'API depuis la clé API individuelle. Vous pouvez choisir de limiter le montant en Diem ou en USD autorisé dans un epoch donné (24 h).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Générer une nouvelle clé API" />
    </Frame>
  </Step>

  <Step title="Générez la clé">
    En cliquant sur Generate, la clé API vous sera affichée.

    <Warning>
      **Important :** Cette clé n'est affichée qu'une seule fois. Assurez-vous de la copier et de la stocker dans un endroit sûr. Si vous la perdez, vous devrez la supprimer et en créer une nouvelle.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Votre clé API" />
    </Frame>
  </Step>
</Steps>
