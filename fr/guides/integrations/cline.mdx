[Cline](https://cline.bot/) est un assistant de codage IA pour VS Code qui peut lire et éditer des fichiers, exécuter des commandes et accomplir des tâches de développement en plusieurs étapes. Venice fonctionne via les paramètres du fournisseur compatible OpenAI de Cline.

## Prérequis

- Une clé API Venice depuis [venice.ai/settings/api](https://venice.ai/settings/api)
- [Visual Studio Code](https://code.visualstudio.com/) installé
- L'[extension Cline](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev) installée dans VS Code

---

## Configuration

<Steps>
  <Step title="Ouvrir les paramètres de Cline">
    Sélectionnez l'icône Cline dans la barre d'activité de VS Code, puis sélectionnez l'icône d'engrenage pour ouvrir les paramètres de Cline.
  </Step>

  <Step title="Choisir le fournisseur d'API">
    Définissez **API Provider** sur **OpenAI Compatible**.
  </Step>

  <Step title="Configurer Venice">
    Entrez les valeurs suivantes :

    | Paramètre | Valeur |
    | --- | --- |
    | Base URL | `https://api.venice.ai/api/v1` |
    | API Key | Votre clé API Venice |
    | Model ID | Un ID de modèle depuis le [catalogue de modèles de texte](/models/text) |

    Par exemple, utilisez `zai-org-glm-5-1` comme Model ID.

    Entrez la clé API elle-même, sans préfixe `Bearer`. Conservez la Base URL sur `/api/v1` ; Cline ajoute le chemin `/chat/completions`.
  </Step>

  <Step title="Configurer les capacités du modèle">
    Développez **Model Configuration** et définissez la fenêtre de contexte et le nombre maximum de tokens en sortie pour correspondre au modèle sélectionné. Activez les capacités prises en charge par le modèle, notamment l'entrée d'image ou l'utilisation de l'ordinateur le cas échéant.

    Vérifiez le modèle sélectionné dans le [catalogue de modèles de texte](/models/text) avant d'activer des capacités optionnelles.
  </Step>

  <Step title="Enregistrer et tester">
    Enregistrez les paramètres du fournisseur, ouvrez un projet et confiez à Cline une petite tâche telle que :

    ```txt
    Review the current file and suggest one safe improvement.
    ```

    Passez en revue chaque autorisation de fichier ou de commande avant de l'approuver.
  </Step>
</Steps>

## Vérifier l'API Venice

Si Cline ne parvient pas à se connecter, vérifiez indépendamment la clé API et les ID de modèles disponibles :

```bash
curl https://api.venice.ai/api/v1/models \
  -H "Authorization: Bearer $VENICE_API_KEY"
```

Copiez le `id` d'un modèle exactement dans le champ **Model ID** de Cline.

---

## Notes de configuration

- Utilisez `https://api.venice.ai/api/v1` comme Base URL, pas l'endpoint complet `/chat/completions`.
- Les ID de modèles peuvent évoluer au fil du temps. Utilisez le [catalogue de modèles de texte](/models/text) ou `GET /models` plutôt que de vous fier à un ancien exemple.
- Les paramètres de fenêtre de contexte et de tokens en sortie de Cline affectent la façon dont il budgète les requêtes. Faites-les correspondre au modèle sélectionné pour obtenir les meilleurs résultats.
- Gardez votre clé API Venice privée. Ne la collez pas dans des fichiers source et ne la committez pas dans git.

## Dépannage

<AccordionGroup>
  <Accordion title="Erreur 401 ou d'authentification">
    Vérifiez que la clé API provient de [venice.ai/settings/api](https://venice.ai/settings/api). Collez la clé brute dans Cline sans `Bearer` ni espaces superflus.
  </Accordion>
  <Accordion title="Modèle introuvable">
    Vérifiez que le Model ID apparaît dans le [catalogue de modèles de texte](/models/text) ou dans la réponse de `GET /models`, puis copiez-le exactement.
  </Accordion>
  <Accordion title="Erreur de connexion ou d'endpoint">
    Vérifiez que **API Provider** est **OpenAI Compatible** et que la Base URL est exactement `https://api.venice.ai/api/v1`.
  </Accordion>
  <Accordion title="Cline ne peut pas utiliser les outils ou éditer des fichiers">
    Ouvrez **Model Configuration**, vérifiez que le modèle sélectionné prend en charge les capacités requises et activez l'utilisation de l'ordinateur si Cline expose cette option pour le modèle.
  </Accordion>
</AccordionGroup>
