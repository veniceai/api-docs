[Cursor](https://www.cursor.com/) est un éditeur de code alimenté par IA. Vous pouvez l'utiliser avec Venice AI pour un accès privé et non censuré à une large gamme de modèles.

## Configuration

<Steps>
  <Step title="Obtenez votre clé API">
    Générez une clé depuis [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="Configurer Cursor">
    Ouvrez **Cursor Settings** (icône d'engrenage), faites défiler jusqu'à **Models** et cliquez sur **Add Model**.

    Sous **OpenAI Compatible**, entrez :
    - **Override OpenAI Base URL** : `https://api.venice.ai/api/v1`
    - **OpenAI API Key** : Votre clé API Venice
  </Step>

  <Step title="Ajouter des modèles">
    Dans le champ du nom du modèle, tapez l'ID du modèle Venice et appuyez sur Entrée. Ajoutez chaque modèle que vous souhaitez utiliser. Par exemple :
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (voir la [note sur le préfixe ci-dessous](#the-venice--model-prefix))

    Consultez le [catalogue de modèles](/models/text) pour tous les ID de modèles disponibles.
  </Step>

  <Step title="Sélectionner et vérifier">
    Ouvrez un nouveau chat et utilisez le **sélecteur de modèle en bas du chat** pour choisir l'un des modèles Venice que vous venez d'ajouter. Envoyez un message test. Si vous obtenez une réponse, tout est prêt.
  </Step>
</Steps>

## Le préfixe de modèle `venice-`

<Warning>
**Les modèles Claude nécessitent le préfixe `venice-` lorsqu'ils sont utilisés dans Cursor.**

Cursor réécrit les requêtes pour les modèles `claude-*` dans le format natif d'Anthropic, qui est incompatible avec Venice. Préfixer l'ID du modèle avec `venice-` empêche cette réécriture. Venice supprime automatiquement le préfixe.
</Warning>

| Modèle | ID standard | ID Cursor |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
Les modèles non-Claude (par exemple `minimax-m25`, `kimi-k2-5`, `zai-org-glm-5`) **ne sont pas affectés** et fonctionnent sans le préfixe. Le préfixe `venice-` est sans danger à utiliser sur n'importe quel modèle puisque Venice le supprime toujours, mais il n'est requis que pour les modèles Claude dans Cursor.
</Info>
