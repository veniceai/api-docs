[OpenClaw](https://openclaw.ai) est une passerelle IA open source auto-hébergée qui connecte les plateformes de messagerie (WhatsApp, Telegram, Discord, iMessage, Slack) aux modèles IA. Venice AI est disponible en tant que fournisseur intégré, vous donnant accès à des modèles privés et non censurés depuis n'importe quel canal connecté.

## Configuration

### 1. Installer OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. Exécuter l'assistant d'intégration

```bash
openclaw onboard
```

L'assistant vous guidera dans la configuration. Lorsqu'on vous le demande, sélectionnez **Venice AI** comme fournisseur dans la liste, puis collez votre clé API. Obtenez-en une depuis [venice.ai/settings/api](https://venice.ai/settings/api) si vous n'en avez pas encore.

### 3. Choisir un modèle

Pendant l'intégration, OpenClaw affiche tous les modèles Venice disponibles. Quelques recommandations :

| Cas d'usage | Modèle | Confidentialité |
|---|---|---|
| Général | `venice/zai-org-glm-5` | Private |
| Raisonnement | `venice/kimi-k2-5` | Private |
| Codage | `venice/claude-opus-4-6` | Anonymized |
| Vision | `venice/qwen3-vl-235b-a22b` | Private |
| Non censuré | `venice/venice-uncensored` | Private |

Changez votre modèle par défaut à tout moment :

```bash
openclaw models set venice/zai-org-glm-5
```

Listez tous les modèles disponibles :

```bash
openclaw models list | grep venice
```

### 4. Commencer à discuter

Ouvrez l'interface terminal :

```bash
openclaw tui
```

Ou le tableau de bord web :

```bash
openclaw dashboard
```

Ou connectez un canal de messagerie (WhatsApp, Telegram, Discord, etc.) :

```bash
openclaw channels login
openclaw gateway
```

## Modes de confidentialité

Les modèles Venice dans OpenClaw suivent les mêmes [niveaux de confidentialité](/overview/privacy) que l'API Venice :

- Les modèles **Private** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) s'exécutent sur la flotte GPU de Venice. Les prompts ne sont jamais stockés ni journalisés.
- Les modèles **Anonymized** (Claude, GPT, Gemini, Grok) sont transmis via Venice avec toutes les informations d'identification supprimées. Le fournisseur tiers voit Venice comme le client, pas vous.

## Génération d'image et de vidéo

Installez la skill Venice AI Media pour la génération d'image et de vidéo :

```bash
openclaw skills install nhannah/venice-ai-media
```
