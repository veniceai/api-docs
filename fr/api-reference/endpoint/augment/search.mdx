<Warning>Cette API est expérimentale. Le format de la requête et de la réponse peut changer sans préavis.</Warning>

Envoyez une requête de recherche dans le champ `query`. L'API renvoie des résultats structurés comprenant les titres, URL, extraits de contenu et dates.

**Fournisseurs de recherche :**
- `brave` (par défaut) — Brave Search avec Zero Data Retention (ZDR). Les requêtes de recherche ne sont jamais stockées ni enregistrées par le fournisseur de recherche.
- `google` — Google Search avec des requêtes anonymisées. Les recherches transitent par l'infrastructure de Venice afin que votre identité ne soit pas associée à la requête envoyée à Google. Venice ne stocke ni n'enregistre les requêtes de recherche.

**Tarification :** 0,01 $ par requête.

### Exemple (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/search \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "dernières actualités sur l''IA"}'
```
