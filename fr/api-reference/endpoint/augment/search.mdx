<Warning>Il s'agit d'une API expérimentale. Le format des requêtes et des réponses peut évoluer sans préavis.</Warning>

Envoyez une requête de recherche dans le champ `query`. L'API renvoie des résultats structurés comprenant les titres, URLs, extraits de contenu et dates.

**Fournisseurs de recherche :**
- `brave` (par défaut) — Brave Search avec Zero Data Retention (ZDR). Les requêtes de recherche ne sont jamais stockées ni journalisées par le fournisseur de recherche.
- `google` — Google Search avec requêtes anonymisées. Les recherches transitent par l'infrastructure de Venice afin que votre identité ne soit pas associée à la requête de recherche envoyée à Google. Venice ne stocke ni ne journalise les requêtes de recherche.

**Tarification :** 0,01 $ par requête.

### Exemple (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/search \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "latest news about AI"}'
```
