<Warning>Cette API est expérimentale. Le format de la requête et de la réponse peut changer sans préavis.</Warning>

Envoyez une URL publiquement accessible dans le champ `url`. L'API renvoie le contenu de la page en **markdown**.

Le scraper essaie d'abord la prise en charge native du markdown du site cible (via [Cloudflare Markdown for Agents](https://blog.cloudflare.com/markdown-for-agents/)), puis se rabat sur une extraction par navigateur headless. Certains sites qui bloquent l'accès automatisé (par exemple X/Twitter, Reddit) sont rejetés immédiatement avec une erreur `400`.

**Tarification :** 0,01 $ par requête.

### Exemple (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/scrape \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```
