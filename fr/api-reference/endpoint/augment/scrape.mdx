<Warning>Il s'agit d'une API expérimentale. Le format des requêtes et des réponses peut évoluer sans préavis.</Warning>

Envoyez une URL publiquement accessible dans le champ `url`. L'API renvoie le contenu de la page au format **markdown**.

Le scraper essaie d'abord la prise en charge native du markdown par le site cible (via [Cloudflare Markdown for Agents](https://blog.cloudflare.com/markdown-for-agents/)), puis se rabat sur une extraction par navigateur headless. Certains sites qui bloquent les accès automatisés (par ex. X/Twitter, Reddit) sont rejetés immédiatement avec une erreur `400`.

**Tarification :** 0,01 $ par requête.

### Exemple (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/scrape \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```
