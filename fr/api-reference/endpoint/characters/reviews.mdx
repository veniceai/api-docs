## Endpoint expérimental

<Warning>
Il s'agit d'un endpoint expérimental qui peut être sujet à modification.
</Warning>

## Ce que renvoie cet endpoint

Cet endpoint renvoie les avis publics paginés pour un personnage unique.

- Utilisez le paramètre de chemin `slug` pour identifier le personnage.
- Utilisez les paramètres de requête `page` et `pageSize` pour paginer les avis.
- Les métadonnées de pagination sont renvoyées à la fois dans le corps de la réponse et dans les en-têtes de réponse `x-pagination-*`.
