Appelez `/audio/quote` pour estimer le coût, puis interrogez `/audio/retrieve` avec le `queue_id` renvoyé jusqu'à la fin de la génération. Si vous conservez les médias générés après récupération, appelez `/audio/complete` une fois que vous les avez téléchargés.
