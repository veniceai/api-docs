Renvoie la liste triée par ordre alphabétique des slugs de réseau pris en charge par le proxy RPC crypto de Venice. Utilisez ceci pour découvrir quelles valeurs `:network` sont valides pour `POST /crypto/rpc/{network}`.

Cet endpoint est **public** — aucune clé API ni authentification par portefeuille requise.

## Format de la réponse

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

La liste fait autorité — si un slug n'y figure pas, l'endpoint proxy renvoie `400 Unsupported RPC network`.
