Venice prend en charge des capacités supplémentaires dans ses modèles qui peuvent être activées via l'entrée `venice_parameters` sur l'endpoint chat completions.

Dans certaines circonstances, vous pouvez utiliser un client qui ne vous permet pas de modifier le corps de la requête. Pour ces plateformes, vous pouvez utiliser l'offre de suffixe de fonctionnalité de modèle de Venice pour transmettre des flags via l'ID du modèle.

## Syntaxe

Le suffixe de fonctionnalité de modèle suit ce schéma :

```
<model_id>:<parameter>=<value>
```

Pour plusieurs paramètres, chaînez-les avec `&` :

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## Exemples

### Définir la recherche web sur Auto
```
default:enable_web_search=auto
```

### Activer la recherche web et désactiver le prompt système
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### Activer la recherche web et ajouter des citations à la réponse
```
default:enable_web_search=on&enable_web_citations=true
```

### Activer la recherche web avec scraping complet de page
```
default:enable_web_search=on&enable_web_scraping=true
```

### Utiliser un personnage
```
default:character_slug=alan-watts
```

### Masquer les blocs de réflexion sur la réponse d'un modèle de raisonnement
```
qwen3-4b:strip_thinking_response=true
```

### Désactiver la réflexion sur les modèles de raisonnement pris en charge
Certains modèles de raisonnement (comme Qwen 3) prennent en charge la désactivation du processus de réflexion. Vous pouvez l'activer en utilisant le suffixe ci-dessous :
```
qwen3-4b:disable_thinking=true
```

### Ajouter les résultats de recherche web à une réponse en streaming
Ceci active la recherche web, ajoute des citations au corps de la réponse et inclut les résultats de recherche dans le stream en tant que message de réponse final.

Vous pouvez voir un exemple de ceci dans notre [collection Postman ici](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041).

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
