Exporte les données d'utilisation d'un utilisateur. Les descriptions des champs de réponse sont indiquées ci-dessous :

- **timestamp** : L'horodatage auquel l'entrée d'utilisation de facturation a été créée
- **sku** : Le produit associé à l'entrée d'utilisation de facturation
- **pricePerUnitUsd** : Le prix unitaire en USD
- **unit** : Le nombre d'unités consommées
- **amount** : Le montant total facturé pour l'entrée d'utilisation de facturation
- **currency** : La devise facturée pour l'entrée d'utilisation de facturation
- **notes** : Notes concernant l'entrée d'utilisation de facturation
- **inferenceDetails.requestId** : L'identifiant de requête associé à l'inférence
- **inferenceDetails.inferenceExecutionTime** : Temps d'exécution de l'inférence en millisecondes
- **inferenceDetails.promptTokens** : Nombre de tokens demandés dans le prompt. Présent uniquement pour l'utilisation des LLM.
- **inferenceDetails.completionTokens** : Nombre de tokens utilisés dans la complétion. Présent uniquement pour l'utilisation des LLM.
