Exporte les données d'utilisation pour un utilisateur. Les descriptions des champs de la réponse sont fournies ci-dessous :

- **timestamp** : L'horodatage de création de l'entrée d'utilisation de facturation
- **sku** : Le produit associé à l'entrée d'utilisation de facturation
- **pricePerUnitUsd** : Le prix par unité en USD
- **unit** : Le nombre d'unités consommées
- **amount** : Le montant total facturé pour l'entrée d'utilisation de facturation
- **currency** : La devise facturée pour l'entrée d'utilisation de facturation
- **notes** : Notes concernant l'entrée d'utilisation de facturation
- **inferenceDetails.requestId** : L'ID de requête associé à l'inférence
- **inferenceDetails.inferenceExecutionTime** : Temps pris pour l'exécution de l'inférence en millisecondes
- **inferenceDetails.promptTokens** : Nombre de tokens demandés dans le prompt. Uniquement présent pour l'utilisation des LLM.
- **inferenceDetails.completionTokens** : Nombre de tokens utilisés dans la complétion. Uniquement présent pour l'utilisation des LLM.
