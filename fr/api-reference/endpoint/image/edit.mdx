<Info>
**Tarification :** Le prix de l'édition d'image varie selon le modèle. Le modèle par défaut (`qwen-edit`) coûte **0,04 $ par édition**. Voir l'[endpoint Models](/api-reference/endpoint/models/list) pour tous les modèles d'édition disponibles et leur tarification.
</Info>

## Niveaux de qualité

Certains modèles d'édition acceptent un paramètre `quality` optionnel (`low`, `medium`, `high`) qui arbitre entre fidélité visuelle et coût. Actuellement pris en charge par `gpt-image-2-edit` ; les autres modèles ignorent ce paramètre.

```json
{
  "model": "gpt-image-2-edit",
  "image": "iVBORw0KGgo...",
  "prompt": "change the sky to a sunrise",
  "resolution": "2K",
  "quality": "medium"
}
```

Lorsque vous omettez `quality`, le modèle utilise son niveau par défaut (`high` pour `gpt-image-2-edit`). Les prix par niveau (1K/2K/4K × low/medium/high) se trouvent sous `model_spec.pricing.quality` dans l'[endpoint Models](/api-reference/endpoint/models/list) et sont listés dans l'[aperçu de la tarification](/overview/pricing).

<Info>
La tarification consciente de la qualité est la valeur par défaut pour tous les appelants de l'API et du SDK — aucun en-tête d'opt-in requis. Les requêtes qui omettent `quality` sont facturées au niveau par défaut du modèle (`high` pour `gpt-image-2-edit`).
</Info>

Le modèle par défaut (`qwen-edit`) bloque les requêtes qui tentent de générer des images sexuelles explicites, de sexualiser des mineurs ou de représenter de la violence réelle. D'autres modèles peuvent avoir des politiques de contenu différentes.
