<Info>
**Tarification :** la tarification de l'édition d'image varie selon le modèle. Le modèle par défaut (`qwen-edit`) coûte **0,04 $ par édition**. Consultez l'[endpoint Models](/fr/api-reference/endpoint/models/list) pour tous les modèles d'édition disponibles et leurs prix.
</Info>

## Paliers de qualité

Certains modèles d'édition acceptent un paramètre optionnel `quality` (`low`, `medium`, `high`) qui arbitre entre fidélité visuelle et coût. Actuellement pris en charge par `gpt-image-2-edit` ; les autres modèles ignorent ce paramètre.

```json
{
  "model": "gpt-image-2-edit",
  "image": "iVBORw0KGgo...",
  "prompt": "change the sky to a sunrise",
  "resolution": "2K",
  "quality": "medium"
}
```

Lorsque vous omettez `quality`, le modèle utilise son palier par défaut (`high` pour `gpt-image-2-edit`). Les prix par palier (1K/2K/4K × low/medium/high) se trouvent sous `model_spec.pricing.quality` dans l'[endpoint Models](/fr/api-reference/endpoint/models/list) et sont listés dans l'[aperçu de la tarification](/fr/overview/pricing).

<Info>
La tarification tenant compte de la qualité est la valeur par défaut pour tous les appelants API et SDK — aucun en-tête d'opt-in requis. Les requêtes qui omettent `quality` sont facturées au palier par défaut du modèle (`high` pour `gpt-image-2-edit`).
</Info>

<Warning>
Le modèle par défaut (`qwen-edit`) bloque les requêtes qui tentent de générer des images à caractère sexuel explicite, de sexualiser des mineurs ou de représenter de la violence du monde réel. D'autres modèles peuvent avoir des politiques de contenu différentes.
</Warning>
