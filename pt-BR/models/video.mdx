## Tipos de modelo

**Text to Video:** Gere vídeos a partir de prompts de texto

**Image to Video:** Anime imagens estáticas em clipes de vídeo

**Upscaling de vídeo:** Aprimore vídeos existentes para resoluções mais altas usando upscaling com IA. Veja o [Guia de upscaling de vídeo](/guides/media/video-upscaling) para detalhes.

<Note>
Geração e upscaling de vídeo usam um sistema de filas assíncrono. Veja a [API de fila de vídeo](/api-reference/endpoint/video/queue) para iniciar a geração e a [API de recuperação de vídeo](/api-reference/endpoint/video/retrieve) para buscar os resultados.
</Note>

## Preços

Ajuste os dropdowns para ver como duração, resolução e áudio afetam o preço. Modelos marcados como **FIXED** têm uma tarifa fixa.

Para cotações exatas antes da geração, use a [API de cotação de vídeo](/api-reference/endpoint/video/quote).
