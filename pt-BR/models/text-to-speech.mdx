## Catálogo de vozes

As vozes são **específicas por modelo**. A `voice` que você passa deve vir do catálogo
do `model` que você selecionou. Escolha um modelo abaixo para navegar pelas suas vozes.

<div id="tts-voice-picker-placeholder">Loading voices...</div>

<Note>
Os IDs de voz são case-sensitive e **só são válidos para o `model` correspondente**. Passe
ambos os campos juntos no payload da sua requisição. Veja a
[API Audio Speech](/api-reference/endpoint/audio/speech) para exemplos.
</Note>

### Exemplo de requisição

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

Para trocar de modelo, altere **ambos** `model` e `voice` para um par do
modelo selecionado acima.
