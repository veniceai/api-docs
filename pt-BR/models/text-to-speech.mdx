## Catálogo de vozes

As vozes são **específicas por modelo**. A `voice` que você envia deve vir do catálogo
do `model` selecionado. Escolha um modelo abaixo para navegar pelas suas vozes.

<div id="tts-voice-picker-placeholder">Loading voices...</div>

<Note>
Os IDs de voz são case-sensitive e **válidos somente para o `model` correspondente**. Envie
ambos os campos juntos no payload da sua requisição. Veja a
[API Audio Speech](/pt-BR/api-reference/endpoint/audio/speech) para exemplos.
</Note>

### Exemplo de requisição

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

Para trocar de modelo, mude **ambos** `model` e `voice` para um par do
modelo selecionado acima.
