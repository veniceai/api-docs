Chame `/audio/quote` para estimar o custo, depois faça polling em `/audio/retrieve` com o `queue_id` retornado até que a geração seja concluída. Se você mantiver a mídia gerada após a recuperação, chame `/audio/complete` assim que tiver feito o download.
