<Warning>Esta é uma API experimental. O formato da requisição e da resposta pode mudar sem aviso prévio.</Warning>

Envie uma URL publicamente acessível no campo `url`. A API retorna o conteúdo da página como **markdown**.

O scraper tenta primeiro o suporte nativo a markdown do site de destino (via [Cloudflare Markdown for Agents](https://blog.cloudflare.com/markdown-for-agents/)) e, em seguida, recorre a uma extração com navegador headless. Alguns sites que bloqueiam acesso automatizado (por exemplo, X/Twitter, Reddit) são rejeitados imediatamente com um erro `400`.

**Preço:** US$ 0,01 por requisição.

### Exemplo (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/scrape \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```
