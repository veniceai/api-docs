Exporta dados de uso de um usuário. As descrições dos campos da resposta podem ser encontradas abaixo:

- **timestamp**: O carimbo de data/hora em que a entrada de uso de cobrança foi criada
- **sku**: O produto associado à entrada de uso de cobrança
- **pricePerUnitUsd**: O preço por unidade em USD
- **unit**: O número de unidades consumidas
- **amount**: O valor total cobrado pela entrada de uso de cobrança
- **currency**: A moeda cobrada pela entrada de uso de cobrança
- **notes**: Notas sobre a entrada de uso de cobrança
- **inferenceDetails.requestId**: O ID da requisição associada à inferência
- **inferenceDetails.inferenceExecutionTime**: Tempo necessário para a execução da inferência em milissegundos
- **inferenceDetails.promptTokens**: Número de tokens solicitados no prompt. Presente apenas para uso de LLM.
- **inferenceDetails.completionTokens**: Número de tokens usados na conclusão. Presente apenas para uso de LLM.
