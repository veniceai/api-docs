## Endpoint experimental

<Warning>
Este é um endpoint experimental e está sujeito a alterações.
</Warning>

## O que este endpoint retorna

Este endpoint retorna avaliações públicas paginadas para um único personagem.

- Use o parâmetro de path `slug` para identificar o personagem.
- Use os parâmetros de consulta `page` e `pageSize` para paginar as avaliações.
- Os metadados de paginação são retornados tanto no corpo da resposta quanto nos headers de resposta `x-pagination-*`.
