Retorna a lista de slugs de redes ordenados alfabeticamente que são suportados pelo proxy RPC cripto da Venice. Use isto para descobrir quais valores de `:network` são válidos para `POST /crypto/rpc/{network}`.

Este endpoint é **público** — não é necessária autenticação com chave de API ou carteira.

## Formato da resposta

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "robinhood-mainnet",
    "robinhood-testnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

A lista é autoritativa — se um slug não estiver aqui, o endpoint proxy retorna `400 Unsupported RPC network`.
