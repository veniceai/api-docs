**Preço:** O preço da edição de imagem varia por modelo. O modelo padrão (`qwen-edit`) é **$0,04 por edição**. Veja o [endpoint de Modelos](/pt-BR/api-reference/endpoint/models/list) para todos os modelos de edição disponíveis e seus preços.
</Info>

## Tiers de qualidade

Alguns modelos de edição aceitam um parâmetro opcional `quality` (`low`, `medium`, `high`) que troca fidelidade visual por custo. Atualmente é suportado por `gpt-image-2-edit`; outros modelos ignoram o parâmetro.

```json
{
  "model": "gpt-image-2-edit",
  "image": "iVBORw0KGgo...",
  "prompt": "change the sky to a sunrise",
  "resolution": "2K",
  "quality": "medium"
}
```

Quando você omite `quality`, o modelo usa seu tier padrão (`high` para `gpt-image-2-edit`). Os preços por tier (1K/2K/4K × low/medium/high) ficam em `model_spec.pricing.quality` no [endpoint de Modelos](/pt-BR/api-reference/endpoint/models/list) e estão listados na [Visão geral de preços](/pt-BR/overview/pricing).

<Info>
A precificação por qualidade é padrão para todos os chamadores da API e do SDK — nenhum header de opt-in é necessário. Requisições que omitem `quality` são cobradas no tier padrão do modelo (`high` para `gpt-image-2-edit`).
</Info>
