<Info>
**Preço:** O preço da edição de imagens varia por modelo. O modelo padrão (`qwen-edit`) custa **US$ 0,04 por edição**. Veja o [endpoint de Modelos](/api-reference/endpoint/models/list) para todos os modelos de edição disponíveis e seus preços.
</Info>

## Níveis de qualidade

Alguns modelos de edição aceitam um parâmetro opcional `quality` (`low`, `medium`, `high`) que troca fidelidade visual por custo. Atualmente suportado por `gpt-image-2-edit`; outros modelos ignoram o parâmetro.

```json
{
  "model": "gpt-image-2-edit",
  "image": "iVBORw0KGgo...",
  "prompt": "change the sky to a sunrise",
  "resolution": "2K",
  "quality": "medium"
}
```

Quando você omite `quality`, o modelo usa seu nível padrão (`high` para `gpt-image-2-edit`). Os preços por nível (1K/2K/4K × low/medium/high) ficam em `model_spec.pricing.quality` no [endpoint de Modelos](/api-reference/endpoint/models/list) e estão listados na [visão geral de preços](/overview/pricing).

<Info>
Preços baseados em qualidade são o padrão para todas as chamadas de API e SDK — não é necessário cabeçalho de opt-in. Requisições que omitem `quality` são cobradas pelo nível padrão do modelo (`high` para `gpt-image-2-edit`).
</Info>
