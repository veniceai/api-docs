A Venice oferece suporte a recursos adicionais em seus modelos que podem ser ativados pelo input `venice_parameters` no endpoint de chat completions.

Em determinadas circunstâncias, você pode estar usando um cliente que não permite modificar o corpo da requisição. Para essas plataformas, você pode utilizar a oferta de Sufixo de Recursos do Modelo da Venice para passar flags por meio do ID do modelo.

## Sintaxe

O Sufixo de Recursos do Modelo segue este padrão:

```
<model_id>:<parameter>=<value>
```

Para múltiplos parâmetros, encadeie-os com `&`:

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## Exemplos

### Para definir a busca na web como automática
```
default:enable_web_search=auto
```

### Para habilitar busca na web e desabilitar o prompt do sistema
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### Para habilitar busca na web e adicionar citações à resposta
```
default:enable_web_search=on&enable_web_citations=true
```

### Para habilitar busca na web com scraping completo da página
```
default:enable_web_search=on&enable_web_scraping=true
```

### Para usar um personagem
```
default:character_slug=alan-watts
```

### Para ocultar blocos de raciocínio em uma resposta de modelo de raciocínio
```
qwen3-4b:strip_thinking_response=true
```

### Para desabilitar o raciocínio em modelos de raciocínio compatíveis
Alguns modelos de raciocínio (como Qwen 3) oferecem suporte à desativação do processo de raciocínio. Você pode ativá-la usando o sufixo abaixo:
```
qwen3-4b:disable_thinking=true
```

### Para adicionar resultados de busca na web a uma resposta em streaming
Isso habilitará a busca na web, adicionará citações ao corpo da resposta e incluirá os resultados da busca no stream como a mensagem final da resposta.

Você pode ver um exemplo disso na nossa [coleção do Postman aqui](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041).

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
