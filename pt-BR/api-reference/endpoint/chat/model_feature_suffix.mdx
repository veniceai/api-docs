A Venice oferece suporte a recursos adicionais em seus modelos, que podem ser acionados pelo input `venice_parameters` no endpoint de chat completions.

Em certas circunstâncias, você pode estar usando um cliente que não permite modificar o corpo da requisição. Para essas plataformas, você pode utilizar o recurso Model Feature Suffix da Venice para passar flags através do ID do modelo.

## Sintaxe

O Model Feature Suffix segue este padrão:

```
<model_id>:<parameter>=<value>
```

Para múltiplos parâmetros, encadeie-os com `&`:

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## Exemplos

### Para definir Web Search como Auto
```
default:enable_web_search=auto
```

### Para habilitar Web Search e desabilitar o System Prompt
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### Para habilitar Web Search e adicionar citações à resposta
```
default:enable_web_search=on&enable_web_citations=true
```

### Para habilitar Web Search com scraping de página completa
```
default:enable_web_search=on&enable_web_scraping=true
```

### Para usar um personagem
```
default:character_slug=alan-watts
```

### Para ocultar os blocos de thinking na resposta de um modelo de reasoning
```
qwen3-4b:strip_thinking_response=true
```

### Para desabilitar o thinking em modelos de reasoning compatíveis
Certos modelos de reasoning (como Qwen 3) permitem desabilitar o processo de thinking. Você pode ativá-lo usando o sufixo abaixo:
```
qwen3-4b:disable_thinking=true
```

### Para adicionar resultados de Web Search a uma resposta em streaming
Isto habilita web search, adiciona citações ao corpo da resposta e inclui os resultados de busca no stream como a mensagem final de resposta.

Você pode ver um exemplo disto em nossa [Coleção do Postman aqui](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041).

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
