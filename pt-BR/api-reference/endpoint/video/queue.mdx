Chame `/video/quote` para obter uma estimativa de preço, depois faça polling em `/video/retrieve` com o `queue_id` retornado até a conclusão.

Modelos privados também retornam uma `download_url` para o vídeo finalizado. Trata-se de uma URL de entrega de curta duração (algumas retentativas são aceitáveis se um download cair); veja o [guia de Geração de Vídeo](/pt-BR/guides/media/video-generation#private-download-links) para detalhes e `DELETE` opcional para privacidade.

### Seedance 2.0

Para os modelos `seedance-2-0-*` (text-to-video, image-to-video, reference-to-video, além das variantes `-fast-*`), consulte o [Guia do Seedance 2.0](/pt-BR/guides/media/seedance-2-0) para o modelo de quatro workflows (Reference / Edit / Extend / Stitch), os padrões canônicos de prompt, os limites de entrada multimodal e os detalhes de preço.

### Consentimento Seedance

Quando uma requisição image- ou reference-to-video do Seedance contém um rosto humano, a API retorna `409 needs_consent` com o texto da política, e você reenvia a mesma requisição com uma atestação `consents.seedance`. Veja o [Guia de Consentimento de Mídia Facial Seedance](/pt-BR/guides/media/seedance-face-consent) para o fluxo completo, o objeto de consentimento, dedupe e revogação.

### Upscaling de vídeo

Para o modelo `topaz-video-upscale`, use `upscale_factor` (1, 2 ou 4) em vez de `resolution`, e forneça uma `video_url`. A duração e o FPS são detectados automaticamente a partir do arquivo de vídeo. Veja o [Guia de Upscaling de Vídeo](/pt-BR/guides/media/video-upscaling) para detalhes completos e exemplos.
