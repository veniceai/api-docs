Chame `/video/quote` para obter uma estimativa de preço e, em seguida, consulte `/video/retrieve` com o `queue_id` retornado até a conclusão.

Os modelos privados também retornam uma `download_url` para o vídeo finalizado. É uma URL de entrega de curta duração (algumas tentativas são aceitáveis se um download falhar); veja o [guia de geração de vídeo](/guides/media/video-generation#private-download-links) para detalhes e o `DELETE` opcional para privacidade.

### Seedance 2.0

Para modelos `seedance-2-0-*` (text-to-video, image-to-video, reference-to-video, além das variantes `-fast-*`), consulte o [Guia do Seedance 2.0](/guides/media/seedance-2-0) para o modelo de quatro workflows (Reference / Edit / Extend / Stitch), padrões canônicos de prompt, limites de entrada multimodal e detalhes de preço.

### Consentimento Seedance

Quando uma requisição image- ou reference-to-video do Seedance contém um rosto humano, a API retorna `409 needs_consent` com o texto da política, e você reenvia a mesma requisição com uma atestação `consents.seedance`. Veja o [Guia de consentimento para mídia facial do Seedance](/guides/media/seedance-face-consent) para o fluxo completo, o objeto de consentimento, deduplicação e revogação.

### Upscaling de vídeo

Para o modelo `topaz-video-upscale`, use `upscale_factor` (1, 2 ou 4) em vez de `resolution` e forneça uma `video_url`. A duração e o FPS são detectados automaticamente a partir do arquivo de vídeo. Veja o [Guia de upscaling de vídeo](/guides/media/video-upscaling) para detalhes completos e exemplos.
