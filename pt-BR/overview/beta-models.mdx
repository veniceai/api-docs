Às vezes lançamos modelos em beta para coletar feedback e confirmar seu desempenho antes de um lançamento completo em produção. Os modelos beta estão disponíveis para todos os usuários, mas **não são recomendados para uso em produção**.

O status beta não garante promoção a produção. Um modelo beta pode ser removido se for caro demais para executar, tiver desempenho ruim em escala ou levantar preocupações de segurança. Modelos beta podem mudar sem aviso prévio e podem ter documentação ou suporte limitados. Modelos que se mostram estáveis, amplamente úteis e alinhados aos nossos padrões são promovidos à disponibilidade geral.

## Considerações importantes

Ao usar modelos beta, tenha em mente:

- Podem ser alterados ou removidos a qualquer momento sem o período padrão de aviso de descontinuação
- Não são adequados para aplicações em produção ou fluxos críticos
- Podem apresentar desempenho, disponibilidade ou comportamento inconsistentes
- Suporte limitado ou inexistente para migração se forem removidos
- Mais indicados para testes, avaliação e projetos experimentais

Para aplicações em produção, recomendamos usar os modelos estáveis da nossa [lista principal de modelos](/models/overview).

### Verificando o status beta pela API

Você pode verificar se um modelo está em beta chamando o endpoint [List Models](/api-reference/endpoint/models/list). Modelos beta incluem um campo `betaModel` definido como `true` em seu `model_spec`:

```json
{
  "id": "some-beta-model",
  "model_spec": {
    "name": "Some Beta Model",
    "betaModel": true,
    "privacy": "private"
  },
  "type": "text",
  "object": "model",
  "owned_by": "venice.ai"
}
```

Você pode verificar `if (model.model_spec.betaModel)` para identificar modelos beta e avisar usuários ou tratá-los de forma diferente em sua aplicação.
