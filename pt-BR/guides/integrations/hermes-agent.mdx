O [Hermes Agent](https://hermes-agent.nousresearch.com) é um agente de IA de código aberto, auto-hospedado, construído pela [Nous Research](https://nousresearch.com). Ele oferece memória persistente, criação autônoma de skills e um loop de aprendizado embutido que fica mais capaz quanto mais tempo roda. Aponte-o para a API Venice e seu agente ganha acesso a mais de 230 modelos e ferramentas para texto, imagem, vídeo, áudio, embeddings e mais.

## Por que Venice + Hermes Agent?

A API Venice dá ao seu Hermes Agent acesso à plataforma Venice completa por meio de um único endpoint compatível com OpenAI.

| Capacidade | O que você ganha |
|---|---|
| **Texto e chat** | Modelos privados e anonimizados (GLM, Qwen, Claude, GPT e mais) |
| **Imagem** | Geração, edição, upscaling e remoção de fundo |
| **Vídeo** | Geração e transcrição |
| **Áudio** | Síntese de fala (TTS), geração de música e fala para texto |
| **Embeddings** | Embeddings vetoriais para RAG e busca semântica |
| **Ferramentas** | Web scraping, busca na web, parsing de texto e RPC crypto |

## Configuração

### 1. Instale o Hermes Agent

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>

  <Tab title="WSL2 (Windows)">
    Instale o [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install) primeiro, depois execute o mesmo comando:
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>
</Tabs>

Após a instalação, recarregue seu shell:

```bash
source ~/.zshrc   # ou: source ~/.bashrc
```

### 2. Configure a Venice como seu provedor

Execute o assistente de configuração de modelo:

```bash
hermes model
```

Selecione **Custom endpoint (self-hosted / VLLM / etc.)** e insira o seguinte quando solicitado:

| Campo | Valor |
|---|---|
| **API base URL** | `https://api.venice.ai/api/v1` |
| **API key** | Sua chave de API Venice |
| **Model name** | Um ID de modelo Venice (veja abaixo) |

Obtenha uma chave de API em [venice.ai/settings/api](https://venice.ai/settings/api) se ainda não tiver uma.

<Tip>
Você também pode configurar a Venice diretamente em `~/.hermes/config.yaml`:

```yaml
model:
  default: zai-org-glm-5
  provider: custom
  base_url: https://api.venice.ai/api/v1
  api_key: ${VENICE_API_KEY}
```

E definir a chave em `~/.hermes/.env`:

```bash
VENICE_API_KEY=your-key-here
```
</Tip>

### 3. Escolha um modelo

Quando o assistente perguntar sobre um modelo, escolha um com base no seu caso de uso:

| Caso de uso | Modelo | Privacidade |
|---|---|---|
| Geral | `zai-org-glm-5` | Privado |
| Raciocínio | `kimi-k2-6` | Privado |
| Codificação | `claude-opus-4-7` | Anonimizado |
| Visão | `z-ai-glm-5v-turbo` | Anonimizado |
| Sem censura | `venice-uncensored-1-2` | Privado |

Troque seu modelo a qualquer momento, sem precisar reiniciar:

```bash
hermes model              # assistente completo
```

Ou troque no meio da sessão:

```text
/model custom:claude-opus-4-7
```

### 4. Comece a conversar

Abra o CLI clássico ou a TUI moderna:

```bash
hermes            # CLI clássico
hermes --tui      # TUI moderna (recomendada)
```

Experimente um prompt para verificar se tudo funciona:

```text
Summarize this repo in 5 bullets and tell me what the main entrypoint is.
```

## Conecte plataformas de mensagens

Quando o CLI estiver funcionando, conecte seus aplicativos de mensagens via gateway:

```bash
hermes gateway setup
```

Isso te guia pela conexão de Telegram, Discord, Slack, WhatsApp, Signal e outras plataformas. Seu agente fica acessível por qualquer canal conectado, tudo impulsionado pela Venice.

```bash
hermes gateway      # inicia o gateway de mensagens
```

## Modos de privacidade

Os modelos Venice no Hermes Agent seguem os mesmos [níveis de privacidade](/overview/privacy) da API Venice:

- Modelos **Privados** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) rodam na frota de GPUs da Venice. Os prompts nunca são armazenados nem registrados.
- Modelos **Anonimizados** (Claude, GPT, Gemini, Grok) são intermediados pela Venice com todas as informações identificáveis removidas. O provedor terceiro vê a Venice como cliente, não você.

## Skills da API Venice

O Hermes Agent tem um sistema de skills integrado compatível com o formato [Agent Skills](https://github.com/veniceai/skills). A Venice publica skills oficiais que ensinam seu agente a usar cada endpoint Venice (chat, geração de imagens, vídeo, áudio, embeddings, ferramentas de augment e mais).

Instale as skills Venice diretamente do GitHub:

```bash
hermes skills install veniceai/skills
```

Ou pesquise por skills individuais:

```bash
hermes skills search venice
```

O Hermes descobre cada skill pelo frontmatter de seu `SKILL.md` e o carrega sob demanda.

## Comandos principais

| Comando | Descrição |
|---|---|
| `hermes` | Iniciar conversa |
| `hermes model` | Mudar provedor ou modelo |
| `hermes tools` | Configurar quais ferramentas estão habilitadas |
| `hermes gateway` | Iniciar o gateway de mensagens |
| `hermes --continue` | Retomar sua última sessão |
| `hermes doctor` | Diagnosticar problemas |
| `hermes update` | Atualizar para a versão mais recente |
