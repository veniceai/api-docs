O [Cursor](https://www.cursor.com/) é um editor de código com IA. Você pode usá-lo com a Venice AI para acesso privado e sem censura a uma ampla variedade de modelos.

## Configuração

<Steps>
  <Step title="Obtenha sua chave de API">
    Gere uma chave em [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="Configure o Cursor">
    Abra as **Configurações do Cursor** (ícone de engrenagem), role até **Models** e clique em **Add Model**.

    Em **OpenAI Compatible**, insira:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: Sua chave de API Venice
  </Step>

  <Step title="Adicione modelos">
    No campo do nome do modelo, digite o ID do modelo Venice e pressione Enter. Adicione cada modelo que você deseja usar. Por exemplo:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (veja a [nota sobre o prefixo abaixo](#the-venice--model-prefix))

    Veja o [catálogo de modelos](/models/text) para todos os IDs de modelos disponíveis.
  </Step>

  <Step title="Selecione e verifique">
    Abra um novo chat e use o **seletor de modelo na parte inferior do chat** para escolher um dos modelos Venice que você acabou de adicionar. Envie uma mensagem de teste. Se receber uma resposta, está tudo pronto.
  </Step>
</Steps>

## O prefixo de modelo `venice-`

<Warning>
**Modelos Claude exigem o prefixo `venice-` ao serem usados no Cursor.**

O Cursor reescreve requisições para modelos `claude-*` no formato nativo da Anthropic, que é incompatível com a Venice. Prefixar o ID do modelo com `venice-` evita essa reescrita. A Venice remove o prefixo automaticamente.
</Warning>

| Modelo | ID padrão | ID no Cursor |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
Modelos não-Claude (por exemplo, `minimax-m25`, `kimi-k2-5`, `zai-org-glm-5`) **não são afetados** e funcionam sem o prefixo. O prefixo `venice-` é seguro de usar em qualquer modelo, já que a Venice sempre o remove, mas só é obrigatório para modelos Claude no Cursor.
</Info>
