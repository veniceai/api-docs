O [OpenClaw](https://openclaw.ai) é um gateway de IA de código aberto e auto-hospedado que conecta plataformas de mensagens (WhatsApp, Telegram, Discord, iMessage, Slack) a modelos de IA. A Venice AI está disponível como provedor integrado, dando-lhe acesso a modelos privados e sem censura a partir de qualquer canal conectado.

## Configuração

### 1. Instale o OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. Execute o assistente de onboarding

```bash
openclaw onboard
```

O assistente te guiará pela configuração. Quando solicitado, selecione **Venice AI** como seu provedor na lista, depois cole sua chave de API. Obtenha uma em [venice.ai/settings/api](https://venice.ai/settings/api) se ainda não tiver.

### 3. Escolha um modelo

Durante o onboarding, o OpenClaw mostra todos os modelos Venice disponíveis. Algumas recomendações:

| Caso de uso | Modelo | Privacidade |
|---|---|---|
| Geral | `venice/zai-org-glm-5` | Privado |
| Raciocínio | `venice/kimi-k2-5` | Privado |
| Codificação | `venice/claude-opus-4-6` | Anonimizado |
| Visão | `venice/qwen3-vl-235b-a22b` | Privado |
| Sem censura | `venice/venice-uncensored` | Privado |

Mude seu modelo padrão a qualquer momento:

```bash
openclaw models set venice/zai-org-glm-5
```

Liste todos os modelos disponíveis:

```bash
openclaw models list | grep venice
```

### 4. Comece a conversar

Abra a UI no terminal:

```bash
openclaw tui
```

Ou o dashboard web:

```bash
openclaw dashboard
```

Ou conecte um canal de mensagens (WhatsApp, Telegram, Discord, etc.):

```bash
openclaw channels login
openclaw gateway
```

## Modos de privacidade

Os modelos Venice no OpenClaw seguem os mesmos [níveis de privacidade](/overview/privacy) da API Venice:

- Modelos **Privados** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) rodam na frota de GPUs da Venice. Os prompts nunca são armazenados nem registrados.
- Modelos **Anonimizados** (Claude, GPT, Gemini, Grok) são intermediados pela Venice com todas as informações identificáveis removidas. O provedor terceiro vê a Venice como cliente, não você.

## Geração de imagem e vídeo

Instale a skill Venice AI Media para geração de imagens e vídeos:

```bash
openclaw skills install nhannah/venice-ai-media
```
