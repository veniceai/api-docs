A API da Venice é protegida por chaves de API. Para começar a usar a API Venice, você precisará primeiro gerar uma nova chave. Siga estes passos para começar.

<Steps>
  <Step title="Acesse a página de configurações da API">
    Para chegar à página de configurações da API, visite https://venice.ai/settings/api. Esta página pode ser acessada clicando em "API" na barra de ferramentas à esquerda, ou clicando em "API" dentro das configurações do usuário.

    Dentro deste dashboard, você pode visualizar seus saldos em Diem e USD, seu tier da API, seu uso da API e suas chaves de API.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="Visão geral da API" />
    </Frame>
  </Step>

  <Step title='Clique em "Generate New API Key"'>
    Role a tela do dashboard e selecione "Generate New API Key". Você verá uma lista de opções.

    * **Description:** Usado para nomear sua chave de API

    * **API Key Type:**

      * Chaves "Admin" têm a capacidade de excluir ou gerar chaves de API adicionais programaticamente.

      * Chaves "Inference Only" têm permissão apenas para executar inferência.

    * **Expires at:** Você pode optar por definir uma data de expiração para a chave de API, após a qual ela deixará de funcionar. Por padrão, nenhuma data é definida e a chave funciona indefinidamente.

    * **Epoch Consumption Limits:** Permite criar limites para o uso da API a partir da chave de API individual. Você pode optar por limitar o valor em Diem ou USD permitido dentro de uma determinada época (24h).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Gerar nova chave de API" />
    </Frame>
  </Step>

  <Step title="Gere a chave">
    Clicar em Generate exibirá a chave de API.

    <Warning>
      **Importante:** Esta chave é mostrada apenas uma vez. Certifique-se de copiá-la e armazená-la em um local seguro. Se você perdê-la, precisará excluí-la e criar uma nova.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Sua chave de API" />
    </Frame>
  </Step>
</Steps>
