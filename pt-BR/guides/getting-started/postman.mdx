## Visão geral

A Venice fornece uma coleção abrangente do Postman que permite aos desenvolvedores explorar e testar todas as capacidades da nossa API. Esta coleção inclui requisições pré-configuradas, exemplos e variáveis de ambiente para ajudar você a começar rapidamente com os serviços de IA da Venice.

## Acessando a coleção

Nossa coleção oficial do Postman está disponível no Venice AI Workspace:

* [Workspace Postman Venice AI](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Exemplos Postman Venice AI](https://postman.venice.ai/)

## Recursos da coleção

- **Requisições prontas para uso**: Chamadas de API pré-configuradas para todos os endpoints da Venice
- **Templates de ambiente**: Variáveis de ambiente estruturadas adequadamente
- **Exemplos de requisição**: Exemplos de uso reais para cada endpoint
- **Amostras de resposta**: Respostas de exemplo para ajudar você a entender a saída da API
- **Documentação**: Documentação inline para cada requisição

## Começando

<Steps>
   <Step title="Faça fork da coleção">
      - Navegue até o Venice AI Workspace
      - Clique em "Fork" para criar sua própria cópia da coleção
      - Escolha o workspace de destino
   </Step>

   <Step title="Configure seu ambiente">
      - Crie um novo ambiente no Postman
      - Adicione sua chave de API Venice
      - Configure a URL base: `https://api.venice.ai/api/v1`
   </Step>

   <Step title="Faça sua primeira requisição">
      - Selecione qualquer requisição da coleção
      - Certifique-se de que seu ambiente está selecionado
      - Clique em "Send" para testar a API
   </Step>
</Steps>

## Endpoints disponíveis

A coleção inclui exemplos para todos os endpoints da API Venice:

- Geração de texto
- Geração de imagens
- Informações de modelos
- Upscaling de imagens
- Configuração de prompt do sistema

## Melhores práticas

- Mantenha sua chave de API segura e nunca a compartilhe
- Use variáveis de ambiente para informações sensíveis
- Teste respostas no console do Postman antes da implementação
- Revise os exemplos de resposta para conhecer as estruturas de dados esperadas

<Note>*Nota: A coleção do Postman é atualizada regularmente para refletir as últimas mudanças e recursos da API.*</Note>
