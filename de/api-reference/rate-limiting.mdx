Rate Limits variieren je nach Modell und Stufe. Die unten genannten Standardgrenzen sind eine nützliche Referenz, doch der API-Endpoint `/api_keys/rate_limits` ist die maßgebliche Quelle, um Ihre aktuellen Limits abzurufen. Sie können Ihre genauen Limits jederzeit prüfen:

```bash
curl https://api.venice.ai/api/v1/api_keys/rate_limits \
  -H "Authorization: Bearer $VENICE_API_KEY"
```

## Fehlerbehandlung

Fehlgeschlagene Anfragen (500, 503, 429) sollten mit exponentiellem Backoff wiederholt werden.

Bei 429-Fehlern prüfen Sie speziell den Header `x-ratelimit-reset-requests` für den genauen Unix-Zeitstempel, ab dem ein Retry möglich ist. Die meisten HTTP-Bibliotheken haben eingebaute Retry-Mechanismen, die dies automatisch handhaben.

### Schutz vor Missbrauch

Wenn Sie innerhalb von 30 Sekunden mehr als 20 fehlgeschlagene Anfragen erzeugen, blockiert die API weitere Anfragen für 30 Sekunden:

```
Too many failed attempts (> 20) resulting in a non-success status code. Please wait 30s and try again.
```

## Response-Header

Jede Antwort enthält diese Header:

| Header | Beschreibung |
|:-------|:------------|
| `x-ratelimit-limit-requests` | Maximal erlaubte Requests im aktuellen Fenster |
| `x-ratelimit-remaining-requests` | Verbleibende Requests im aktuellen Fenster |
| `x-ratelimit-reset-requests` | Unix-Zeitstempel, an dem das Fenster zurückgesetzt wird |
| `x-ratelimit-limit-tokens` | Maximal erlaubte Tokens pro Minute |
| `x-ratelimit-remaining-tokens` | Verbleibende Tokens in der aktuellen Minute |
| `x-ratelimit-reset-tokens` | Sekunden, bis das Token-Limit zurückgesetzt wird |

## Partner-Tier

Partner erhalten deutlich höhere Rate Limits:

| Tier | Requests/Min. | Tokens/Min. |
|:-----|-------------:|-----------:|
| XS | 500 | 2.000.000 |
| S | 150 | 1.500.000 |
| M | 100 | 1.500.000 |
| L | 60 | 1.000.000 |

| Typ | Requests/Min. |
|:-----|-------------:|
| Image | 60 |
| Audio | 120 |
| Embedding | 500 |

Wenn Sie konstant an Ihre Rate Limits stoßen und Ihre Nutzungsmuster eine **dauerhafte Nachfrage über die Zeit** zeigen, melden Sie sich, um Partner-Zugang zu besprechen: [api@venice.ai](mailto:api@venice.ai).

Die Partner-Tier-Limits können auf Ihre spezifischen Anforderungen angepasst werden.
