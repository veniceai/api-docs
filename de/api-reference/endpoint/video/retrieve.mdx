Für private Modelle wird die fertige Datei von der `download_url` heruntergeladen, die `/video/queue` zurückgibt (nicht als Binärdatei von diesem Endpoint). Siehe [Private Download-Links](/de/guides/media/video-generation#private-download-links) für Informationen zur Funktionsweise des Links, zu Wiederholungsversuchen und zum optionalen `DELETE`.
