<Warning>
Dies ist ein experimenteller Endpoint und kann Änderungen unterliegen.
</Warning>

## Was dieser Endpoint zurückgibt

Dieser Endpoint gibt paginierte öffentliche Bewertungen für einen einzelnen Character zurück.

- Verwenden Sie den Pfadparameter `slug`, um den Character zu identifizieren.
- Verwenden Sie die Query-Parameter `page` und `pageSize`, um durch die Bewertungen zu blättern.
- Paginierungs-Metadaten werden sowohl im Antworttext als auch in den `x-pagination-*` Response-Headern zurückgegeben.
