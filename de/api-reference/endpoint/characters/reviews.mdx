## Experimenteller Endpoint

<Warning>
Dies ist ein experimenteller Endpoint und kann Änderungen unterliegen.
</Warning>

## Was dieser Endpoint zurückgibt

Dieser Endpoint gibt paginierte öffentliche Bewertungen für einen einzelnen Charakter zurück.

- Verwenden Sie den Pfadparameter `slug`, um den Charakter zu identifizieren.
- Verwenden Sie die Query-Parameter `page` und `pageSize`, um durch die Bewertungen zu paginieren.
- Paginierungs-Metadaten werden sowohl im Antwort-Body als auch in den `x-pagination-*`-Response-Headern zurückgegeben.
