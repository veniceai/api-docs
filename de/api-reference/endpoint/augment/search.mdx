<Warning>Dies ist eine experimentelle API. Das Format von Anfrage und Antwort kann sich ohne Vorankündigung ändern.</Warning>

Senden Sie eine Suchanfrage im Feld `query`. Die API gibt strukturierte Ergebnisse zurück, darunter Titel, URLs, Inhalts-Snippets und Datumsangaben.

**Suchanbieter:**
- `brave` (Standard) — Brave Search mit Zero Data Retention (ZDR). Suchanfragen werden vom Suchanbieter niemals gespeichert oder protokolliert.
- `google` — Google-Suche mit anonymisierten Anfragen. Suchen werden über die Venice-Infrastruktur proxied, sodass Ihre Identität nicht mit der an Google gesendeten Suchanfrage verknüpft wird. Venice speichert oder protokolliert Suchanfragen nicht.

**Preis:** 0,01 $ pro Anfrage.

### Beispiel (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/search \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "latest news about AI"}'
```
