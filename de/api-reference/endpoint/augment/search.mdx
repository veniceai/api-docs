<Warning>Dies ist eine experimentelle API. Das Request- und Response-Format kann sich ohne Vorankündigung ändern.</Warning>

Senden Sie eine Suchanfrage im Feld `query`. Die API gibt strukturierte Ergebnisse zurück, einschließlich Titel, URLs, Inhaltsausschnitte und Daten.

**Suchanbieter:**
- `brave` (Standard) — Brave Search mit Zero Data Retention (ZDR). Suchanfragen werden vom Suchanbieter nie gespeichert oder protokolliert.
- `google` — Google Search mit anonymisierten Anfragen. Suchen werden über die Infrastruktur von Venice geleitet, sodass Ihre Identität nicht mit der an Google gesendeten Suchanfrage verknüpft wird. Venice speichert oder protokolliert keine Suchanfragen.

**Preise:** $0.01 pro Anfrage.

### Beispiel (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/search \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "latest news about AI"}'
```
