<Warning>Dies ist eine experimentelle API. Das Request- und Response-Format kann sich ohne Vorankündigung ändern.</Warning>

Laden Sie eine Dokumentdatei per multipart/form-data über das Feld `file` hoch. Unterstützte Formate sind **PDF**, **DOCX**, **XLSX** und **Klartext**-Dateien (bis zu 25 MB).

Setzen Sie `response_format` auf `json` (Standard) für strukturierte Ausgabe mit extrahiertem Text und Token-Anzahl, oder auf `text` für den reinen extrahierten Text.

**Datenschutz:** Das Text-Parsing läuft vollständig im Arbeitsspeicher auf der Infrastruktur von Venice mit Zero Data Retention. Ihre Dokumente werden verarbeitet und sofort verworfen — kein Inhalt wird gespeichert oder protokolliert.

**Preise:** $0.01 pro Anfrage.

### Beispiel (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/text-parser \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -F "file=@document.pdf" \
  -F "response_format=json"
```
