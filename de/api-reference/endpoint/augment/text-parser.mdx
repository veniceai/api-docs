<Warning>Dies ist eine experimentelle API. Das Format von Anfrage und Antwort kann sich ohne Vorankündigung ändern.</Warning>

Laden Sie eine Dokumentdatei per multipart/form-data über das Feld `file` hoch. Unterstützte Formate umfassen **PDF**, **DOCX**, **XLSX** und **Klartext**-Dateien (bis zu 25 MB).

Setzen Sie `response_format` auf `json` (Standard) für strukturierte Ausgabe mit extrahiertem Text und Token-Anzahl, oder auf `text` für den reinen extrahierten Text.

**Datenschutz:** Das Text-Parsing läuft vollständig im Arbeitsspeicher der Venice-Infrastruktur ohne Datenspeicherung. Ihre Dokumente werden verarbeitet und sofort verworfen — es werden keine Inhalte gespeichert oder protokolliert.

**Preis:** 0,01 $ pro Anfrage.

### Beispiel (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/text-parser \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -F "file=@document.pdf" \
  -F "response_format=json"
```
