Gibt die alphabetisch sortierte Liste der Netzwerk-Slugs zurück, die vom Venice Crypto RPC-Proxy unterstützt werden. Verwenden Sie dies, um herauszufinden, welche `:network`-Werte für `POST /crypto/rpc/{network}` gültig sind.

Dieser Endpoint ist **öffentlich** — es ist kein API-Schlüssel oder Wallet-Authentifizierung erforderlich.

## Antwortformat

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "robinhood-mainnet",
    "robinhood-testnet",
    "solana-devnet",
    "solana-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

Die Liste ist maßgeblich — wenn ein Slug nicht hier aufgeführt ist, gibt der Proxy-Endpoint `400 Unsupported RPC network` zurück.
