Rufen Sie `/audio/quote` auf, um die Kosten zu schätzen, und pollen Sie anschließend `/audio/retrieve` mit der zurückgegebenen `queue_id`, bis die Generierung abgeschlossen ist. Wenn Sie die generierten Medien nach dem Abruf behalten, rufen Sie `/audio/complete` auf, sobald Sie sie heruntergeladen haben.
