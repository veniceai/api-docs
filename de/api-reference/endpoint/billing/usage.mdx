Exportiert Nutzungsdaten für ein Konto. Beschreibungen der Antwortfelder finden Sie unten:

- **timestamp**: Der Zeitstempel, zu dem der Abrechnungs-Usage-Eintrag erstellt wurde
- **sku**: Das mit dem Abrechnungs-Usage-Eintrag verbundene Produkt
- **pricePerUnitUsd**: Der Preis pro Einheit in USD
- **unit**: Die Anzahl der verbrauchten Einheiten
- **amount**: Der gesamte für den Abrechnungs-Usage-Eintrag berechnete Betrag
- **currency**: Die für den Abrechnungs-Usage-Eintrag berechnete Währung
- **notes**: Notizen zum Abrechnungs-Usage-Eintrag
- **inferenceDetails.requestId**: Die mit der Inference verbundene Request-ID
- **inferenceDetails.inferenceExecutionTime**: Zeit für die Inference-Ausführung in Millisekunden
- **inferenceDetails.promptTokens**: Anzahl der im Prompt angeforderten Tokens. Nur bei LLM-Nutzung vorhanden.
- **inferenceDetails.completionTokens**: Anzahl der in der Completion verwendeten Tokens. Nur bei LLM-Nutzung vorhanden.
