Die Venice-API ist durch API-Schlüssel geschützt. Um die Venice-API zu nutzen, müssen Sie zunächst einen neuen Schlüssel erzeugen. Folgen Sie diesen Schritten, um loszulegen.

<Steps>
  <Step title="Die API-Einstellungsseite besuchen">
    Sie erreichen die API-Einstellungsseite unter https://venice.ai/settings/api. Diese Seite ist über "API" in der linken Symbolleiste oder über "API" in Ihren Benutzereinstellungen zugänglich.

    In diesem Dashboard können Sie Ihre Diem- und USD-Guthaben, Ihren API-Tarif, Ihre API-Nutzung und Ihre API-Schlüssel einsehen.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API-Übersicht" />
    </Frame>
  </Step>

  <Step title="Auf 'Generate New API Key' klicken">
    Scrollen Sie im Dashboard nach unten und wählen Sie "Generate New API Key". Sie erhalten eine Liste von Optionen.

    * **Description:** Wird verwendet, um Ihren API-Schlüssel zu benennen

    * **API Key Type:**

      * "Admin"-Schlüssel können weitere API-Schlüssel programmatisch löschen oder erstellen.

      * "Inference Only"-Schlüssel sind nur zur Ausführung von Inference berechtigt.

    * **Expires at:** Sie können ein Ablaufdatum für den API-Schlüssel festlegen, nach dem er nicht mehr funktioniert. Standardmäßig wird kein Datum gesetzt, und der Schlüssel funktioniert unbegrenzt.

    * **Epoch Consumption Limits:** Damit können Sie Limits für die API-Nutzung des einzelnen API-Schlüssels festlegen. Sie können den zulässigen Diem- oder USD-Betrag innerhalb einer bestimmten Epoche (24 Stunden) begrenzen.

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Neuen API-Schlüssel erzeugen" />
    </Frame>
  </Step>

  <Step title="Den Schlüssel erzeugen">
    Ein Klick auf Generate zeigt Ihnen den API-Schlüssel an.

    <Warning>
      **Wichtig:** Dieser Schlüssel wird nur einmal angezeigt. Achten Sie darauf, ihn zu kopieren und an einem sicheren Ort aufzubewahren. Wenn Sie ihn verlieren, müssen Sie ihn löschen und einen neuen erstellen.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Ihr API-Schlüssel" />
    </Frame>
  </Step>
</Steps>
