Die Venice API ist durch API-Schlüssel geschützt. Um die Venice API zu nutzen, müssen Sie zunächst einen neuen Schlüssel generieren. Folgen Sie diesen Schritten zum Einstieg.

<Steps>
  <Step title="Besuchen Sie die API-Einstellungsseite">
    Um zur API-Einstellungsseite zu gelangen, besuchen Sie https://venice.ai/settings/api. Diese Seite ist über den Link "API" in der linken Seitenleiste oder durch Klick auf "API" in Ihren Benutzereinstellungen erreichbar.

    In diesem Dashboard können Sie Ihre Diem- und USD-Guthaben, Ihre API-Stufe, Ihre API-Nutzung und Ihre API-Schlüssel einsehen.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API Overview" />
    </Frame>
  </Step>

  <Step title="Klicken Sie auf 'Generate New API Key'">
    Scrollen Sie im Dashboard nach unten und wählen Sie "Generate New API Key". Es wird eine Liste mit Optionen angezeigt.

    * **Description:** Wird verwendet, um Ihren API-Schlüssel zu benennen

    * **API Key Type:**

      * "Admin"-Schlüssel können programmatisch zusätzliche API-Schlüssel erstellen oder löschen.

      * "Inference Only"-Schlüssel dürfen nur Inferenz ausführen.

    * **Expires at:** Sie können ein Ablaufdatum für den API-Schlüssel festlegen, nach dem er nicht mehr funktioniert. Standardmäßig wird kein Datum festgelegt, und der Schlüssel funktioniert unbegrenzt.

    * **Epoch Consumption Limits:** Hiermit können Sie Limits für die API-Nutzung des einzelnen API-Schlüssels festlegen. Sie können den innerhalb einer Epoche (24 Stunden) zulässigen Diem- oder USD-Betrag begrenzen.

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Generate New API Key" />
    </Frame>
  </Step>

  <Step title="Schlüssel generieren">
    Durch Klick auf "Generate" wird Ihnen der API-Schlüssel angezeigt.

    <Warning>
      **Wichtig:** Dieser Schlüssel wird nur einmal angezeigt. Stellen Sie sicher, dass Sie ihn kopieren und an einem sicheren Ort aufbewahren. Wenn Sie ihn verlieren, müssen Sie ihn löschen und einen neuen erstellen.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Your API Key" />
    </Frame>
  </Step>
</Steps>
