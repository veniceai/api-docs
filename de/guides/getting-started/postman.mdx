## Überblick

Venice stellt eine umfassende Postman-Collection bereit, mit der Entwickler die gesamten Möglichkeiten unserer API erkunden und testen können. Diese Collection enthält vorkonfigurierte Requests, Beispiele und Umgebungsvariablen, damit Sie schnell mit Venice's KI-Diensten loslegen können.

## Zugriff auf die Collection

Unsere offizielle Postman-Collection ist im Venice AI Workspace verfügbar:

* [Venice AI Postman Workspace](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman Examples](https://postman.venice.ai/)

## Funktionen der Collection

- **Direkt einsatzbereite Requests**: Vorkonfigurierte API-Aufrufe für alle Venice-Endpoints
- **Umgebungs-Templates**: Sauber strukturierte Umgebungsvariablen
- **Request-Beispiele**: Praxisbeispiele für jeden Endpoint
- **Antwort-Beispiele**: Beispielantworten, um Sie beim Verstehen der API-Ausgabe zu unterstützen
- **Dokumentation**: Inline-Dokumentation für jeden Request

## Erste Schritte

<Steps>
   <Step title="Collection forken">
      - Navigieren Sie zum Venice AI Workspace
      - Klicken Sie auf "Fork", um eine eigene Kopie der Collection zu erstellen
      - Wählen Sie das Ziel-Workspace
   </Step>

   <Step title="Umgebung einrichten">
      - Erstellen Sie eine neue Umgebung in Postman
      - Fügen Sie Ihren Venice-API-Schlüssel hinzu
      - Konfigurieren Sie die Base-URL: `https://api.venice.ai/api/v1`
   </Step>

   <Step title="Erste Anfrage senden">
      - Wählen Sie einen beliebigen Request aus der Collection
      - Stellen Sie sicher, dass Ihre Umgebung ausgewählt ist
      - Klicken Sie auf "Send", um die API zu testen
   </Step>
</Steps>

## Verfügbare Endpoints

Die Collection enthält Beispiele für alle Venice-API-Endpoints:

- Textgenerierung
- Bilderzeugung
- Modellinformationen
- Bild-Hochskalierung
- Konfiguration des System-Prompts

## Best Practices

- Halten Sie Ihren API-Schlüssel sicher und teilen Sie ihn niemals
- Verwenden Sie Umgebungsvariablen für sensible Informationen
- Testen Sie Antworten in der Postman-Konsole, bevor Sie sie implementieren
- Prüfen Sie die Beispielantworten, um die erwarteten Datenstrukturen zu verstehen

<Note>*Hinweis: Die Postman-Collection wird regelmäßig aktualisiert, um die neuesten API-Änderungen und -Funktionen widerzuspiegeln.*</Note>
