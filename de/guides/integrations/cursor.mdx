## Einrichtung

<Steps>
  <Step title="API-Schlüssel erstellen">
    Erstelle einen Schlüssel unter [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="Cursor konfigurieren">
    Öffne **Cursor Settings** (Zahnrad-Symbol), scrolle zu **Models** und klicke auf **Add Model**.

    Gib unter **OpenAI Compatible** ein:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: Dein Venice API-Schlüssel
  </Step>

  <Step title="Modelle hinzufügen">
    Tippe im Feld für den Modellnamen die Venice-Modell-ID ein und drücke Enter. Füge jedes Modell hinzu, das du verwenden möchtest. Zum Beispiel:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (siehe [Präfix-Hinweis unten](#the-venice--model-prefix))

    Alle verfügbaren Modell-IDs findest du im [Modellkatalog](/models/text).
  </Step>

  <Step title="Auswählen und prüfen">
    Öffne einen neuen Chat und wähle über die **Modellauswahl am unteren Rand des Chats** eines der gerade hinzugefügten Venice-Modelle aus. Sende eine Testnachricht. Wenn du eine Antwort erhältst, ist alles bereit.
  </Step>
</Steps>

## Das `venice-`-Modell-Präfix

<Warning>
**Claude-Modelle benötigen in Cursor das Präfix `venice-`.**

Cursor schreibt Anfragen für `claude-*`-Modelle in Anthropics natives Format um, das mit Venice nicht kompatibel ist. Das Präfix `venice-` verhindert dieses Umschreiben. Venice entfernt das Präfix automatisch.
</Warning>

| Modell | Standard-ID | Cursor-ID |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
Nicht-Claude-Modelle (z. B. `minimax-m25`, `kimi-k2-5`, `zai-org-glm-5`) sind **nicht betroffen** und funktionieren ohne Präfix. Das Präfix `venice-` kann für jedes Modell verwendet werden, da Venice es stets entfernt — erforderlich ist es jedoch nur für Claude-Modelle in Cursor.
</Info>
