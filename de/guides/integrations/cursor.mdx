[Cursor](https://www.cursor.com/) ist ein KI-gestützter Code-Editor. Sie können ihn mit Venice AI für privaten, unbeschränkten Zugriff auf eine Vielzahl von Modellen nutzen.

## Einrichtung

<Steps>
  <Step title="API-Schlüssel holen">
    Erzeugen Sie einen Schlüssel auf [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="Cursor konfigurieren">
    Öffnen Sie die **Cursor Settings** (Zahnradsymbol), scrollen Sie zu **Models** und klicken Sie auf **Add Model**.

    Geben Sie unter **OpenAI Compatible** ein:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: Ihr Venice-API-Schlüssel
  </Step>

  <Step title="Modelle hinzufügen">
    Geben Sie im Feld für den Modellnamen die Venice-Modell-ID ein und drücken Sie Enter. Fügen Sie jedes Modell hinzu, das Sie verwenden möchten. Zum Beispiel:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (siehe [Präfix-Hinweis unten](#the-venice--model-prefix))

    Alle verfügbaren Modell-IDs finden Sie im [Modellkatalog](/de/models/text).
  </Step>

  <Step title="Auswählen und prüfen">
    Öffnen Sie einen neuen Chat und wählen Sie über die **Modellauswahl unten im Chat** eines der Venice-Modelle, die Sie gerade hinzugefügt haben. Senden Sie eine Testnachricht. Wenn Sie eine Antwort erhalten, sind Sie startklar.
  </Step>
</Steps>

## Das `venice-`-Modellpräfix

<Warning>
**Claude-Modelle erfordern in Cursor das `venice-`-Präfix.**

Cursor schreibt Anfragen für `claude-*`-Modelle in das native Anthropic-Format um, das mit Venice nicht kompatibel ist. Wird der Modell-ID das Präfix `venice-` vorangestellt, verhindert das diese Umschreibung. Venice entfernt das Präfix automatisch.
</Warning>

| Modell | Standard-ID | Cursor-ID |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
Nicht-Claude-Modelle (z. B. `minimax-m25`, `kimi-k2-5`, `zai-org-glm-5`) sind **nicht betroffen** und funktionieren ohne Präfix. Das `venice-`-Präfix kann bei jedem Modell verwendet werden, da Venice es immer entfernt, ist aber nur für Claude-Modelle in Cursor erforderlich.
</Info>
