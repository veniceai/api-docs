[OpenClaw](https://openclaw.ai) ist ein Open-Source-, selbstgehostetes KI-Gateway, das Messaging-Plattformen (WhatsApp, Telegram, Discord, iMessage, Slack) mit KI-Modellen verbindet. Venice AI ist als integrierter Anbieter verfügbar und gibt Ihnen über jeden verbundenen Kanal Zugriff auf private und unbeschränkte Modelle.

## Einrichtung

### 1. OpenClaw installieren

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. Onboarding-Assistent ausführen

```bash
openclaw onboard
```

Der Assistent führt Sie durch die Einrichtung. Wählen Sie bei der Aufforderung **Venice AI** als Anbieter aus der Liste aus und fügen Sie Ihren API-Schlüssel ein. Falls Sie noch keinen haben, holen Sie sich einen auf [venice.ai/settings/api](https://venice.ai/settings/api).

### 3. Modell auswählen

Während des Onboardings zeigt OpenClaw alle verfügbaren Venice-Modelle. Einige Empfehlungen:

| Anwendungsfall | Modell | Privatsphäre |
|---|---|---|
| Allgemein | `venice/zai-org-glm-5` | Privat |
| Reasoning | `venice/kimi-k2-5` | Privat |
| Coding | `venice/claude-opus-4-6` | Anonymisiert |
| Vision | `venice/qwen3-vl-235b-a22b` | Privat |
| Unbeschränkt | `venice/venice-uncensored` | Privat |

Standardmodell jederzeit ändern:

```bash
openclaw models set venice/zai-org-glm-5
```

Alle verfügbaren Modelle auflisten:

```bash
openclaw models list | grep venice
```

### 4. Loslegen

Terminal-UI öffnen:

```bash
openclaw tui
```

Oder das Web-Dashboard:

```bash
openclaw dashboard
```

Oder einen Messaging-Kanal verbinden (WhatsApp, Telegram, Discord usw.):

```bash
openclaw channels login
openclaw gateway
```

## Datenschutz-Modi

Venice-Modelle in OpenClaw folgen denselben [Privacy-Tiers](/de/overview/privacy) wie die Venice-API:

- **Private**-Modelle (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) laufen auf Venice's GPU-Flotte. Prompts werden niemals gespeichert oder protokolliert.
- **Anonymized**-Modelle (Claude, GPT, Gemini, Grok) werden durch Venice proxied; alle identifizierenden Informationen werden entfernt. Der Drittanbieter sieht Venice als Kunden, nicht Sie.

## Bild- und Videogenerierung

Installieren Sie die Venice-AI-Media-Skill für Bild- und Videogenerierung:

```bash
openclaw skills install nhannah/venice-ai-media
```
