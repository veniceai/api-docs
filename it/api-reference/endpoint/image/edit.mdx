</Warning>

<Info>
**Prezzo:** Il prezzo della modifica immagini varia per modello. Il modello predefinito (`qwen-edit`) è **$0,04 per modifica**. Consulta l'[endpoint Models](/it/api-reference/endpoint/models/list) per tutti i modelli di edit disponibili e il loro prezzo.
</Info>

## Fasce di qualità

Alcuni modelli di edit accettano un parametro opzionale `quality` (`low`, `medium`, `high`) che bilancia fedeltà visiva e costo. Attualmente supportato da `gpt-image-2-edit`; gli altri modelli ignorano il parametro.

```json
{
  "model": "gpt-image-2-edit",
  "image": "iVBORw0KGgo...",
  "prompt": "change the sky to a sunrise",
  "resolution": "2K",
  "quality": "medium"
}
```

Quando ometti `quality`, il modello usa la sua fascia predefinita (`high` per `gpt-image-2-edit`). I prezzi per fascia (1K/2K/4K × low/medium/high) sono esposti sotto `model_spec.pricing.quality` nell'[endpoint Models](/it/api-reference/endpoint/models/list) e sono elencati nella [panoramica dei prezzi](/it/overview/pricing).

<Info>
Il pricing basato sulla qualità è l'impostazione predefinita per tutti i chiamanti API e SDK — non è richiesto alcun header opt-in. Le richieste che omettono `quality` vengono fatturate alla fascia predefinita del modello (`high` per `gpt-image-2-edit`).
</Info>
