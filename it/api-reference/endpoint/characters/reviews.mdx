## Endpoint sperimentale

<Warning>
Questo è un endpoint sperimentale e può essere soggetto a modifiche.
</Warning>

## Cosa restituisce

Questo endpoint restituisce recensioni pubbliche paginate per un singolo personaggio.

- Usa il parametro di path `slug` per identificare il personaggio.
- Usa i parametri di query `page` e `pageSize` per paginare le recensioni.
- I metadati di paginazione sono restituiti sia nel corpo della risposta sia negli header `x-pagination-*`.
