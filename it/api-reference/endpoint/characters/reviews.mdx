## Endpoint sperimentale

<Warning>
Questo è un endpoint sperimentale e potrebbe essere soggetto a modifiche.
</Warning>

## Cosa restituisce

Questo endpoint restituisce le recensioni pubbliche paginate per un singolo character.

- Usa il parametro di path `slug` per identificare il character.
- Usa i parametri di query `page` e `pageSize` per paginare le recensioni.
- I metadati di paginazione vengono restituiti sia nel body della risposta sia negli header di risposta `x-pagination-*`.
