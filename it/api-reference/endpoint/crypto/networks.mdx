Restituisce l'elenco in ordine alfabetico degli slug di rete supportati dal proxy crypto RPC di Venice. Usalo per scoprire quali valori di `:network` sono validi per `POST /crypto/rpc/{network}`.

Questo endpoint è **pubblico** — non richiede API key né autenticazione tramite wallet.

## Struttura della risposta

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

L'elenco è autoritativo — se uno slug non è qui, l'endpoint proxy restituisce `400 Unsupported RPC network`.
