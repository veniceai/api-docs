Restituisce l'elenco ordinato alfabeticamente degli slug delle reti supportate dal proxy RPC crypto di Venice. Usalo per scoprire quali valori `:network` sono validi per `POST /crypto/rpc/{network}`.

Questo endpoint è **pubblico** — non sono richieste API key né autenticazione tramite wallet.

## Forma della risposta

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "robinhood-mainnet",
    "robinhood-testnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

L'elenco è autorevole — se uno slug non è qui, l'endpoint del proxy restituisce `400 Unsupported RPC network`.
