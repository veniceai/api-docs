Esporta i dati di utilizzo per un utente. La descrizione dei campi della risposta è riportata di seguito:

- **timestamp**: Il timestamp di creazione della voce di utilizzo della fatturazione
- **sku**: Il prodotto associato alla voce di utilizzo della fatturazione
- **pricePerUnitUsd**: Il prezzo per unità in USD
- **unit**: Il numero di unità consumate
- **amount**: L'importo totale addebitato per la voce di utilizzo della fatturazione
- **currency**: La valuta addebitata per la voce di utilizzo della fatturazione
- **notes**: Note sulla voce di utilizzo della fatturazione
- **inferenceDetails.requestId**: L'ID della richiesta associata all'inferenza
- **inferenceDetails.inferenceExecutionTime**: Tempo impiegato per l'esecuzione dell'inferenza in millisecondi
- **inferenceDetails.promptTokens**: Numero di token richiesti nel prompt. Presente solo per l'utilizzo LLM.
- **inferenceDetails.completionTokens**: Numero di token utilizzati nel completamento. Presente solo per l'utilizzo LLM.
