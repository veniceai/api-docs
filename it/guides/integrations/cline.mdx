[Cline](https://cline.bot/) è un assistente di coding AI per VS Code che può leggere e modificare file, eseguire comandi e lavorare su task di sviluppo multi-step. Venice funziona tramite le impostazioni del provider compatibile con OpenAI di Cline.

<CardGroup cols={3}>
  <Card title="Integrazione con VS Code" icon="code">
    Usa i modelli Venice senza uscire dal tuo editor
  </Card>
  <Card title="Compatibile con OpenAI" icon="plug">
    Collegati tramite l'endpoint `/chat/completions` di Venice
  </Card>
  <Card title="Flessibilità nei modelli" icon="microchip">
    Scegli qualsiasi modello di testo Venice compatibile
  </Card>
</CardGroup>

---

## Prerequisiti

- Una API key Venice da [venice.ai/settings/api](https://venice.ai/settings/api)
- [Visual Studio Code](https://code.visualstudio.com/) installato
- L'[estensione Cline](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev) installata in VS Code

---

## Configurazione

<Steps>
  <Step title="Apri le impostazioni di Cline">
    Seleziona l'icona di Cline nella activity bar di VS Code, poi seleziona l'icona a forma di ingranaggio per aprire le impostazioni di Cline.
  </Step>

  <Step title="Scegli il provider API">
    Imposta **API Provider** su **OpenAI Compatible**.
  </Step>

  <Step title="Configura Venice">
    Inserisci i seguenti valori:

    | Impostazione | Valore |
    | --- | --- |
    | Base URL | `https://api.venice.ai/api/v1` |
    | API Key | La tua API key Venice |
    | Model ID | Un ID modello dal [catalogo dei modelli di testo](/models/text) |

    Ad esempio, usa `zai-org-glm-5-1` come Model ID.

    Inserisci la sola API key, senza il prefisso `Bearer`. Mantieni il Base URL su `/api/v1`; Cline aggiunge il percorso `/chat/completions`.
  </Step>

  <Step title="Configura le capacità del modello">
    Espandi **Model Configuration** e imposta la finestra di contesto e il numero massimo di token in output in base al modello selezionato. Abilita le capacità supportate dal modello, incluso l'input di immagini o il computer use quando applicabile.

    Verifica il modello selezionato nel [catalogo dei modelli di testo](/models/text) prima di abilitare capacità opzionali.
  </Step>

  <Step title="Salva e testa">
    Salva le impostazioni del provider, apri un progetto e assegna a Cline un piccolo task come:

    ```txt
    Review the current file and suggest one safe improvement.
    ```

    Rivedi ogni permesso su file o comandi prima di approvarlo.
  </Step>
</Steps>

## Verifica l'API di Venice

Se Cline non riesce a connettersi, verifica la API key e gli ID dei modelli disponibili in modo indipendente:

```bash
curl https://api.venice.ai/api/v1/models \
  -H "Authorization: Bearer $VENICE_API_KEY"
```

Copia l'`id` di un modello esattamente nel campo **Model ID** di Cline.

---

## Note di configurazione

- Usa `https://api.venice.ai/api/v1` come Base URL, non l'endpoint completo `/chat/completions`.
- Gli ID dei modelli possono cambiare nel tempo. Usa il [catalogo dei modelli di testo](/models/text) o `GET /models` invece di affidarti a un esempio più vecchio.
- Le impostazioni di context window e token in output di Cline influiscono su come pianifica le richieste. Adattale al modello selezionato per ottenere i migliori risultati.
- Mantieni privata la tua API key Venice. Non incollarla nei file sorgente e non committarla su git.

## Risoluzione dei problemi

<AccordionGroup>
  <Accordion title="Errore 401 o di autenticazione">
    Verifica che la API key provenga da [venice.ai/settings/api](https://venice.ai/settings/api). Incolla la chiave grezza in Cline senza `Bearer` o spazi extra.
  </Accordion>
  <Accordion title="Modello non trovato">
    Verifica che il Model ID compaia nel [catalogo dei modelli di testo](/models/text) o nella risposta di `GET /models`, quindi copialo esattamente.
  </Accordion>
  <Accordion title="Errore di connessione o di endpoint">
    Verifica che **API Provider** sia **OpenAI Compatible** e che il Base URL sia esattamente `https://api.venice.ai/api/v1`.
  </Accordion>
  <Accordion title="Cline non può usare strumenti o modificare file">
    Apri **Model Configuration**, verifica che il modello selezionato supporti le capacità richieste e abilita computer use se Cline espone quell'opzione per il modello.
  </Accordion>
</AccordionGroup>
