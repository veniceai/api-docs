[OpenClaw](https://openclaw.ai) è un gateway AI open-source e self-hosted che connette le piattaforme di messaggistica (WhatsApp, Telegram, Discord, iMessage, Slack) ai modelli AI. Venice AI è disponibile come provider integrato, dandoti accesso a modelli privati e senza censura da qualsiasi canale connesso.

## Configurazione

### 1. Installa OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. Esegui la procedura guidata di onboarding

```bash
openclaw onboard
```

La procedura guidata ti accompagnerà nella configurazione. Quando richiesto, seleziona **Venice AI** come provider dall'elenco, poi incolla la tua API key. Ottienine una da [venice.ai/settings/api](https://venice.ai/settings/api) se non ne hai ancora.

### 3. Scegli un modello

Durante l'onboarding, OpenClaw mostra tutti i modelli Venice disponibili. Alcune raccomandazioni:

| Caso d'uso | Modello | Privacy |
|---|---|---|
| Generale | `venice/zai-org-glm-5` | Privato |
| Ragionamento | `venice/kimi-k2-5` | Privato |
| Coding | `venice/claude-opus-4-6` | Anonimizzato |
| Vision | `venice/qwen3-vl-235b-a22b` | Privato |
| Senza censura | `venice/venice-uncensored` | Privato |

Cambia il tuo modello predefinito in qualsiasi momento:

```bash
openclaw models set venice/zai-org-glm-5
```

Elenca tutti i modelli disponibili:

```bash
openclaw models list | grep venice
```

### 4. Inizia a chattare

Apri la TUI:

```bash
openclaw tui
```

Oppure la dashboard web:

```bash
openclaw dashboard
```

Oppure connetti un canale di messaggistica (WhatsApp, Telegram, Discord, ecc.):

```bash
openclaw channels login
openclaw gateway
```

## Modalità di privacy

I modelli Venice in OpenClaw seguono le stesse [fasce di privacy](/it/overview/privacy) dell'API Venice:

- I modelli **Private** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) girano sulla flotta GPU di Venice. I prompt non sono mai memorizzati o registrati.
- I modelli **Anonymized** (Claude, GPT, Gemini, Grok) sono inoltrati tramite Venice con tutte le informazioni identificative rimosse. Il provider di terze parti vede Venice come cliente, non te.

## Generazione immagini e video

Installa la skill Venice AI Media per la generazione di immagini e video:

```bash
openclaw skills install nhannah/venice-ai-media
```
