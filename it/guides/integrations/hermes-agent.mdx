[Hermes Agent](https://hermes-agent.nousresearch.com) è un agente AI open-source e self-hosted costruito da [Nous Research](https://nousresearch.com). Offre memoria persistente, creazione autonoma di skill e un ciclo di apprendimento integrato che diventa più capace più a lungo viene eseguito. Puntalo all'API Venice e il tuo agente ottiene accesso a oltre 230 modelli e tool per testo, immagini, video, audio, embeddings e altro.

## Perché Venice + Hermes Agent?

L'API Venice dà al tuo Hermes Agent accesso all'intera piattaforma Venice tramite un singolo endpoint compatibile OpenAI.

| Capacità | Cosa ottieni |
|---|---|
| **Testo e chat** | Modelli privati e anonimizzati (GLM, Qwen, Claude, GPT e altri) |
| **Immagini** | Generazione, modifica, upscaling e rimozione sfondo |
| **Video** | Generazione e trascrizione |
| **Audio** | Sintesi vocale (TTS), generazione musicale e speech-to-text |
| **Embeddings** | Vettori di embedding per RAG e ricerca semantica |
| **Tool** | Web scraping, web search, parsing testo e crypto RPC |

## Configurazione

### 1. Installa Hermes Agent

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>

  <Tab title="WSL2 (Windows)">
    Installa prima [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install), poi esegui lo stesso comando:
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>
</Tabs>

Dopo l'installazione, ricarica la shell:

```bash
source ~/.zshrc   # or: source ~/.bashrc
```

### 2. Configura Venice come provider

Esegui la procedura guidata di configurazione del modello:

```bash
hermes model
```

Seleziona **Custom endpoint (self-hosted / VLLM / etc.)** e inserisci quanto segue quando richiesto:

| Campo | Valore |
|---|---|
| **API base URL** | `https://api.venice.ai/api/v1` |
| **API key** | La tua API key Venice |
| **Model name** | Un ID modello Venice (vedi sotto) |

Ottieni una API key da [venice.ai/settings/api](https://venice.ai/settings/api) se non ne hai ancora.

<Tip>
Puoi anche configurare Venice direttamente in `~/.hermes/config.yaml`:

```yaml
model:
  default: zai-org-glm-5
  provider: custom
  base_url: https://api.venice.ai/api/v1
  api_key: ${VENICE_API_KEY}
```

E imposta la key in `~/.hermes/.env`:

```bash
VENICE_API_KEY=your-key-here
```
</Tip>

### 3. Scegli un modello

Quando la procedura guidata chiede un modello, scegline uno in base al tuo caso d'uso:

| Caso d'uso | Modello | Privacy |
|---|---|---|
| Generale | `zai-org-glm-5` | Privato |
| Ragionamento | `kimi-k2-6` | Privato |
| Coding | `claude-opus-4-7` | Anonimizzato |
| Vision | `z-ai-glm-5v-turbo` | Anonimizzato |
| Senza censura | `venice-uncensored-1-2` | Privato |

Cambia il modello in qualsiasi momento senza riavvio:

```bash
hermes model              # procedura guidata completa
```

Oppure cambia a metà sessione:

```text
/model custom:claude-opus-4-7
```

### 4. Inizia a chattare

Apri la CLI classica o la moderna TUI:

```bash
hermes            # CLI classica
hermes --tui      # TUI moderna (raccomandata)
```

Prova un prompt per verificare che tutto funzioni:

```text
Summarize this repo in 5 bullets and tell me what the main entrypoint is.
```

## Connetti piattaforme di messaggistica

Una volta che la CLI funziona, connetti le tue app di messaggistica tramite il gateway:

```bash
hermes gateway setup
```

Questo ti accompagna nella connessione di Telegram, Discord, Slack, WhatsApp, Signal e altre piattaforme. Il tuo agente diventa raggiungibile da qualsiasi canale connesso, tutto alimentato da Venice.

```bash
hermes gateway      # avvia il gateway di messaggistica
```

## Modalità di privacy

I modelli Venice in Hermes Agent seguono le stesse [fasce di privacy](/it/overview/privacy) dell'API Venice:

- I modelli **Private** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) girano sulla flotta GPU di Venice. I prompt non sono mai memorizzati o registrati.
- I modelli **Anonymized** (Claude, GPT, Gemini, Grok) sono inoltrati tramite Venice con tutte le informazioni identificative rimosse. Il provider di terze parti vede Venice come cliente, non te.

## Skill API Venice

Hermes Agent ha un sistema di skill integrato compatibile con il formato [Agent Skills](https://github.com/veniceai/skills). Venice pubblica skill ufficiali che insegnano al tuo agente come usare ogni endpoint Venice (chat, generazione immagini, video, audio, embedding, augment tool e altri).

Installa le skill Venice direttamente da GitHub:

```bash
hermes skills install veniceai/skills
```

Oppure cerca singole skill:

```bash
hermes skills search venice
```

Hermes scoprirà ogni skill dal suo frontmatter `SKILL.md` e la caricherà on demand.
