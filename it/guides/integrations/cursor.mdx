[Cursor](https://www.cursor.com/) è un code editor potenziato dall'AI. Puoi usarlo con Venice AI per accedere in modo privato e senza censura a una vasta gamma di modelli.

## Configurazione

<Steps>
  <Step title="Ottieni la tua API key">
    Genera una key da [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="Configura Cursor">
    Apri **Cursor Settings** (icona ingranaggio), scorri fino a **Models** e clicca **Add Model**.

    Sotto **OpenAI Compatible**, inserisci:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: La tua API key Venice
  </Step>

  <Step title="Aggiungi i modelli">
    Nel campo del nome modello, digita l'ID del modello Venice e premi Invio. Aggiungi ciascun modello che vuoi usare. Ad esempio:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (vedi la [nota sul prefisso qui sotto](#the-venice--model-prefix))

    Consulta il [catalogo modelli](/it/models/text) per tutti gli ID modello disponibili.
  </Step>

  <Step title="Seleziona e verifica">
    Apri una nuova chat e usa il **selettore di modello in fondo alla chat** per scegliere uno dei modelli Venice appena aggiunti. Invia un messaggio di test. Se ricevi una risposta, sei a posto.
  </Step>
</Steps>

## Il prefisso modello `venice-`

<Warning>
**I modelli Claude richiedono il prefisso `venice-` quando usati in Cursor.**

Cursor riscrive le richieste per i modelli `claude-*` nel formato nativo di Anthropic, che è incompatibile con Venice. Anteponendo `venice-` all'ID del modello si previene questa riscrittura. Venice rimuove il prefisso automaticamente.
</Warning>

| Modello | ID standard | ID Cursor |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
I modelli non-Claude (ad esempio `minimax-m25`, `kimi-k2-5`, `zai-org-glm-5`) **non sono interessati** e funzionano senza il prefisso. Il prefisso `venice-` è sicuro da usare su qualsiasi modello dato che Venice lo rimuove sempre, ma è richiesto solo per i modelli Claude in Cursor.
</Info>
