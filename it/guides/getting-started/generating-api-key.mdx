L'API di Venice è protetta da API key. Per iniziare a usare l'API Venice, devi prima generare una nuova key. Segui questi passi per iniziare.

<Steps>
  <Step title="Visita la pagina delle impostazioni API">
    Per raggiungere la pagina delle impostazioni API, visita https://venice.ai/settings/api. Questa pagina è accessibile cliccando "API" nella toolbar a sinistra, o cliccando "API" nelle impostazioni utente.

    All'interno di questa dashboard puoi visualizzare i tuoi saldi Diem e USD, il tuo API Tier, l'utilizzo API e le tue API Key.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="Panoramica API" />
    </Frame>
  </Step>

  <Step title='Clicca "Generate New API Key"'>
    Scorri la dashboard e seleziona "Generate New API Key". Ti verrà presentata una lista di opzioni.

    * **Description:** Usata per nominare la tua API key

    * **API Key Type:**

      * Le key "Admin" possono eliminare o generare ulteriori API key in modo programmatico.

      * Le key "Inference Only" sono autorizzate solo a eseguire inferenza.

    * **Expires at:** Puoi scegliere di impostare una data di scadenza per l'API key dopo la quale cesserà di funzionare. Per impostazione predefinita non viene impostata una data e la key funzionerà a tempo indeterminato.

    * **Epoch Consumption Limits:** Ti permette di creare limiti di utilizzo API per la singola API key. Puoi scegliere di limitare l'importo in Diem o in USD consentito entro una determinata epoca (24h).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Genera nuova API Key" />
    </Frame>
  </Step>

  <Step title="Genera la key">
    Cliccando Generate ti verrà mostrata l'API key.

    <Warning>
      **Importante:** Questa key viene mostrata solo una volta. Assicurati di copiarla e conservarla in un luogo sicuro. Se la perdi, dovrai eliminarla e crearne una nuova.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="La tua API Key" />
    </Frame>
  </Step>
</Steps>
