L'API di Venice è protetta tramite API key. Per iniziare a usare l'API Venice, devi prima generare una nuova chiave. Segui questi passaggi per iniziare.

<Steps>
  <Step title="Visita la pagina delle impostazioni API">
    Per arrivare alla pagina delle impostazioni API, visita https://venice.ai/settings/api. Questa pagina è accessibile cliccando "API" nella barra degli strumenti a sinistra o cliccando "API" all'interno delle impostazioni utente.

    All'interno di questa dashboard puoi visualizzare i tuoi saldi Diem e USD, il tuo API Tier, l'utilizzo dell'API e le tue API key.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API Overview" />
    </Frame>
  </Step>

  <Step title="Clicca su Generate New API Key">
    Scorri la dashboard e seleziona "Generate New API Key". Ti verrà presentato un elenco di opzioni.

    * **Description:** usato per dare un nome alla tua API key

    * **API Key Type:**

      * Le chiavi "Admin" possono eliminare o generare API key aggiuntive in modo programmatico.

      * Le chiavi "Inference Only" possono eseguire solo inferenza.

    * **Expires at:** puoi scegliere di impostare una data di scadenza per l'API key, dopo la quale smetterà di funzionare. Per impostazione predefinita non viene impostata alcuna data e la chiave funziona a tempo indeterminato.

    * **Epoch Consumption Limits:** consente di creare limiti per l'utilizzo dell'API dalla singola API key. Puoi scegliere di limitare l'importo in Diem o USD consentito entro una determinata epoca (24 ore).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Generate New API Key" />
    </Frame>
  </Step>

  <Step title="Genera la chiave">
    Cliccando su Generate verrà mostrata l'API key.

    <Warning>
      **Importante:** questa chiave viene mostrata una sola volta. Assicurati di copiarla e conservarla in un luogo sicuro. Se la perdi, dovrai eliminarla e crearne una nuova.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Your API Key" />
    </Frame>
  </Step>
</Steps>
