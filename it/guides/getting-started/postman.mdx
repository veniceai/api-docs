## Panoramica

Venice fornisce una completa collection Postman che permette agli sviluppatori di esplorare e testare tutte le capacità della nostra API. Questa collection include richieste pre-configurate, esempi e variabili d'ambiente per aiutarti a iniziare rapidamente con i servizi AI di Venice.

## Accedere alla collection

La nostra collection Postman ufficiale è disponibile nel Venice AI Workspace:

* [Venice AI Postman Workspace](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman Examples](https://postman.venice.ai/)

## Funzionalità della collection

- **Richieste pronte all'uso**: Chiamate API pre-configurate per tutti gli endpoint Venice
- **Template di ambiente**: Variabili d'ambiente strutturate correttamente
- **Esempi di richiesta**: Esempi di utilizzo reale per ogni endpoint
- **Esempi di risposta**: Risposte di esempio per aiutarti a comprendere l'output dell'API
- **Documentazione**: Documentazione inline per ogni richiesta

## Per iniziare

<Steps>
   <Step title="Fai il fork della collection">
      - Vai al Venice AI Workspace
      - Clicca "Fork" per creare la tua copia della collection
      - Scegli la destinazione del workspace
   </Step>

   <Step title="Configura il tuo ambiente">
      - Crea un nuovo ambiente in Postman
      - Aggiungi la tua API key Venice
      - Configura la base URL: `https://api.venice.ai/api/v1`
   </Step>

   <Step title="Fai la tua prima richiesta">
      - Seleziona una richiesta qualsiasi dalla collection
      - Assicurati che il tuo ambiente sia selezionato
      - Clicca "Send" per testare l'API
   </Step>
</Steps>

## Endpoint disponibili

La collection include esempi per tutti gli endpoint dell'API Venice:

- Generazione di testo
- Generazione di immagini
- Informazioni sul modello
- Upscaling immagini
- Configurazione System Prompt

## Best practice

- Mantieni sicura la tua API key e non condividerla mai
- Usa variabili d'ambiente per le informazioni sensibili
- Testa le risposte nella console Postman prima dell'implementazione
- Esamina le risposte di esempio per le strutture dati attese

<Note>*Nota: La collection Postman viene aggiornata regolarmente per riflettere le ultime modifiche e funzionalità dell'API.*</Note>
