## Catalogo voci

Le voci sono **specifiche per modello**. Il valore `voice` che passi deve provenire dal catalogo
del `model` che hai selezionato. Scegli un modello qui sotto per sfogliarne le voci.

<div id="tts-voice-picker-placeholder">Loading voices...</div>

<Note>
Gli ID voce sono case-sensitive e **validi solo per il `model` corrispondente**. Passa
entrambi i campi insieme nel payload della richiesta. Consulta l'
[API Audio Speech](/api-reference/endpoint/audio/speech) per esempi.
</Note>

### Esempio di richiesta

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

Per cambiare modello, modifica **sia** `model` sia `voice` con una coppia presa dal modello selezionato sopra.
