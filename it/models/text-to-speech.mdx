## Catalogo voci

Le voci sono **specifiche per modello**. La `voice` che passi deve provenire dal catalogo
del `model` selezionato. Scegli un modello qui sotto per sfogliare le sue voci.

<div id="tts-voice-picker-placeholder">Caricamento voci...</div>

<Note>
Gli ID delle voci sono case-sensitive e **validi solo per il `model` corrispondente**. Passa
entrambi i campi insieme nel payload della richiesta. Consulta l'
[API Audio Speech](/it/api-reference/endpoint/audio/speech) per esempi.
</Note>

### Richiesta di esempio

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

Per cambiare modello, modifica **sia** `model` sia `voice` con una coppia tratta dal
modello selezionato sopra.
