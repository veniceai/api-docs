Venice admite capacidades adicionales dentro de sus modelos que pueden activarse mediante la entrada `venice_parameters` en el endpoint de chat completions.

En ciertos casos, puede que estés utilizando un cliente que no te permita modificar el cuerpo de la solicitud. Para esas plataformas, puedes utilizar la función Model Feature Suffix de Venice para pasar flags a través del ID del modelo.

## Sintaxis

El Model Feature Suffix sigue este patrón:

```
<model_id>:<parameter>=<value>
```

Para múltiples parámetros, encadénalos con `&`:

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## Ejemplos

### Establecer la búsqueda web en automático
```
default:enable_web_search=auto
```

### Activar la búsqueda web y desactivar el system prompt
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### Activar la búsqueda web y añadir citas a la respuesta
```
default:enable_web_search=on&enable_web_citations=true
```

### Activar la búsqueda web con scraping de página completa
```
default:enable_web_search=on&enable_web_scraping=true
```

### Usar un personaje
```
default:character_slug=alan-watts
```

### Ocultar los bloques de pensamiento en una respuesta de modelo de razonamiento
```
qwen3-4b:strip_thinking_response=true
```

### Desactivar el pensamiento en los modelos de razonamiento compatibles
Algunos modelos de razonamiento (como Qwen 3) admiten desactivar el proceso de pensamiento. Puedes activarlo usando el sufijo de abajo:
```
qwen3-4b:disable_thinking=true
```

### Añadir resultados de búsqueda web a una respuesta en streaming
Esto activará la búsqueda web, añadirá citas al cuerpo de la respuesta e incluirá los resultados de búsqueda en el stream como mensaje final de la respuesta.

Puedes ver un ejemplo de esto en nuestra [Colección de Postman aquí](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041).

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
