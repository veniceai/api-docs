Venice admite capacidades adicionales en sus modelos que pueden activarse mediante el parámetro `venice_parameters` en el endpoint de chat completions.

En ciertos casos, puedes estar usando un cliente que no te permite modificar el cuerpo de la solicitud. Para esas plataformas, puedes utilizar la función Model Feature Suffix de Venice para pasar opciones a través del ID del modelo.

## Sintaxis

El Model Feature Suffix sigue este patrón:

```
<model_id>:<parameter>=<value>
```

Para varios parámetros, encadénalos con `&`:

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## Ejemplos

### Para configurar la búsqueda web en Auto
```
default:enable_web_search=auto
```

### Para activar la búsqueda web y desactivar el system prompt
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### Para activar la búsqueda web y añadir citas a la respuesta
```
default:enable_web_search=on&enable_web_citations=true
```

### Para activar la búsqueda web con scraping de página completa
```
default:enable_web_search=on&enable_web_scraping=true
```

### Para usar un personaje
```
default:character_slug=alan-watts
```

### Para ocultar los bloques de razonamiento en la respuesta de un modelo de razonamiento
```
qwen3-4b:strip_thinking_response=true
```

### Para desactivar el razonamiento en modelos de razonamiento compatibles
Algunos modelos de razonamiento (como Qwen 3) permiten desactivar el proceso de razonamiento. Puedes activarlo usando el siguiente sufijo:
```
qwen3-4b:disable_thinking=true
```

### Para añadir resultados de búsqueda web a una respuesta en streaming
Esto activará la búsqueda web, añadirá citas al cuerpo de la respuesta e incluirá los resultados de búsqueda en el stream como el mensaje de respuesta final.

Puedes ver un ejemplo en nuestra [colección de Postman aquí](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041).

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
