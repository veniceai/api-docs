Llama a `/video/quote` para obtener una estimación de precio, luego consulta `/video/retrieve` con el `queue_id` devuelto hasta que se complete.

Los modelos privados también devuelven un `download_url` para el video terminado. Es una URL de entrega de corta vida (no pasa nada por reintentar unas cuantas veces si una descarga se corta); consulta la [guía de generación de video](/es/guides/media/video-generation#private-download-links) para más detalles y el `DELETE` opcional para mayor privacidad.

### Seedance 2.0

Para los modelos `seedance-2-0-*` (texto-a-video, imagen-a-video, referencia-a-video, además de las variantes `-fast-*`), consulta la [guía de Seedance 2.0](/es/guides/media/seedance-2-0) para conocer el modelo de cuatro flujos (Reference / Edit / Extend / Stitch), los patrones canónicos de prompt, los límites de entrada multimodal y los detalles de precio.

### Consentimiento de Seedance

Cuando una solicitud de Seedance imagen-a-video o referencia-a-video contiene un rostro humano, la API devuelve un `409 needs_consent` con el texto de la política, y debes reenviar la misma solicitud con una atestación `consents.seedance`. Consulta la [guía de consentimiento de medios faciales de Seedance](/es/guides/media/seedance-face-consent) para conocer el flujo completo, el objeto de consentimiento, la deduplicación y la revocación.

### Escalado de video

Para el modelo `topaz-video-upscale`, usa `upscale_factor` (1, 2 o 4) en lugar de `resolution`, y proporciona un `video_url`. La duración y los FPS se detectan automáticamente desde el archivo de video. Consulta la [guía de escalado de video](/es/guides/media/video-upscaling) para más detalles y ejemplos.
