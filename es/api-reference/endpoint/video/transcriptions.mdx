"og:description": "Documentación de la API de transcripción de video de Venice para extraer texto del audio del video."
---

Envía la URL accesible públicamente de un video en `url`. Opcionalmente, establece `response_format` como `json` (predeterminado) o `text` según quieras una salida estructurada o el texto plano de la transcripción.
