<Warning>Esta es una API experimental. El formato de la solicitud y la respuesta puede cambiar sin previo aviso.</Warning>

Sube un archivo de documento mediante multipart/form-data usando el campo `file`. Los formatos compatibles incluyen **PDF**, **DOCX**, **XLSX** y archivos de **texto plano** (hasta 25 MB).

Configura `response_format` como `json` (predeterminado) para obtener una salida estructurada con el texto extraído y el recuento de tokens, o como `text` para el texto extraído sin procesar.

**Privacidad:** el análisis de texto se ejecuta íntegramente en memoria en la infraestructura de Venice con cero retención de datos. Tus documentos se procesan y se descartan inmediatamente: no se almacena ni se registra ningún contenido.

**Precio:** $0,01 por solicitud.

### Ejemplo (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/text-parser \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -F "file=@document.pdf" \
  -F "response_format=json"
```
