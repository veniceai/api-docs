<Warning>Esta es una API experimental. El formato de la solicitud y la respuesta puede cambiar sin previo aviso.</Warning>

Envía una consulta de búsqueda en el campo `query`. La API devuelve resultados estructurados que incluyen títulos, URLs, fragmentos de contenido y fechas.

**Proveedores de búsqueda:**
- `brave` (predeterminado) — Brave Search con Zero Data Retention (ZDR). El proveedor de búsqueda nunca almacena ni registra las consultas.
- `google` — Google Search con consultas anonimizadas. Las búsquedas se redirigen a través de la infraestructura de Venice para que tu identidad no quede asociada a la petición enviada a Google. Venice no almacena ni registra las consultas.

**Precio:** $0,01 por solicitud.

### Ejemplo (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/search \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "latest news about AI"}'
```
