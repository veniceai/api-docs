Llama a `/audio/quote` para estimar el coste y, a continuación, consulta `/audio/retrieve` con el `queue_id` devuelto hasta que finalice la generación. Si conservas los medios generados tras la recuperación, llama a `/audio/complete` una vez los hayas descargado.
