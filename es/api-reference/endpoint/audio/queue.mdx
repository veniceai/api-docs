Llama a `/audio/quote` para estimar el coste, luego consulta `/audio/retrieve` con el `queue_id` devuelto hasta que termine la generación. Si conservas el contenido generado tras recuperarlo, llama a `/audio/complete` cuando lo hayas descargado.
