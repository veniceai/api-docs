<Info>
**Precio:** el precio de la edición de imágenes varía según el modelo. El modelo predeterminado (`qwen-edit`) cuesta **$0,04 por edición**. Consulta el [endpoint de modelos](/es/api-reference/endpoint/models/list) para ver todos los modelos de edición disponibles y sus precios.
</Info>

## Niveles de calidad

Algunos modelos de edición aceptan un parámetro opcional `quality` (`low`, `medium`, `high`) que intercambia fidelidad visual por coste. Actualmente lo admite `gpt-image-2-edit`; otros modelos ignoran el parámetro.

```json
{
  "model": "gpt-image-2-edit",
  "image": "iVBORw0KGgo...",
  "prompt": "change the sky to a sunrise",
  "resolution": "2K",
  "quality": "medium"
}
```

Si omites `quality`, el modelo usa su nivel predeterminado (`high` para `gpt-image-2-edit`). Los precios por nivel (1K/2K/4K × low/medium/high) viven bajo `model_spec.pricing.quality` en el [endpoint de modelos](/es/api-reference/endpoint/models/list) y se listan en la [visión general de precios](/es/overview/pricing).

<Info>
La facturación según calidad es el comportamiento por defecto para todos los clientes de la API y los SDK — no se requiere ninguna cabecera para activarlo. Las solicitudes que omiten `quality` se cobran al nivel predeterminado del modelo (`high` para `gpt-image-2-edit`).
</Info>
