Devuelve la lista ordenada alfabéticamente de slugs de redes admitidas por el proxy RPC crypto de Venice. Úsala para descubrir qué valores de `:network` son válidos para `POST /crypto/rpc/{network}`.

Este endpoint es **público** — no requiere API key ni autenticación con cartera.

## Forma de la respuesta

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

La lista es autoritativa: si un slug no aparece aquí, el endpoint del proxy devuelve `400 Unsupported RPC network`.
