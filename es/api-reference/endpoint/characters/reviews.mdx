## Endpoint experimental

<Warning>
Este es un endpoint experimental y puede estar sujeto a cambios.
</Warning>

## Qué devuelve

Este endpoint devuelve reseñas públicas paginadas para un único personaje.

- Usa el parámetro de ruta `slug` para identificar al personaje.
- Usa los parámetros de consulta `page` y `pageSize` para paginar las reseñas.
- Los metadatos de paginación se devuelven tanto en el cuerpo de la respuesta como en las cabeceras `x-pagination-*`.
