Empieza a usar la API de Venice en minutos. Genera una clave de API, realiza tu primera solicitud y comienza a construir.

## Inicio rápido

<Steps>
  <Step title="Obtén tu clave de API">
    Ve a la [configuración de la API de Venice](https://venice.ai/settings/api) y genera una nueva clave de API.

    Para ver un tutorial detallado, consulta la [guía de claves de API](/guides/getting-started/generating-api-key).
  </Step>
  <Step title="Configura tu clave de API">
    Añade tu clave de API a tu entorno. Puedes exportarla en tu shell:

    ```bash
    export VENICE_API_KEY='your-api-key-here'
    ```

    O añadirla a un archivo `.env` en tu proyecto:

    ```bash
    VENICE_API_KEY=your-api-key-here
    ```
  </Step>
  <Step title="Instala el SDK">
    Venice es compatible con OpenAI, por lo que puedes usar el SDK de OpenAI. Si prefieres usar cURL o solicitudes HTTP sin procesar, puedes omitir este paso.

    <CodeGroup>

    ```bash Python
    pip install openai
    ```

    ```bash Node.js
    npm install openai
    ```

    </CodeGroup>
  </Step>
  <Step title="Envía tu primera solicitud">
    <CodeGroup>

    ```python Python
    import os
    from openai import OpenAI
    
    client = OpenAI(
        api_key=os.getenv("VENICE_API_KEY"),
        base_url="https://api.venice.ai/api/v1"
    )
    
    completion = client.chat.completions.create(
        model="zai-org-glm-5",
        messages=[
            {"role": "system", "content": "You are a helpful AI assistant"},
            {"role": "user", "content": "Why is privacy important?"}
        ]
    )
    
    print(completion.choices[0].message.content)
    ```

    ```javascript Node.js
    import OpenAI from 'openai';
    
    const client = new OpenAI({
        apiKey: process.env.VENICE_API_KEY,
        baseURL: 'https://api.venice.ai/api/v1'
    });
    
    const completion = await client.chat.completions.create({
        model: 'zai-org-glm-5',
        messages: [
            { role: 'system', content: 'You are a helpful AI assistant' },
            { role: 'user', content: 'Why is privacy important?' }
        ]
    });
    
    console.log(completion.choices[0].message.content);
    ```

    ```bash cURL
    curl https://api.venice.ai/api/v1/chat/completions \
      -H "Authorization: Bearer $VENICE_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "zai-org-glm-5",
        "messages": [
          {"role": "system", "content": "You are a helpful AI assistant"},
          {"role": "user", "content": "Why is privacy important?"}
        ]
      }'
    ```

    </CodeGroup>

    **Roles de los mensajes:**

    - `system` - Instrucciones sobre cómo debe comportarse el modelo
    - `user` - Tus indicaciones o preguntas
    - `assistant` - Respuestas previas del modelo (para conversaciones de varios turnos)
    - `tool` - Resultados de llamadas a funciones (cuando se usan herramientas)
  </Step>
  <Step title="Cambia de modelo modificando el ID del modelo">
    Cada solicitud incluye un ID de `model`. Para usar un modelo diferente, cambia el valor de `model` en tu solicitud. Opciones populares:

    - `zai-org-glm-5` - Modelo predeterminado para la mayoría de los casos de uso
    - `kimi-k2-6` - Razonamiento sólido para tareas más complejas
    - `claude-opus-4-8` - Modelo de alta inteligencia para tareas complejas
    - `venice-uncensored-1-2` - El modelo sin censura de Venice

    <Card title="Ver todos los modelos" icon="database" href="/overview/models">
      Explora la lista completa de modelos con precios, capacidades y límites de contexto
    </Card>
  </Step>
  <Step title="Usa los parámetros de Venice">
    Puedes optar por habilitar funciones específicas de Venice, como la búsqueda web, mediante `venice_parameters`:

    <CodeGroup>

    ```python Python
    import os
    from openai import OpenAI
    
    client = OpenAI(
        api_key=os.environ.get("VENICE_API_KEY"),
        base_url="https://api.venice.ai/api/v1"
    )
    
    completion = client.chat.completions.create(
        model="zai-org-glm-5",
        messages=[
            {"role": "user", "content": "What are the latest developments in AI?"}
        ],
        extra_body={
            "venice_parameters": {
                "enable_web_search": "auto",
                "include_venice_system_prompt": True
            }
        }
    )
    
    print(completion.choices[0].message.content)
    ```

    ```javascript Node.js
    import OpenAI from 'openai';
    
    const client = new OpenAI({
        apiKey: process.env.VENICE_API_KEY,
        baseURL: 'https://api.venice.ai/api/v1'
    });
    
    const completion = await client.chat.completions.create({
        model: 'zai-org-glm-5',
        messages: [
            { role: 'user', content: 'What are the latest developments in AI?' }
        ],
        venice_parameters: {
            enable_web_search: 'auto',
            include_venice_system_prompt: true
        }
    });
    
    console.log(completion.choices[0].message.content);
    ```

    ```bash cURL
    curl https://api.venice.ai/api/v1/chat/completions \
      -H "Authorization: Bearer $VENICE_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "zai-org-glm-5",
        "messages": [
          {"role": "user", "content": "What are the latest developments in AI?"}
        ],
        "venice_parameters": {
          "enable_web_search": "auto",
          "include_venice_system_prompt": true
        }
      }'
    ```

    </CodeGroup>

    Consulta todos los [parámetros disponibles](https://docs.venice.ai/api-reference/api-spec#venice-parameters).
  </Step>
  <Step title="Habilita el streaming (opcional)">
    Transmite las respuestas en tiempo real usando `stream=True`:

    <CodeGroup>

    ```python Python
    import os
    from openai import OpenAI
    
    client = OpenAI(
        api_key=os.environ.get("VENICE_API_KEY"),
        base_url="https://api.venice.ai/api/v1"
    )
    
    stream = client.chat.completions.create(
        model="zai-org-glm-5",
        messages=[{"role": "user", "content": "Write a short story about AI"}],
        stream=True
    )
    
    for chunk in stream:
        if chunk.choices and chunk.choices[0].delta.content is not None:
            print(chunk.choices[0].delta.content, end="")
    ```

    ```javascript Node.js
    import OpenAI from 'openai';
    
    const client = new OpenAI({
        apiKey: process.env.VENICE_API_KEY,
        baseURL: 'https://api.venice.ai/api/v1'
    });
    
    const stream = await client.chat.completions.create({
        model: 'zai-org-glm-5',
        messages: [{ role: 'user', content: 'Write a short story about AI' }],
        stream: true
    });
    
    for await (const chunk of stream) {
        if (chunk.choices && chunk.choices[0]?.delta?.content) {
            process.stdout.write(chunk.choices[0].delta.content);
        }
    }
    ```

    ```bash cURL
    curl https://api.venice.ai/api/v1/chat/completions \
      -H "Authorization: Bearer $VENICE_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "zai-org-glm-5",
        "messages": [
          {"role": "user", "content": "Write a short story about AI"}
        ],
        "stream": true
      }'
    ```

    </CodeGroup>
  </Step>
  <Step title="Personaliza el comportamiento de las respuestas (opcional)">
    Controla cómo responde el modelo con parámetros como temperature, max tokens y más:

    <CodeGroup>

    ```python Python
    import os
    from openai import OpenAI
    
    client = OpenAI(
        api_key=os.environ.get("VENICE_API_KEY"),
        base_url="https://api.venice.ai/api/v1"
    )
    
    completion = client.chat.completions.create(
        model="zai-org-glm-5",
        messages=[
            {"role": "system", "content": "You are a creative storyteller"},
            {"role": "user", "content": "Tell me a creative story"}
        ],
        temperature=0.8,
        max_tokens=500,
        top_p=0.9,
        frequency_penalty=0.5,
        presence_penalty=0.5,
        extra_body={
            "venice_parameters": {
                "include_venice_system_prompt": False
            }
        }
    )
    
    print(completion.choices[0].message.content)
    ```

    ```javascript Node.js
    import OpenAI from 'openai';
    
    const client = new OpenAI({
        apiKey: process.env.VENICE_API_KEY,
        baseURL: 'https://api.venice.ai/api/v1'
    });
    
    const completion = await client.chat.completions.create({
        model: 'zai-org-glm-5',
        messages: [
            { role: 'system', content: 'You are a creative storyteller' },
            { role: 'user', content: 'Tell me a creative story' }
        ],
        temperature: 0.8,
        max_tokens: 500,
        top_p: 0.9,
        frequency_penalty: 0.5,
        presence_penalty: 0.5,
        venice_parameters: {
            include_venice_system_prompt: false
        }
    });
    
    console.log(completion.choices[0].message.content);
    ```

    ```bash cURL
    curl https://api.venice.ai/api/v1/chat/completions \
      -H "Authorization: Bearer $VENICE_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "zai-org-glm-5",
        "messages": [
          {"role": "system", "content": "You are a creative storyteller"},
          {"role": "user", "content": "Tell me a creative story"}
        ],
        "temperature": 0.8,
        "max_tokens": 500,
        "top_p": 0.9,
        "frequency_penalty": 0.5,
        "presence_penalty": 0.5,
        "stream": false,
        "venice_parameters": {
          "include_venice_system_prompt": false
        }
      }'
    ```

    </CodeGroup>

    Consulta la [documentación de Chat Completions](/api-reference/endpoint/chat/completions) para más información sobre todos los parámetros compatibles.
  </Step>
</Steps>
