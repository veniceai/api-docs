## Resumen

Venice ofrece una completa colección de Postman que permite a los desarrolladores explorar y probar todas las capacidades de nuestra API. Esta colección incluye solicitudes preconfiguradas, ejemplos y variables de entorno para ayudarte a empezar rápidamente con los servicios de IA de Venice.

## Acceso a la colección

Nuestra colección oficial de Postman está disponible en el Venice AI Workspace:

* [Venice AI Postman Workspace](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman Examples](https://postman.venice.ai/)

## Características de la colección

- **Solicitudes listas para usar**: llamadas a la API preconfiguradas para todos los endpoints de Venice
- **Plantillas de entorno**: variables de entorno correctamente estructuradas
- **Ejemplos de solicitudes**: ejemplos de uso reales para cada endpoint
- **Ejemplos de respuesta**: respuestas de ejemplo para ayudarte a entender la salida de la API
- **Documentación**: documentación en línea para cada solicitud

## Empezar

<Steps>
   <Step title="Haz fork de la colección">
      - Navega al Venice AI Workspace
      - Haz clic en "Fork" para crear tu propia copia de la colección
      - Elige el destino en tu workspace
   </Step>

   <Step title="Configura tu entorno">
      - Crea un nuevo entorno en Postman
      - Añade tu API key de Venice
      - Configura la URL base: `https://api.venice.ai/api/v1`
   </Step>

   <Step title="Haz tu primera solicitud">
      - Selecciona cualquier solicitud de la colección
      - Asegúrate de que tu entorno esté seleccionado
      - Haz clic en "Send" para probar la API
   </Step>
</Steps>

## Endpoints disponibles

La colección incluye ejemplos para todos los endpoints de la API de Venice:

- Generación de texto
- Generación de imágenes
- Información de modelos
- Escalado de imágenes
- Configuración del system prompt

## Mejores prácticas

- Mantén tu API key segura y nunca la compartas
- Usa variables de entorno para información sensible
- Prueba las respuestas en la consola de Postman antes de implementarlas
- Revisa las respuestas de ejemplo para conocer las estructuras de datos esperadas

<Note>*Nota: la colección de Postman se actualiza regularmente para reflejar los últimos cambios y funciones de la API.*</Note>
