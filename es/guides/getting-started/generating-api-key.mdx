La API de Venice está protegida con API keys. Para empezar a usar la API de Venice, primero tendrás que generar una nueva clave. Sigue estos pasos para empezar.

<Steps>
  <Step title="Visita la página de ajustes de la API">
    Para acceder a la página de ajustes de la API, visita https://venice.ai/settings/api. Esta página es accesible haciendo clic en "API" en la barra lateral izquierda, o haciendo clic en "API" dentro de los ajustes de usuario.

    Desde este panel puedes ver tus saldos en Diem y USD, tu nivel de API, tu uso de la API y tus API keys.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="Vista general de la API" />
    </Frame>
  </Step>

  <Step title="Haz clic en Generate New API Key">
    Baja en el panel y selecciona "Generate New API Key". Verás una lista de opciones.

    * **Description:** se usa para nombrar tu API key

    * **API Key Type:**

      * Las claves "Admin" tienen la capacidad de eliminar o generar API keys adicionales de forma programática.

      * Las claves "Inference Only" solo pueden ejecutar inferencia.

    * **Expires at:** puedes establecer una fecha de expiración tras la cual la API key dejará de funcionar. Por defecto no se establece fecha y la clave funcionará indefinidamente.

    * **Epoch Consumption Limits:** te permite crear límites de uso de la API por cada clave. Puedes limitar el importe en Diem o USD permitido dentro de una epoch determinada (24 h).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Generar nueva API key" />
    </Frame>
  </Step>

  <Step title="Genera la clave">
    Al hacer clic en Generate se mostrará la API key.

    <Warning>
      **Importante:** esta clave solo se muestra una vez. Asegúrate de copiarla y guardarla en un lugar seguro. Si la pierdes, tendrás que eliminarla y crear una nueva.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Tu API key" />
    </Frame>
  </Step>
</Steps>
