La API de Venice está protegida con API keys. Para empezar a usar la API de Venice, primero tienes que generar una nueva clave. Sigue estos pasos para comenzar.

<Steps>
  <Step title="Visita la página de configuración de la API">
    Para ir a la página de configuración de la API, visita https://venice.ai/settings/api. Esta página es accesible haciendo clic en "API" en la barra lateral izquierda, o haciendo clic en "API" dentro de la configuración de tu usuario.

    Dentro de este dashboard, puedes ver tus saldos de Diem y USD, tu nivel de API, tu uso de API y tus API Keys.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API Overview" />
    </Frame>
  </Step>

  <Step title="Haz clic en Generate New API Key">
    Desplázate por el dashboard y selecciona "Generate New API Key". Se te presentará una lista de opciones.

    * **Description:** se usa para nombrar tu API key

    * **API Key Type:**

      * Las claves "Admin" tienen la capacidad de eliminar o generar API keys adicionales de forma programática.

      * Las claves "Inference Only" solo pueden ejecutar inferencia.

    * **Expires at:** puedes optar por establecer una fecha de expiración para la API key, después de la cual dejará de funcionar. Por defecto no se establece una fecha y la clave funcionará a perpetuidad.

    * **Epoch Consumption Limits:** te permite crear límites de uso de la API desde una API key individual. Puedes optar por limitar el importe en Diem o USD permitido dentro de una época determinada (24 h).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Generate New API Key" />
    </Frame>
  </Step>

  <Step title="Genera la clave">
    Al hacer clic en Generate se te mostrará la API key.

    <Warning>
      **Importante:** esta clave solo se muestra una vez. Asegúrate de copiarla y guardarla en un lugar seguro. Si la pierdes, tendrás que eliminarla y crear una nueva.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Your API Key" />
    </Frame>
  </Step>
</Steps>
