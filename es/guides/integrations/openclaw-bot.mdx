[OpenClaw](https://openclaw.ai) es un gateway de IA open-source y auto-hospedado que conecta plataformas de mensajería (WhatsApp, Telegram, Discord, iMessage, Slack) con modelos de IA. Venice AI está disponible como proveedor integrado, dándote acceso a modelos privados y sin censura desde cualquier canal conectado.

## Configuración

### 1. Instala OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. Ejecuta el asistente de onboarding

```bash
openclaw onboard
```

El asistente te guiará en la configuración. Cuando se te pida, selecciona **Venice AI** como tu proveedor en la lista y pega tu API key. Obtén una en [venice.ai/settings/api](https://venice.ai/settings/api) si aún no tienes.

### 3. Elige un modelo

Durante el onboarding, OpenClaw muestra todos los modelos de Venice disponibles. Algunas recomendaciones:

| Caso de uso | Modelo | Privacidad |
|---|---|---|
| General | `venice/zai-org-glm-5` | Privado |
| Razonamiento | `venice/kimi-k2-5` | Privado |
| Programación | `venice/claude-opus-4-6` | Anonimizado |
| Visión | `venice/qwen3-vl-235b-a22b` | Privado |
| Sin censura | `venice/venice-uncensored` | Privado |

Cambia tu modelo predeterminado en cualquier momento:

```bash
openclaw models set venice/zai-org-glm-5
```

Lista todos los modelos disponibles:

```bash
openclaw models list | grep venice
```

### 4. Empieza a chatear

Abre la interfaz de terminal:

```bash
openclaw tui
```

O el panel web:

```bash
openclaw dashboard
```

O conecta un canal de mensajería (WhatsApp, Telegram, Discord, etc.):

```bash
openclaw channels login
openclaw gateway
```

## Modos de privacidad

Los modelos de Venice en OpenClaw siguen los mismos [niveles de privacidad](/es/overview/privacy) que la API de Venice:

- Los modelos **privados** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) se ejecutan en la flota de GPUs de Venice. Los prompts nunca se almacenan ni se registran.
- Los modelos **anonimizados** (Claude, GPT, Gemini, Grok) se proxifican a través de Venice eliminando toda información identificativa. El proveedor externo ve a Venice como cliente, no a ti.

## Generación de imagen y video

Instala la skill de Venice AI Media para generación de imagen y video:

```bash
openclaw skills install nhannah/venice-ai-media
```
