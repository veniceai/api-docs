[OpenClaw](https://openclaw.ai) es un gateway de IA de código abierto y autoalojado que conecta plataformas de mensajería (WhatsApp, Telegram, Discord, iMessage, Slack) con modelos de IA. Venice AI está disponible como proveedor integrado, dándote acceso a modelos privados y sin censura desde cualquier canal conectado.

## Configuración

### 1. Instala OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. Ejecuta el asistente de onboarding

```bash
openclaw onboard
```

El asistente te guiará por la configuración. Cuando se te pida, selecciona **Venice AI** como proveedor de la lista y pega tu API key. Obtén una en [venice.ai/settings/api](https://venice.ai/settings/api) si aún no tienes una.

### 3. Elige un modelo

Durante el onboarding, OpenClaw muestra todos los modelos disponibles de Venice. Algunas recomendaciones:

| Caso de uso | Modelo | Privacidad |
|---|---|---|
| General | `venice/zai-org-glm-5` | Private |
| Razonamiento | `venice/kimi-k2-5` | Private |
| Programación | `venice/claude-opus-4-6` | Anonymized |
| Visión | `venice/qwen3-vl-235b-a22b` | Private |
| Sin censura | `venice/venice-uncensored` | Private |

Cambia tu modelo predeterminado en cualquier momento:

```bash
openclaw models set venice/zai-org-glm-5
```

Lista todos los modelos disponibles:

```bash
openclaw models list | grep venice
```

### 4. Empieza a chatear

Abre la TUI:

```bash
openclaw tui
```

O el dashboard web:

```bash
openclaw dashboard
```

O conecta un canal de mensajería (WhatsApp, Telegram, Discord, etc.):

```bash
openclaw channels login
openclaw gateway
```

## Modos de privacidad

Los modelos de Venice en OpenClaw siguen los mismos [niveles de privacidad](/overview/privacy) que la API de Venice:

- Los modelos **Private** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) se ejecutan en la flota de GPU de Venice. Los prompts nunca se almacenan ni se registran.
- Los modelos **Anonymized** (Claude, GPT, Gemini, Grok) se enrutan a través de Venice eliminando toda la información identificativa. El proveedor externo ve a Venice como cliente, no a ti.

## Generación de imagen y vídeo

Instala la skill Venice AI Media para generación de imagen y vídeo:

```bash
openclaw skills install nhannah/venice-ai-media
```
