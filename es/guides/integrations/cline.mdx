[Cline](https://cline.bot/) es un asistente de programación con IA para VS Code que puede leer y editar archivos, ejecutar comandos y trabajar en tareas de desarrollo de varios pasos. Venice funciona a través de la configuración de proveedor compatible con OpenAI de Cline.

  <Card title="Integración con VS Code" icon="code">
    Usa modelos de Venice sin salir de tu editor
  </Card>
  <Card title="Compatible con OpenAI" icon="plug">
    Conéctate mediante el endpoint `/chat/completions` de Venice
  </Card>
  <Card title="Flexibilidad de modelos" icon="microchip">
    Elige cualquier modelo de texto compatible de Venice
  </Card>
</CardGroup>

## Requisitos previos

- Una API key de Venice desde [venice.ai/settings/api](https://venice.ai/settings/api)
- [Visual Studio Code](https://code.visualstudio.com/) instalado
- La [extensión Cline](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev) instalada en VS Code

---

## Configuración

<Steps>
  <Step title="Abre la configuración de Cline">
    Selecciona el icono de Cline en la barra de actividad de VS Code y luego selecciona el icono de engranaje para abrir la configuración de Cline.
  </Step>

  <Step title="Elige el proveedor de API">
    Establece **API Provider** en **OpenAI Compatible**.
  </Step>

  <Step title="Configura Venice">
    Introduce los siguientes valores:

    | Ajuste | Valor |
    | --- | --- |
    | Base URL | `https://api.venice.ai/api/v1` |
    | API Key | Tu API key de Venice |
    | Model ID | Un ID de modelo del [catálogo de modelos de texto](/models/text) |

    Por ejemplo, usa `zai-org-glm-5-1` como Model ID.

    Introduce la API key directamente, sin el prefijo `Bearer`. Mantén la Base URL en `/api/v1`; Cline añade la ruta `/chat/completions`.
  </Step>

  <Step title="Configura las capacidades del modelo">
    Expande **Model Configuration** y establece la ventana de contexto y los tokens de salida máximos para que coincidan con el modelo seleccionado. Habilita las capacidades que soporta el modelo, incluidas la entrada de imágenes o el uso del ordenador cuando corresponda.

    Comprueba el modelo seleccionado en el [catálogo de modelos de texto](/models/text) antes de habilitar capacidades opcionales.
  </Step>

  <Step title="Guarda y prueba">
    Guarda la configuración del proveedor, abre un proyecto y dale a Cline una pequeña tarea como:

    ```txt
    Review the current file and suggest one safe improvement.
    ```

    Revisa cada permiso de archivo o comando antes de aprobarlo.
  </Step>
</Steps>

## Verifica la API de Venice

Si Cline no puede conectarse, confirma la API key y los IDs de modelo disponibles de forma independiente:

```bash
curl https://api.venice.ai/api/v1/models \
  -H "Authorization: Bearer $VENICE_API_KEY"
```

Copia el `id` de un modelo exactamente al campo **Model ID** de Cline.

---

## Notas de configuración

- Usa `https://api.venice.ai/api/v1` como Base URL, no el endpoint completo `/chat/completions`.
- Los IDs de modelo pueden cambiar con el tiempo. Usa el [catálogo de modelos de texto](/models/text) o `GET /models` en lugar de depender de un ejemplo antiguo.
- Los ajustes de ventana de contexto y tokens de salida de Cline afectan a cómo presupuesta las solicitudes. Ajústalos al modelo seleccionado para obtener los mejores resultados.
- Mantén tu API key de Venice privada. No la pegues en archivos fuente ni la subas a git.

## Solución de problemas

<AccordionGroup>
  <Accordion title="Error 401 o de autenticación">
    Confirma que la API key proviene de [venice.ai/settings/api](https://venice.ai/settings/api). Pega la clave sin procesar en Cline, sin `Bearer` ni espacios adicionales.
  </Accordion>
  <Accordion title="Modelo no encontrado">
    Confirma que el Model ID aparece en el [catálogo de modelos de texto](/models/text) o en la respuesta de `GET /models`, y luego cópialo exactamente.
  </Accordion>
  <Accordion title="Error de conexión o de endpoint">
    Confirma que **API Provider** es **OpenAI Compatible** y que la Base URL es exactamente `https://api.venice.ai/api/v1`.
  </Accordion>
  <Accordion title="Cline no puede usar herramientas ni editar archivos">
    Abre **Model Configuration**, confirma que el modelo seleccionado soporta las capacidades requeridas y habilita el uso del ordenador si Cline expone esa opción para el modelo.
  </Accordion>
</AccordionGroup>
