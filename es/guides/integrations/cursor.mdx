[Cursor](https://www.cursor.com/) es un editor de código impulsado por IA. Puedes usarlo con Venice AI para obtener acceso privado y sin censura a una amplia gama de modelos.

## Configuración

<Steps>
  <Step title="Obtén tu API key">
    Genera una clave desde [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="Configura Cursor">
    Abre **Cursor Settings** (icono de engranaje), desplázate hasta **Models** y haz clic en **Add Model**.

    En **OpenAI Compatible**, introduce:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: tu API key de Venice
  </Step>

  <Step title="Añade modelos">
    En el campo de nombre del modelo, escribe el ID del modelo de Venice y pulsa Enter. Añade cada modelo que quieras usar. Por ejemplo:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (consulta la [nota del prefijo más abajo](#the-venice--model-prefix))

    Consulta el [catálogo de modelos](/models/text) para todos los IDs de modelo disponibles.
  </Step>

  <Step title="Selecciona y verifica">
    Abre un nuevo chat y usa el **selector de modelo en la parte inferior del chat** para elegir uno de los modelos de Venice que acabas de añadir. Envía un mensaje de prueba. Si obtienes respuesta, ya está todo listo.
  </Step>
</Steps>

## El prefijo de modelo `venice-`

<Warning>
**Los modelos Claude requieren el prefijo `venice-` cuando se usan en Cursor.**

Cursor reescribe las solicitudes para los modelos `claude-*` al formato nativo de Anthropic, que es incompatible con Venice. Anteponer el prefijo `venice-` al ID del modelo evita esta reescritura. Venice elimina el prefijo automáticamente.
</Warning>

| Modelo | ID estándar | ID en Cursor |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
Los modelos que no son Claude (p. ej., `minimax-m25`, `kimi-k2-5`, `zai-org-glm-5`) **no se ven afectados** y funcionan sin el prefijo. El prefijo `venice-` se puede usar de forma segura en cualquier modelo ya que Venice siempre lo elimina, pero solo es obligatorio para los modelos Claude en Cursor.
</Info>
