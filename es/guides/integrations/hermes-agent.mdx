[Hermes Agent](https://hermes-agent.nousresearch.com) es un agente de IA de código abierto y autoalojado creado por [Nous Research](https://nousresearch.com). Cuenta con memoria persistente, creación autónoma de skills y un bucle de aprendizaje integrado que se vuelve más capaz cuanto más tiempo se ejecuta. Apúntalo a la API de Venice y tu agente obtendrá acceso a más de 230 modelos y herramientas para texto, imagen, vídeo, audio, embeddings y más.

## ¿Por qué Venice + Hermes Agent?

La API de Venice ofrece a tu Hermes Agent acceso a toda la plataforma Venice a través de un único endpoint compatible con OpenAI.

| Capacidad | Lo que obtienes |
|---|---|
| **Texto y chat** | Modelos privados y anonimizados (GLM, Qwen, Claude, GPT y más) |
| **Imagen** | Generación, edición, escalado y eliminación de fondo |
| **Vídeo** | Generación y transcripción |
| **Audio** | Síntesis de voz (TTS), generación de música y voz a texto |
| **Embeddings** | Embeddings vectoriales para RAG y búsqueda semántica |
| **Herramientas** | Web scraping, búsqueda web, parsing de texto y RPC cripto |

## Configuración

### 1. Instala Hermes Agent

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>

  <Tab title="WSL2 (Windows)">
    Instala primero [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install) y ejecuta luego el mismo comando:
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>
</Tabs>

Después de la instalación, recarga tu shell:

```bash
source ~/.zshrc   # o: source ~/.bashrc
```

### 2. Configura Venice como tu proveedor

Ejecuta el asistente de configuración de modelos:

```bash
hermes model
```

Selecciona **Custom endpoint (self-hosted / VLLM / etc.)** e introduce lo siguiente cuando se te pida:

| Campo | Valor |
|---|---|
| **API base URL** | `https://api.venice.ai/api/v1` |
| **API key** | Tu API key de Venice |
| **Nombre del modelo** | Un ID de modelo de Venice (ver abajo) |

Obtén una API key en [venice.ai/settings/api](https://venice.ai/settings/api) si aún no tienes una.

<Tip>
También puedes configurar Venice directamente en `~/.hermes/config.yaml`:

```yaml
model:
  default: zai-org-glm-5
  provider: custom
  base_url: https://api.venice.ai/api/v1
  api_key: ${VENICE_API_KEY}
```

Y establece la clave en `~/.hermes/.env`:

```bash
VENICE_API_KEY=your-key-here
```
</Tip>

### 3. Elige un modelo

Cuando el asistente te pida un modelo, elige uno según tu caso de uso:

| Caso de uso | Modelo | Privacidad |
|---|---|---|
| General | `zai-org-glm-5` | Private |
| Razonamiento | `kimi-k2-6` | Private |
| Programación | `claude-opus-4-7` | Anonymized |
| Visión | `z-ai-glm-5v-turbo` | Anonymized |
| Sin censura | `venice-uncensored-1-2` | Private |

Cambia de modelo en cualquier momento sin reiniciar:

```bash
hermes model              # asistente completo
```

O cambia a mitad de sesión:

```text
/model custom:claude-opus-4-7
```

### 4. Empieza a chatear

Abre la CLI clásica o la TUI moderna:

```bash
hermes            # CLI clásica
hermes --tui      # TUI moderna (recomendada)
```

Prueba un prompt para verificar que todo funciona:

```text
Summarize this repo in 5 bullets and tell me what the main entrypoint is.
```

## Conecta plataformas de mensajería

Una vez que funcione la CLI, conecta tus apps de mensajería a través del gateway:

```bash
hermes gateway setup
```

Esto te guía para conectar Telegram, Discord, Slack, WhatsApp, Signal y otras plataformas. Tu agente se vuelve accesible desde cualquier canal conectado, todo impulsado por Venice.

```bash
hermes gateway      # inicia el gateway de mensajería
```

## Modos de privacidad

Los modelos de Venice en Hermes Agent siguen los mismos [niveles de privacidad](/overview/privacy) que la API de Venice:

- Los modelos **Private** (GLM, Qwen, DeepSeek, Llama, Venice Uncensored) se ejecutan en la flota de GPU de Venice. Los prompts nunca se almacenan ni se registran.
- Los modelos **Anonymized** (Claude, GPT, Gemini, Grok) se enrutan a través de Venice eliminando toda la información identificativa. El proveedor externo ve a Venice como cliente, no a ti.

## Venice API skills

Hermes Agent tiene un sistema de skills integrado compatible con el formato [Agent Skills](https://github.com/veniceai/skills). Venice publica skills oficiales que enseñan a tu agente a usar todos los endpoints de Venice (chat, generación de imágenes, vídeo, audio, embeddings, herramientas augment y más).

Instala las skills de Venice directamente desde GitHub:

```bash
hermes skills install veniceai/skills
```

O busca skills individuales:

```bash
hermes skills search venice
```

Hermes descubrirá cada skill por su frontmatter de `SKILL.md` y la cargará bajo demanda.

| `hermes doctor` | Diagnosticar problemas |
| `hermes update` | Actualizar a la última versión |
