## Catálogo de voces

Las voces son **específicas de cada modelo**. La `voice` que pases debe provenir del catálogo
del `model` que hayas seleccionado. Elige un modelo abajo para explorar sus voces.

<div id="tts-voice-picker-placeholder">Loading voices...</div>

<Note>
Los IDs de voz distinguen mayúsculas y minúsculas y **solo son válidos para el `model` correspondiente**. Pasa
ambos campos juntos en la carga útil de tu solicitud. Consulta la
[API de voz](/api-reference/endpoint/audio/speech) para ejemplos.
</Note>

### Ejemplo de solicitud

```bash
curl https://api.venice.ai/api/v1/audio/speech \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-kokoro",
    "voice": "af_sky",
    "input": "Hello from Venice."
  }' \
  --output speech.mp3
```

Para cambiar de modelo, cambia **tanto** `model` como `voice` a un par del
modelo seleccionado arriba.
