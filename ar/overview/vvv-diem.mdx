يمثّل VVV و DIEM طبقة التمويل على السلسلة (on-chain) الخاصة بـ Venice لواجهة الـ API. أودِع DIEM لتحصل على مخصّص استدلال يومي ثابت — **1 DIEM = 1 دولار يومياً** من رصيد Venice API — دون الحاجة إلى شراء رصيد USD مسبق الدفع لكل طلب.

| العملة | الدور |
| --- | --- |
| **VVV** | العملة الأساسية لـ Venice على شبكة Base. أودِعها لتحصل على عوائد ولصكّ DIEM. |
| **sVVV** | إيصال مقابل VVV المُودَع. اقفل sVVV بالـ [Mint Rate](https://diem-calculator.venice.ai/) الحالي لصكّ DIEM. |
| **DIEM** | وحدة حوسبة مُرمَّزة. أودِع DIEM لفتح رصيد API يومي دائم يعادل 1 دولار لكل DIEM. |

DIEM هي عملة ERC-20 على شبكة Base. يمكنك تحويلها أو تداولها أو إيداعها. وعلى خلاف رصيد USD المدفوع مسبقاً حسب الاستخدام، فإن DIEM المُودَع هو سعة تمتلكها: يتجدّد المخصّص اليومي في كل حقبة (epoch) ولا يقلّ مع نمو استخدام الشبكة.

<Info>
  للاطلاع على أسعار النماذج المخصومة من هذا المخصّص، راجع [أسعار API](/overview/pricing). وللدفع لكل طلب عبر المحفظة دون مفتاح API، راجع [x402](/guides/integrations/x402-venice-api).
</Info>

## كيف يعمل التمويل

تنفق مفاتيح API من حساب Venice المرتبط بهذا الترتيب: **DIEM**، ثم [الأرصدة المُجمّعة (bundled credits)](/overview/pricing#bundled-credits)، ثم USD.

عندما تكون DIEM هي العملة النشطة:

* يعادل تخصيصك اليومي مقدار DIEM الذي أودعتَه (1 DIEM مُودَع ← 1 دولار من رصيد API لتلك الحقبة).
* لا يُرحَّل DIEM غير المستخدم في حقبة ما إلى الحقبة التالية. ويتجدّد المخصّص عند **00:00 UTC**.
* تحتاج الحسابات إلى ما لا يقل عن **0.1** DIEM مُودَع قبل أن يصبح أي رصيد DIEM قابلاً للإنفاق.
* تُحتسب تكاليف الطلب باستخدام نفس قائمة أسعار USD المعتمدة للدفع حسب الاستخدام؛ حيث تكون DIEM هي عملة التسوية بسعر 1:1 مقابل الدولار ضمن مخصّص ذلك اليوم.

تحقّق من رصيدك الحالي ورصيد الحقبة التالية من DIEM عبر لوحة التحكم على [venice.ai/settings/api](https://venice.ai/settings/api)، أو عبر نقطة النهاية [Billing Balance](/api-reference/endpoint/billing/balance).

## احصل على DIEM وأودعه

استخدم لوحة التحكم الخاصة بالعملة على [venice.ai/token](https://venice.ai/token). قم بربط محفظة Base — لا ترسل العملات مباشرة إلى عنوان عقد.

<Steps>
  <Step title="اقتناء VVV أو DIEM">
    اشترِ VVV أو DIEM على منصة DEX على شبكة Base مثل [Aerodrome](https://aerodrome.finance) أو [Uniswap](https://app.uniswap.org)، أو استقبل DIEM محوَّلاً من محفظة أخرى.

    | الأصل | العقد (Base) |
    | --- | --- |
    | VVV | [`0xacfE6019Ed1A7Dc6f7B508C02d1b04ec88cC21bf`](https://basescan.org/token/0xacfE6019Ed1A7Dc6f7B508C02d1b04ec88cC21bf) |
    | DIEM | [`0xF4d97F2da56e8c3098f3a8D538DB630A2606a024`](https://basescan.org/token/0xF4d97F2da56e8c3098f3a8D538DB630A2606a024) |
  </Step>

  <Step title="إيداع VVV (عند صكّ DIEM)">
    على [venice.ai/token](https://venice.ai/token)، أودِع VVV لتحصل على sVVV. يكسب VVV المُودَع إصدارات (emissions). عقد الإيداع هو [`0x321b7ff75154472B18EDb199033fF4D116F340Ff`](https://basescan.org/address/0x321b7ff75154472B18EDb199033fF4D116F340Ff#code).
  </Step>

  <Step title="صكّ DIEM من sVVV (اختياري)">
    اقفل sVVV بالـ Mint Rate الحالي لصكّ DIEM. الـ Mint Rate هو مقدار sVVV المطلوب لكل DIEM، وهو يرتفع مع نمو المعروض من DIEM — يمكنك معاينته على [حاسبة DIEM](https://diem-calculator.venice.ai/).

    يظهر DIEM المصكوك في محفظتك بوصفه عملة ERC-20. ويظل sVVV المقفَل مقفلاً حتى تحرق (burn) نفس المقدار من DIEM لتحريره.
  </Step>

  <Step title="إيداع DIEM للحصول على رصيد API">
    أودِع DIEM عبر لوحة التحكم نفسها. يمنحك كل DIEM مُودَع **1 دولار يومياً** من رصيد Venice API طالما بقي مُودَعاً.

    يستلزم سحب إيداع DIEM فترة تهدئة (cooldown) مدتها **يوم واحد**. ويستلزم تحرير sVVV الذي يدعم DIEM المصكوك حرق ذلك DIEM؛ ثم يستلزم سحب إيداع sVVV المحرَّر فترة تهدئة مدتها **7 أيام**.
  </Step>

  <Step title="إنشاء مفتاح API أو استخدامه">
    سجّل الدخول إلى Venice باستخدام المحفظة نفسها (Sign-In-With-Ethereum) لكي يرتبط DIEM المُودَع بحسابك، ثم أنشئ مفتاحاً من [إعدادات API](https://venice.ai/settings/api) أو من [دليل مفتاح API](/guides/getting-started/generating-api-key).

    يمكن للوكلاء المستقلين (Autonomous agents) صكّ مفتاح من محفظة تحتوي على VVV مُودَع — راجع [إنشاء مفتاح API للوكيل المستقل](/guides/getting-started/generating-api-key-agent).
  </Step>
</Steps>

<Warning>
  لا ترسل VVV أو DIEM مطلقاً مباشرة إلى عنوان عقد. لا يمكن استرداد العملات المرسلة بهذه الطريقة. أودِع دائماً عبر [venice.ai/token](https://venice.ai/token) أو عبر تدفق معاملة إيداع معروف.
</Warning>

## استخدام DIEM مع الـ API

بمجرد إيداع DIEM وربطه بحسابك، استدعِ الـ API بمفتاح Bearer عادي. لا حاجة إلى ترويسات إضافية أو `venice_parameters` — إذ تخصم Venice من DIEM تلقائياً عندما تكون `consumptionCurrency` قيمتها `DIEM`.

```bash
curl https://api.venice.ai/api/v1/chat/completions \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "venice-uncensored",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

### التحقق من رصيد DIEM

```bash
curl https://api.venice.ai/api/v1/billing/balance \
  -H "Authorization: Bearer $VENICE_API_KEY"
```

مثال على شكل الاستجابة:

```json
{
  "canConsume": true,
  "consumptionCurrency": "DIEM",
  "balances": {
    "diem": 90.5,
    "usd": 25
  },
  "diemEpochAllocation": 100
}
```

| الحقل | المعنى |
| --- | --- |
| `balances.diem` | رصيد DIEM المتبقي للحقبة الحالية (`null` إذا لم تكن مُودِعاً لـ DIEM) |
| `diemEpochAllocation` | إجمالي DIEM المخصَّص لهذه الحقبة (المقدار المُودَع). قارنه بـ `balances.diem` لمعرفة الاستهلاك حتى الآن |
| `consumptionCurrency` | العملة التي سيتم الخصم منها في الطلب التالي (`DIEM` أو `USD` أو `BUNDLED_CREDITS` أو `VCU` القديم) |
| `canConsume` | ما إذا كان الحساب قادراً على دفع تكلفة طلب في الوقت الحالي |

اعرض قائمة النماذج أو تحليلات الاستخدام لرؤية تكلفة كل طلب بكل من USD و DIEM — وهي متطابقة بمعدّل 1 دولار لكل DIEM ضمن مخصّص اليوم. راجع [Billing Usage](/api-reference/endpoint/billing/usage) و [Usage Analytics](/api-reference/endpoint/billing/usage-analytics).

### تحديد سقف الإنفاق لكل مفتاح

عند إنشاء مفتاح، يمكنك ضبط حدّ استهلاك للحقبة مقوَّم بـ `diem` أو `usd` بحيث لا يستنزف أي تكامل منفرد كامل التخصيص اليومي. تعرض لوحة التحكم هذا الخيار باسم **Epoch Consumption Limits**؛ أما مفاتيح Web3 المصكوكة فتقبل `consumptionLimit` في [`POST /api_keys/generate_web3_key`](/api-reference/endpoint/api_keys/generate_web3_key/post).

## DIEM مقابل رصيد USD مقابل x402

| المسار | الأنسب لـ | كيف يتجدّد |
| --- | --- | --- |
| **DIEM المُودَع** | سعة يومية يمكن التنبؤ بها، والوكلاء، وتكلفة ذكاء اصطناعي ثابتة | تخصيص كامل في كل حقبة (00:00 UTC) طالما بقي DIEM مُودَعاً |
| **رصيد USD / العملات المشفّرة** | الاندفاعات التي تتجاوز مخصّص DIEM لديك، وإنفاق مسبق الدفع بسيط | ينقص الرصيد مع الاستخدام؛ ولا تنتهي صلاحية الرصيد أبداً |
| **x402 (USDC)** | الدفع لكل طلب من دون واجهة رأس (headless) وبدون مفتاح API | تُشحن المحفظة بـ USDC عبر `POST /x402/top-up`؛ ومع ذلك يُخصم أولاً من DIEM على الحساب المرتبط إن وُجد |

إذا كان الطلب يتجاوز رصيد DIEM المتبقي ولم يكن هناك رصيد USD أو أرصدة مُجمّعة لتغطيته، تُرجع الـ API الرمز `402` مع `INSUFFICIENT_BALANCE`. أودِع مزيداً من DIEM، أو انتظر الحقبة التالية، أو أضف رصيد USD من [الإعدادات ← API](https://venice.ai/settings/api).
