تدعم Venice قدرات إضافية ضمن نماذجها يمكن تشغيلها عبر مدخل `venice_parameters` في نقطة نهاية إكمالات المحادثة.

في بعض الحالات، قد تستخدم عميلًا لا يتيح لك تعديل نص الطلب. لتلك المنصات، يمكنك استخدام عرض لاحقة ميزات النموذج (Model Feature Suffix) من Venice لتمرير العلامات (flags) عبر مُعرّف النموذج.

## الصياغة

تتبع لاحقة ميزات النموذج هذا النمط:

```
<model_id>:<parameter>=<value>
```

لمعلمات متعددة، اربطها بعلامة `&`:

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## أمثلة

### لضبط بحث الويب على Auto
```
default:enable_web_search=auto
```

### لتفعيل بحث الويب وتعطيل مُوجِّه النظام
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### لتفعيل بحث الويب وإضافة الاستشهادات إلى الاستجابة
```
default:enable_web_search=on&enable_web_citations=true
```

### لتفعيل بحث الويب مع استخراج محتوى الصفحة الكامل
```
default:enable_web_search=on&enable_web_scraping=true
```

### لاستخدام شخصية
```
default:character_slug=alan-watts
```

### لإخفاء كتل التفكير في استجابة نموذج تفكير
```
qwen3-4b:strip_thinking_response=true
```

### لتعطيل التفكير في نماذج التفكير المدعومة
تدعم بعض نماذج التفكير (مثل Qwen 3) تعطيل عملية التفكير. يمكنك تنشيط ذلك باستخدام اللاحقة أدناه:
```
qwen3-4b:disable_thinking=true
```

### لإضافة نتائج بحث الويب إلى استجابة بثّ
سيؤدي هذا إلى تفعيل بحث الويب، وإضافة الاستشهادات إلى نص الاستجابة، وإدراج نتائج البحث في البث كرسالة الاستجابة النهائية.

يمكنك الاطلاع على مثال لذلك في [مجموعة Postman الخاصة بنا هنا](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041).

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
