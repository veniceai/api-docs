تدعم Venice قدرات إضافية ضمن نماذجها يمكن تشغيلها عبر مُدخل `venice_parameters` في endpoint إكمال المحادثات.

في بعض الحالات، قد تستخدم عميلًا لا يسمح لك بتعديل جسم الطلب. لتلك المنصات، يمكنك استخدام عرض Venice للاحقة ميزات النموذج لتمرير الأعلام عبر معرّف النموذج.

## الصيغة

تتبع لاحقة ميزات النموذج هذا النمط:

```
<model_id>:<parameter>=<value>
```

لمعاملات متعددة، اربطها بـ `&`:

```
<model_id>:<parameter1>=<value1>&<parameter2>=<value2>&<parameter3>=<value3>
```

## أمثلة

### تعيين البحث في الويب إلى Auto
```
default:enable_web_search=auto
```

### تفعيل البحث في الويب وتعطيل system prompt
```
default:enable_web_search=on&include_venice_system_prompt=false
```

### تفعيل البحث في الويب وإضافة الاستشهادات إلى الاستجابة
```
default:enable_web_search=on&enable_web_citations=true
```

### تفعيل البحث في الويب مع استخراج الصفحة كاملة
```
default:enable_web_search=on&enable_web_scraping=true
```

### استخدام شخصية
```
default:character_slug=alan-watts
```

### إخفاء كتل التفكير في استجابة نموذج استدلال
```
qwen3-4b:strip_thinking_response=true
```

### تعطيل التفكير على نماذج الاستدلال المدعومة
تدعم بعض نماذج الاستدلال (مثل Qwen 3) تعطيل عملية التفكير. يمكنك التفعيل باستخدام اللاحقة أدناه:
```
qwen3-4b:disable_thinking=true
```

### إضافة نتائج البحث في الويب إلى استجابة streaming
سيؤدي هذا إلى تفعيل البحث في الويب، وإضافة الاستشهادات إلى جسم الاستجابة، وتضمين نتائج البحث في الـ stream كرسالة الاستجابة النهائية.

يمكنك رؤية مثال على هذا في [مجموعة Postman هنا](https://www.postman.com/veniceai/workspace/venice-ai-workspace/request/38652128-ceef3395-451c-4391-bc7e-a40377e0357b?action=share&source=copy-link&creator=38652128&active-environment=ef110f4e-d3e1-43b5-8029-4d6877e62041).

```
qwen3-4b:enable_web_search=on&enable_web_citations=true&include_search_results_in_stream=true
```
