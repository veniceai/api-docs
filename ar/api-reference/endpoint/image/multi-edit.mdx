## مستويات الجودة

تقبل بعض نماذج التحرير المتعدد معاملًا اختياريًا `quality` (`low` أو `medium` أو `high`) يوازن بين دقة الصورة المرئية والتكلفة. مدعوم حاليًا من `gpt-image-2-edit`؛ تتجاهل النماذج الأخرى المعامل.

```json
{
  "modelId": "gpt-image-2-edit",
  "images": ["iVBORw0KGgo...", "iVBORw0KGgo..."],
  "prompt": "blend the two scenes into one cinematic frame",
  "resolution": "2K",
  "quality": "medium"
}
```

عند حذف `quality`، يستخدم النموذج مستواه الافتراضي (`high` لـ `gpt-image-2-edit`). توجد الأسعار لكل مستوى (1K/2K/4K × low/medium/high) ضمن `model_spec.pricing.quality` في [endpoint النماذج](/api-reference/endpoint/models/list) ومذكورة في [نظرة عامة على التسعير](/overview/pricing).

<Info>
التسعير المُدرِك للجودة هو الإعداد الافتراضي لجميع مستدعي API و SDK — لا يلزم header للموافقة. يتم احتساب الطلبات التي تحذف `quality` بالسعر الافتراضي للنموذج (`high` لـ `gpt-image-2-edit`).
</Info>
