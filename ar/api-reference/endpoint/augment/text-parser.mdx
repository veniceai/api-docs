<Warning>هذه واجهة برمجة تطبيقات تجريبية. قد يتغير تنسيق الطلب والاستجابة دون إشعار مسبق.</Warning>

قم بتحميل ملف مستند عبر multipart/form-data باستخدام حقل `file`. تشمل التنسيقات المدعومة ملفات **PDF** و**DOCX** و**XLSX** و**النصوص العادية** (حتى 25 ميغابايت).

اضبط `response_format` على `json` (الافتراضي) للحصول على مخرجات مُهيكلة تتضمن النص المستخرج وعدد الرموز (tokens)، أو على `text` للحصول على النص المستخرج الخام.

**الخصوصية:** يعمل تحليل النص بالكامل في الذاكرة على بنية Venice التحتية دون أي احتفاظ بالبيانات. تُعالَج مستنداتك ثم تُتلَف فورًا — لا يُخزَّن أي محتوى ولا يُسجَّل.

**التسعير:** 0.01 دولار أمريكي لكل طلب.

### مثال (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/text-parser \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -F "file=@document.pdf" \
  -F "response_format=json"
```
