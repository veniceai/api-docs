<Warning>هذه واجهة API تجريبية. قد يتغير تنسيق الطلب والاستجابة دون إشعار.</Warning>

أرسل عنوان URL يمكن الوصول إليه علنًا في حقل `url`. تُعيد الـ API محتوى الصفحة بصيغة **markdown**.

يحاول الـ scraper أولًا الاعتماد على دعم markdown الأصلي للموقع المستهدف (عبر [Cloudflare Markdown for Agents](https://blog.cloudflare.com/markdown-for-agents/))، ثم يعود إلى الاستخراج عبر متصفح headless. بعض المواقع التي تمنع الوصول الآلي (مثل X/Twitter و Reddit) يتم رفضها فورًا بخطأ `400`.

**التسعير:** $0.01 لكل طلب.

### مثال (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/scrape \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```
