<Warning>هذه واجهة برمجة تطبيقات تجريبية. قد يتغير تنسيق الطلب والاستجابة دون إشعار مسبق.</Warning>

أرسل عنوان URL يمكن الوصول إليه علنًا في حقل `url`. تُعيد واجهة برمجة التطبيقات محتوى الصفحة بتنسيق **markdown**.

يحاول المستخرج أولًا استخدام دعم markdown الأصلي للموقع المستهدف (عبر [Cloudflare Markdown for Agents](https://blog.cloudflare.com/markdown-for-agents/))، ثم يعود إلى الاستخراج عبر متصفح بلا واجهة. تُرفض بعض المواقع التي تحظر الوصول الآلي (مثل X/Twitter وReddit) فورًا بخطأ `400`.

**التسعير:** 0.01 دولار أمريكي لكل طلب.

### مثال (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/scrape \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```
