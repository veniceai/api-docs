<Warning>هذه واجهة برمجة تطبيقات تجريبية. قد يتغير تنسيق الطلب والاستجابة دون إشعار مسبق.</Warning>

أرسل استعلام البحث في حقل `query`. تُعيد واجهة برمجة التطبيقات نتائج مُهيكلة تتضمن العناوين وعناوين URL ومقتطفات المحتوى والتواريخ.

**مزودو البحث:**
- `brave` (الافتراضي) — Brave Search مع الاحتفاظ الصفري بالبيانات (ZDR). لا يقوم مزود البحث أبدًا بتخزين استعلامات البحث أو تسجيلها.
- `google` — Google Search مع استعلامات مجهولة الهوية. تُمرَّر عمليات البحث عبر بنية Venice التحتية بحيث لا تُربط هويتك بطلب البحث المُرسَل إلى Google. لا تخزن Venice استعلامات البحث ولا تسجلها.

**التسعير:** 0.01 دولار أمريكي لكل طلب.

### مثال (cURL)

```bash
curl -X POST https://api.venice.ai/api/v1/augment/search \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "latest news about AI"}'
```
