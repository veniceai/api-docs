استدعِ `/audio/quote` لتقدير التكلفة، ثم استعلم عن `/audio/retrieve` باستخدام `queue_id` المُعاد حتى ينتهي التوليد. إذا كنت ستحتفظ بالوسائط المُولَّدة بعد الاسترداد، فاستدعِ `/audio/complete` بمجرد تنزيلها.
