استدعِ `/audio/quote` لتقدير التكلفة، ثم استعلِم `/audio/retrieve` باستخدام الـ `queue_id` المُعاد حتى ينتهي التوليد. إذا كنت تحتفظ بالوسائط المُولَّدة بعد الاسترجاع، فاستدعِ `/audio/complete` بعد تنزيلها.
