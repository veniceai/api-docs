تُعيد قائمة مرتبة أبجديًا بأسماء (slugs) الشبكات المدعومة من وكيل Venice crypto RPC. استخدم ذلك لاكتشاف قيم `:network` الصالحة لـ `POST /crypto/rpc/{network}`.

نقطة النهاية هذه **عامة** — لا تتطلب مفتاح API ولا مصادقة محفظة.

## شكل الاستجابة

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

القائمة مرجعية — إذا لم يكن الـ slug موجودًا هنا، فإن نقطة نهاية الوكيل تُعيد `400 Unsupported RPC network`.
