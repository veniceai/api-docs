تُرجع قائمة معرّفات الشبكات المُرتَّبة أبجديًا والمدعومة من قِبَل وكيل Venice crypto RPC. استخدم هذا لاكتشاف قيم `:network` الصالحة لـ `POST /crypto/rpc/{network}`.

هذه الـ endpoint **عامة** — لا يلزم مفتاح API أو مصادقة محفظة.

## شكل الاستجابة

```json
{
  "networks": [
    "arbitrum-mainnet",
    "arbitrum-sepolia",
    "avalanche-fuji",
    "avalanche-mainnet",
    "base-mainnet",
    "base-sepolia",
    "bsc-mainnet",
    "bsc-testnet",
    "ethereum-holesky",
    "ethereum-mainnet",
    "ethereum-sepolia",
    "linea-mainnet",
    "linea-sepolia",
    "optimism-mainnet",
    "optimism-sepolia",
    "polygon-amoy",
    "polygon-mainnet",
    "starknet-mainnet",
    "starknet-sepolia",
    "zksync-mainnet",
    "zksync-sepolia"
  ]
}
```

القائمة موثوقة — إذا لم يكن المعرّف هنا، فإن endpoint الوكيل يُعيد `400 Unsupported RPC network`.
