## المتطلبات الأساسية

- مفتاح Venice API من [venice.ai/settings/api](https://venice.ai/settings/api)
- Codex CLI مثبَّت ويعمل على جهازك

---

## الإعداد

<Steps>
  <Step title="أنشئ مسار تهيئة المشروع">
    من جذر مشروعك:

    ```bash
    mkdir -p .codex
    ```
  </Step>

  <Step title="أنشئ .codex/config.toml">
    أنشئ الملف والصق التهيئة التالية:

    ```toml
    #:schema https://developers.openai.com/codex/config-schema.json

    model = "openai-gpt-54" # use any Venice model
    model_provider = "venice"
    model_reasoning_effort = "high"
    personality = "pragmatic"
    sandbox_mode = "workspace-write"

    [model_providers.venice]
    name = "Venice"
    base_url = "https://api.venice.ai/api/v1/"
    experimental_bearer_token = "YOUR VENICE API KEY"
    wire_api = "responses"
    ```
  </Step>

  <Step title="استبدل العنصرين النائبين">
    حدّث:

    - `model` بمعرّف نموذج Venice الذي تريد استخدامه
    - `experimental_bearer_token` بمفتاح Venice API الحقيقي الخاص بك

    يمكنك تصفّح معرّفات النماذج المتاحة في [كتالوج نماذج النص](/ar/models/text).
  </Step>

  <Step title="شغّل Codex CLI بشكل طبيعي">
    شغّل Codex CLI من نفس المشروع. سيقوم بتحميل `.codex/config.toml` (للمشاريع الموثوقة) وتوجيه الطلبات عبر Venice.
  </Step>
</Steps>

## مواقع تهيئة Codex الرسمية

- **افتراضيات المستخدم**: `~/.codex/config.toml`
- **تجاوزات المشروع**: `.codex/config.toml` (يُحمَّل فقط للمشاريع الموثوقة)

إذا أردت تطبيق إعدادات Venice في كل مكان، ضع التهيئة نفسها في `~/.codex/config.toml`.

---

## أولوية التهيئة (من الأعلى أولاً)

1. أعلام CLI وتجاوزات `--config`
2. قيم الملف الشخصي (`--profile <name>`)
3. طبقات تهيئة المشروع (`.codex/config.toml`، أقرب مجلد يفوز)
4. تهيئة المستخدم (`~/.codex/config.toml`)
5. تهيئة النظام (`/etc/codex/config.toml`, Unix)
6. الافتراضيات المدمجة

---

## ملاحظات

- حافظ على خصوصية مفتاح API الخاص بك ولا تلتزم بمفاتيح حقيقية في git أبداً.
- يتجاهل Codex تهيئة `.codex/` في المشروع عندما يكون المشروع مُعلَّماً كغير موثوق.
- إذا غيّرت النماذج، حدّث فقط حقل `model`.
- إعداد `wire_api = "responses"` مطلوب لإعداد هذا المزوّد.
