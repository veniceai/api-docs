[OpenClaw](https://openclaw.ai) هي بوابة ذكاء اصطناعي مفتوحة المصدر ومُستضافة ذاتيًا تربط منصات المراسلة (WhatsApp و Telegram و Discord و iMessage و Slack) بنماذج الذكاء الاصطناعي. Venice AI متوفرة كموفّر مدمج، مما يمنحك الوصول إلى نماذج خاصة وغير مقيَّدة من أي قناة متصلة.

## الإعداد

### 1. ثبّت OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. شغّل معالج التهيئة

```bash
openclaw onboard
```

سيرشدك المعالج خلال الإعداد. عندما يُطلب منك، اختر **Venice AI** كموفّر من القائمة، ثم الصق مفتاح API الخاص بك. احصل على واحد من [venice.ai/settings/api](https://venice.ai/settings/api) إذا لم يكن لديك واحد بعد.

### 3. اختر نموذجًا

أثناء التهيئة، يعرض OpenClaw جميع نماذج Venice المتاحة. بعض التوصيات:

| حالة الاستخدام | النموذج | الخصوصية |
|---|---|---|
| عام | `venice/zai-org-glm-5` | خاص |
| استدلال | `venice/kimi-k2-5` | خاص |
| برمجة | `venice/claude-opus-4-6` | مجهول الهوية |
| رؤية | `venice/qwen3-vl-235b-a22b` | خاص |
| غير مقيَّد | `venice/venice-uncensored` | خاص |

غيّر نموذجك الافتراضي في أي وقت:

```bash
openclaw models set venice/zai-org-glm-5
```

اسرد جميع النماذج المتاحة:

```bash
openclaw models list | grep venice
```

### 4. ابدأ الدردشة

افتح واجهة الـ terminal:

```bash
openclaw tui
```

أو لوحة الويب:

```bash
openclaw dashboard
```

أو وصّل قناة مراسلة (WhatsApp و Telegram و Discord، إلخ):

```bash
openclaw channels login
openclaw gateway
```

## أوضاع الخصوصية

تتبع نماذج Venice في OpenClaw نفس [مستويات الخصوصية](/overview/privacy) لـ Venice API:

- النماذج **Private** (GLM و Qwen و DeepSeek و Llama و Venice Uncensored) تعمل على أسطول GPU في Venice. لا يتم تخزين المطالبات أو تسجيلها أبدًا.
- النماذج **Anonymized** (Claude و GPT و Gemini و Grok) يتم تمريرها عبر Venice مع إزالة جميع المعلومات المُعرِّفة. يرى موفّر الطرف الثالث Venice كعميل، وليس أنت.

## توليد الصور والفيديو

ثبّت مهارة Venice AI Media لتوليد الصور والفيديو:

```bash
openclaw skills install nhannah/venice-ai-media
```
