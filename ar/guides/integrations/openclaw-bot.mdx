[OpenClaw](https://openclaw.ai) هو بوابة ذكاء اصطناعي مفتوحة المصدر وذاتية الاستضافة تربط منصات المراسلة (WhatsApp وTelegram وDiscord وiMessage وSlack) بنماذج الذكاء الاصطناعي. Venice AI متاح كمزوّد مدمج، مما يمنحك الوصول إلى نماذج خاصة وبدون رقابة من أي قناة مرتبطة.

## الإعداد

### 1. ثبّت OpenClaw

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://openclaw.ai/install.sh | bash
    ```
  </Tab>

  <Tab title="Windows">
    ```powershell
    iwr -useb https://openclaw.ai/install.ps1 | iex
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </Tab>
</Tabs>

### 2. شغّل معالج الإعداد التعريفي

```bash
openclaw onboard
```

سيأخذك المعالج خطوة بخطوة عبر الإعداد. عند المطالبة، اختر **Venice AI** كمزوّد لك من القائمة، ثم الصق مفتاح API الخاص بك. احصل على واحد من [venice.ai/settings/api](https://venice.ai/settings/api) إذا لم يكن لديك بعد.

### 3. اختر نموذجاً

أثناء الإعداد التعريفي، يعرض OpenClaw جميع نماذج Venice المتاحة. بعض التوصيات:

| حالة الاستخدام | النموذج | الخصوصية |
|---|---|---|
| عام | `venice/zai-org-glm-5` | خاص |
| الاستدلال | `venice/kimi-k2-5` | خاص |
| البرمجة | `venice/claude-opus-4-6` | مجهّل |
| الرؤية | `venice/qwen3-vl-235b-a22b` | خاص |
| بدون رقابة | `venice/venice-uncensored` | خاص |

غيّر نموذجك الافتراضي في أي وقت:

```bash
openclaw models set venice/zai-org-glm-5
```

اعرض جميع النماذج المتاحة:

```bash
openclaw models list | grep venice
```

### 4. ابدأ الدردشة

افتح واجهة الطرفية:

```bash
openclaw tui
```

أو لوحة تحكم الويب:

```bash
openclaw dashboard
```

أو اربط قناة مراسلة (WhatsApp وTelegram وDiscord وغيرها):

```bash
openclaw channels login
openclaw gateway
```

## أوضاع الخصوصية

تتبع نماذج Venice في OpenClaw نفس [مستويات الخصوصية](/ar/overview/privacy) لـ Venice API:

- نماذج **Private** (GLM وQwen وDeepSeek وLlama وVenice Uncensored) تعمل على أسطول GPU الخاص بـ Venice. الطلبات لا تُخزَّن ولا تُسجَّل أبداً.
- نماذج **Anonymized** (Claude وGPT وGemini وGrok) تُمرَّر عبر Venice مع تجريد جميع المعلومات المعرّفة. يرى المزوّد الخارجي Venice كعميل، لا أنت.

## توليد الصور والفيديو

ثبّت مهارة Venice AI Media لتوليد الصور والفيديو:

```bash
openclaw skills install nhannah/venice-ai-media
```
