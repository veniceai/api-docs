[Hermes Agent](https://hermes-agent.nousresearch.com) هو وكيل ذكاء اصطناعي مفتوح المصدر، ذاتي الاستضافة، بُني بواسطة [Nous Research](https://nousresearch.com). يتميز بذاكرة دائمة، وإنشاء مهارات مستقل، وحلقة تعلم مدمجة تصبح أكثر قدرة كلما طال تشغيلها. وجّهه إلى Venice API ليحصل وكيلك على الوصول إلى أكثر من 230 نموذجًا وأدوات عبر النصوص والصور والفيديو والصوت والـ embeddings والمزيد.

## لماذا Venice + Hermes Agent؟

يمنح Venice API وكيل Hermes Agent الخاص بك الوصول إلى منصة Venice الكاملة عبر نقطة نهاية واحدة متوافقة مع OpenAI.

| القدرة | ما تحصل عليه |
|---|---|
| **النصوص والمحادثة** | نماذج خاصة ومجهّلة الهوية (GLM، Qwen، Claude، GPT، والمزيد) |
| **الصور** | التوليد، والتحرير، والترقية، وإزالة الخلفية |
| **الفيديو** | التوليد والنسخ النصي |
| **الصوت** | تركيب الكلام (TTS)، وتوليد الموسيقى، وتحويل الكلام إلى نص |
| **Embeddings** | تضمينات متجهية لـ RAG والبحث الدلالي |
| **الأدوات** | كشط الويب، والبحث على الويب، وتحليل النصوص، و crypto RPC |

## الإعداد

### 1. تثبيت Hermes Agent

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>

  <Tab title="WSL2 (Windows)">
    ثبّت [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install) أولًا، ثم شغّل نفس الأمر:
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>
</Tabs>

بعد التثبيت، أعد تحميل shell الخاص بك:

```bash
source ~/.zshrc   # أو: source ~/.bashrc
```

### 2. تكوين Venice كمزودك

شغّل معالج إعداد النموذج:

```bash
hermes model
```

اختر **Custom endpoint (self-hosted / VLLM / etc.)** وأدخل ما يلي عند الطلب:

| الحقل | القيمة |
|---|---|
| **API base URL** | `https://api.venice.ai/api/v1` |
| **API key** | مفتاح Venice API الخاص بك |
| **Model name** | معرّف نموذج Venice (انظر أدناه) |

احصل على مفتاح API من [venice.ai/settings/api](https://venice.ai/settings/api) إذا لم يكن لديك واحد بعد.

<Tip>
يمكنك أيضًا تكوين Venice مباشرة في `~/.hermes/config.yaml`:

```yaml
model:
  default: zai-org-glm-5
  provider: custom
  base_url: https://api.venice.ai/api/v1
  api_key: ${VENICE_API_KEY}
```

وعيّن المفتاح في `~/.hermes/.env`:

```bash
VENICE_API_KEY=your-key-here
```
</Tip>

### 3. اختر نموذجًا

عندما يطلب المعالج نموذجًا، اختر واحدًا بناءً على حالة استخدامك:

| حالة الاستخدام | النموذج | الخصوصية |
|---|---|---|
| عام | `zai-org-glm-5` | خاص |
| تفكير منطقي | `kimi-k2-6` | خاص |
| برمجة | `claude-opus-4-7` | مجهّل الهوية |
| رؤية | `z-ai-glm-5v-turbo` | مجهّل الهوية |
| غير خاضع للرقابة | `venice-uncensored-1-2` | خاص |

غيّر نموذجك في أي وقت دون الحاجة إلى إعادة تشغيل:

```bash
hermes model              # المعالج الكامل
```

أو بدّل في منتصف الجلسة:

```text
/model custom:claude-opus-4-7
```

### 4. ابدأ الدردشة

افتح CLI الكلاسيكي أو TUI الحديث:

```bash
hermes            # CLI كلاسيكي
hermes --tui      # TUI حديث (موصى به)
```

جرب مطالبة للتحقق من أن كل شيء يعمل:

```text
Summarize this repo in 5 bullets and tell me what the main entrypoint is.
```

## وصل منصات المراسلة

بمجرد عمل CLI، وصّل تطبيقات المراسلة الخاصة بك من خلال البوابة:

```bash
hermes gateway setup
```

يرشدك هذا خلال توصيل Telegram و Discord و Slack و WhatsApp و Signal ومنصات أخرى. يصبح وكيلك قابلًا للوصول من أي قناة متصلة، كلها مدعومة بـ Venice.

```bash
hermes gateway      # ابدأ بوابة المراسلة
```

## أوضاع الخصوصية

تتبع نماذج Venice في Hermes Agent نفس [مستويات الخصوصية](/overview/privacy) مثل Venice API:

- نماذج **Private** (GLM، Qwen، DeepSeek، Llama، Venice Uncensored) تعمل على أسطول GPU الخاص بـ Venice. لا تُخزَّن المطالبات أو تُسجَّل أبدًا.
- نماذج **Anonymized** (Claude، GPT، Gemini، Grok) يتم توجيهها عبر Venice مع إزالة جميع المعلومات المعرّفة. يرى المزود الخارجي Venice كعميل، وليس أنت.

## مهارات Venice API

يحتوي Hermes Agent على نظام مهارات مدمج متوافق مع تنسيق [Agent Skills](https://github.com/veniceai/skills). تنشر Venice مهارات رسمية تعلّم وكيلك كيفية استخدام كل نقطة نهاية Venice (الدردشة، توليد الصور، الفيديو، الصوت، embeddings، أدوات augment، والمزيد).

ثبّت مهارات Venice مباشرة من GitHub:

```bash
hermes skills install veniceai/skills
```

أو ابحث عن مهارات فردية:

```bash
hermes skills search venice
```

سيكتشف Hermes كل مهارة عبر frontmatter في `SKILL.md` ويحمّلها عند الطلب.

## الأوامر الرئيسية

| الأمر | الوصف |
|---|---|
| `hermes` | ابدأ الدردشة |
| `hermes model` | غيّر المزود أو النموذج |
| `hermes tools` | كوّن الأدوات الممكّنة |
| `hermes gateway` | ابدأ بوابة المراسلة |
| `hermes --continue` | استأنف جلستك الأخيرة |
| `hermes doctor` | شخّص المشكلات |
| `hermes update` | حدّث إلى أحدث إصدار |
