[Hermes Agent](https://hermes-agent.nousresearch.com) هو وكيل ذكاء اصطناعي مفتوح المصدر وذاتي الاستضافة بناه [Nous Research](https://nousresearch.com). يتميز بذاكرة دائمة، وإنشاء مهارات تلقائي، وحلقة تعلّم مدمجة تزداد قدرتها كلما طالت مدة تشغيله. وجّهه إلى Venice API ويحصل وكيلك على الوصول إلى أكثر من 230 نموذجاً وأداة عبر النص والصور والفيديو والصوت والتضمينات والمزيد.

## لماذا Venice + Hermes Agent؟

تمنح Venice API وكيل Hermes الخاص بك الوصول إلى منصة Venice الكاملة عبر نقطة نهاية واحدة متوافقة مع OpenAI.

| القدرة | ما تحصل عليه |
|---|---|
| **النص والدردشة** | نماذج خاصة ومجهّلة (GLM وQwen وClaude وGPT والمزيد) |
| **الصور** | التوليد والتحرير والتحسين وإزالة الخلفية |
| **الفيديو** | التوليد والنسخ النصي |
| **الصوت** | تركيب الكلام (TTS) وتوليد الموسيقى وتحويل الكلام إلى نص |
| **التضمينات** | تضمينات متجهية لـ RAG والبحث الدلالي |
| **الأدوات** | كشط الويب، والبحث على الويب، وتحليل النصوص، وCrypto RPC |

## الإعداد

### 1. ثبّت Hermes Agent

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>

  <Tab title="WSL2 (Windows)">
    ثبّت [WSL2](https://learn.microsoft.com/en-us/windows/wsl/install) أولاً، ثم شغّل نفس الأمر:
    ```bash
    curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
    ```
  </Tab>
</Tabs>

بعد التثبيت، أعد تحميل الـ shell الخاص بك:

```bash
source ~/.zshrc   # or: source ~/.bashrc
```

### 2. هيّئ Venice كمزوّد لك

شغّل معالج إعداد النموذج:

```bash
hermes model
```

اختر **Custom endpoint (self-hosted / VLLM / etc.)** وأدخل ما يلي عند طلبها:

| الحقل | القيمة |
|---|---|
| **API base URL** | `https://api.venice.ai/api/v1` |
| **API key** | مفتاح Venice API الخاص بك |
| **Model name** | معرّف نموذج Venice (راجع أدناه) |

احصل على مفتاح API من [venice.ai/settings/api](https://venice.ai/settings/api) إذا لم يكن لديك واحد بعد.

<Tip>
يمكنك أيضاً تهيئة Venice مباشرة في `~/.hermes/config.yaml`:

```yaml
model:
  default: zai-org-glm-5
  provider: custom
  base_url: https://api.venice.ai/api/v1
  api_key: ${VENICE_API_KEY}
```

واضبط المفتاح في `~/.hermes/.env`:

```bash
VENICE_API_KEY=your-key-here
```
</Tip>

### 3. اختر نموذجاً

عندما يطلب المعالج نموذجاً، اختر واحداً بناءً على حالة الاستخدام:

| حالة الاستخدام | النموذج | الخصوصية |
|---|---|---|
| عام | `zai-org-glm-5` | خاص |
| الاستدلال | `kimi-k2-6` | خاص |
| البرمجة | `claude-opus-4-7` | مجهّل |
| الرؤية | `z-ai-glm-5v-turbo` | مجهّل |
| بدون رقابة | `venice-uncensored-1-2` | خاص |

غيّر نموذجك في أي وقت دون الحاجة إلى إعادة التشغيل:

```bash
hermes model              # full wizard
```

أو بدّل في منتصف الجلسة:

```text
/model custom:claude-opus-4-7
```

### 4. ابدأ الدردشة

افتح CLI الكلاسيكي أو TUI الحديث:

```bash
hermes            # classic CLI
hermes --tui      # modern TUI (recommended)
```

جرّب طلباً للتحقق من أن كل شيء يعمل:

```text
Summarize this repo in 5 bullets and tell me what the main entrypoint is.
```

## ربط منصات المراسلة

بمجرد أن يعمل CLI، اربط تطبيقات المراسلة عبر البوابة:

```bash
hermes gateway setup
```

يأخذك هذا خطوة بخطوة عبر ربط Telegram وDiscord وSlack وWhatsApp وSignal ومنصات أخرى. يصبح وكيلك قابلاً للوصول من أي قناة مرتبطة، مدعومة بـ Venice.

```bash
hermes gateway      # start the messaging gateway
```

## أوضاع الخصوصية

تتبع نماذج Venice في Hermes Agent نفس [مستويات الخصوصية](/ar/overview/privacy) لـ Venice API:

- نماذج **Private** (GLM وQwen وDeepSeek وLlama وVenice Uncensored) تعمل على أسطول GPU الخاص بـ Venice. الطلبات لا تُخزَّن ولا تُسجَّل أبداً.
- نماذج **Anonymized** (Claude وGPT وGemini وGrok) تُمرَّر عبر Venice مع تجريد جميع المعلومات المعرّفة. يرى المزوّد الخارجي Venice كعميل، لا أنت.

## مهارات Venice API

يمتلك Hermes Agent نظام مهارات مدمج متوافق مع تنسيق [Agent Skills](https://github.com/veniceai/skills). تنشر Venice مهارات رسمية تعلّم وكيلك كيفية استخدام كل نقطة نهاية Venice (الدردشة وتوليد الصور والفيديو والصوت والتضمينات وأدوات augment والمزيد).

ثبّت مهارات Venice مباشرة من GitHub:

```bash
hermes skills install veniceai/skills
```

أو ابحث عن مهارات فردية:

```bash
hermes skills search venice
```

سيكتشف Hermes كل مهارة عبر frontmatter لـ `SKILL.md` ويحمّلها عند الطلب.

| `hermes tools` | هيّئ الأدوات الممكّنة |
| `hermes gateway` | شغّل بوابة المراسلة |
| `hermes --continue` | استأنف جلستك الأخيرة |
| `hermes doctor` | شخّص المشاكل |
| `hermes update` | حدّث إلى أحدث إصدار |
