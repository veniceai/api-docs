## الإعداد

<Steps>
  <Step title="احصل على مفتاح API الخاص بك">
    ولِّد مفتاحًا من [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="قم بتكوين Cursor">
    افتح **Cursor Settings** (رمز الترس)، ثم انتقل إلى **Models** وانقر **Add Model**.

    تحت **OpenAI Compatible**، أدخل:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: مفتاح Venice API الخاص بك
  </Step>

  <Step title="أضف النماذج">
    في حقل اسم النموذج، اكتب معرّف نموذج Venice واضغط Enter. أضف كل نموذج تريد استخدامه. على سبيل المثال:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (راجع [ملاحظة البادئة أدناه](#the-venice--model-prefix))

    راجع [كتالوج النماذج](/models/text) لجميع معرّفات النماذج المتاحة.
  </Step>

  <Step title="اختر وتحقق">
    افتح محادثة جديدة واستخدم **محدد النموذج في أسفل المحادثة** لاختيار أحد نماذج Venice التي أضفتها للتو. أرسل رسالة اختبار. إذا حصلت على رد، فأنت جاهز.
  </Step>
</Steps>

## بادئة النموذج `venice-`

<Warning>
**تتطلب نماذج Claude بادئة `venice-` عند استخدامها في Cursor.**

يقوم Cursor بإعادة كتابة الطلبات لنماذج `claude-*` إلى تنسيق Anthropic الأصلي، وهو غير متوافق مع Venice. تجنب إضافة البادئة `venice-` إلى معرّف النموذج هذه إعادة الكتابة. تقوم Venice بإزالة البادئة تلقائيًا.
</Warning>

| النموذج | المعرّف القياسي | معرّف Cursor |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
النماذج غير الخاصة بـ Claude (مثل `minimax-m25` و `kimi-k2-5` و `zai-org-glm-5`) **غير متأثرة** وتعمل دون البادئة. البادئة `venice-` آمنة للاستخدام على أي نموذج لأن Venice دائمًا تزيلها، لكنها مطلوبة فقط لنماذج Claude في Cursor.
</Info>
