## الإعداد

<Steps>
  <Step title="احصل على مفتاح API الخاص بك">
    أنشئ مفتاحاً من [venice.ai/settings/api](https://venice.ai/settings/api).
  </Step>

  <Step title="هيّئ Cursor">
    افتح **Cursor Settings** (أيقونة الترس)، انتقل إلى **Models** وانقر على **Add Model**.

    تحت **OpenAI Compatible**، أدخل:
    - **Override OpenAI Base URL**: `https://api.venice.ai/api/v1`
    - **OpenAI API Key**: مفتاح Venice API الخاص بك
  </Step>

  <Step title="أضف النماذج">
    في حقل اسم النموذج، اكتب معرّف نموذج Venice واضغط Enter. أضف كل نموذج تريد استخدامه. على سبيل المثال:
    - `minimax-m25`
    - `kimi-k2-5`
    - `venice-claude-opus-4-6` (راجع [ملاحظة البادئة أدناه](#the-venice--model-prefix))

    راجع [كتالوج النماذج](/ar/models/text) للاطلاع على جميع معرّفات النماذج المتاحة.
  </Step>

  <Step title="اختر وتحقّق">
    افتح محادثة جديدة واستخدم **محدّد النموذج في أسفل الدردشة** لاختيار أحد نماذج Venice التي أضفتها للتو. أرسل رسالة اختبارية. إذا تلقّيت استجابة، فأنت جاهز.
  </Step>
</Steps>

## بادئة النموذج `venice-`

<Warning>
**نماذج Claude تتطلب بادئة `venice-` عند استخدامها في Cursor.**

تعيد Cursor كتابة الطلبات لنماذج `claude-*` إلى التنسيق الأصلي لـ Anthropic، وهو غير متوافق مع Venice. إضافة بادئة `venice-` لمعرّف النموذج يمنع إعادة الكتابة. وتزيل Venice البادئة تلقائياً.
</Warning>

| النموذج | المعرّف القياسي | معرّف Cursor |
|-------|------------|-----------|
| Claude Opus 4.6 | `claude-opus-4-6` | `venice-claude-opus-4-6` |
| Claude Opus 4.6 Fast | `claude-opus-4-6-fast` | `venice-claude-opus-4-6-fast` |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | `venice-claude-sonnet-4-6` |
| Claude Opus 4.5 | `claude-opus-4-5` | `venice-claude-opus-4-5` |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | `venice-claude-sonnet-4-5` |

<Info>
النماذج غير-Claude (مثل `minimax-m25` و`kimi-k2-5` و`zai-org-glm-5`) **غير متأثرة** وتعمل بدون البادئة. استخدام بادئة `venice-` آمن على أي نموذج لأن Venice تزيلها دائماً، لكنها مطلوبة فقط لنماذج Claude في Cursor.
</Info>
