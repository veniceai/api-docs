## نظرة عامة

توفّر Venice مجموعة Postman شاملة تتيح للمطورين استكشاف واختبار القدرات الكاملة لواجهتنا البرمجية. تتضمن هذه المجموعة طلبات مُهيّأة مسبقاً وأمثلة ومتغيرات بيئة لمساعدتك على البدء بسرعة مع خدمات Venice للذكاء الاصطناعي.

## الوصول إلى المجموعة

تتوفر مجموعة Postman الرسمية في Venice AI Workspace:

* [Venice AI Postman Workspace](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman Examples](https://postman.venice.ai/)

## ميزات المجموعة

- **طلبات جاهزة للاستخدام**: استدعاءات API مُهيّأة مسبقاً لجميع نقاط نهاية Venice
- **قوالب بيئة**: متغيرات بيئة مُنظَّمة بشكل صحيح
- **أمثلة طلبات**: أمثلة استخدام واقعية لكل نقطة نهاية
- **عينات استجابات**: استجابات نموذجية لمساعدتك على فهم مخرجات الـ API
- **توثيق**: توثيق داخلي لكل طلب

## البدء

<Steps>
   <Step title="انسخ المجموعة (Fork)">
      - انتقل إلى Venice AI Workspace
      - انقر على "Fork" لإنشاء نسختك الخاصة من المجموعة
      - اختر وجهة workspace الخاصة بك
   </Step>

   <Step title="إعداد البيئة">
      - أنشئ بيئة جديدة في Postman
      - أضف مفتاح Venice API الخاص بك
      - اضبط عنوان URL الأساسي: `https://api.venice.ai/api/v1`
   </Step>

   <Step title="إرسال أول طلب">
      - حدّد أي طلب من المجموعة
      - تأكد من تحديد بيئتك
      - انقر على "Send" لاختبار الـ API
   </Step>
</Steps>

## نقاط النهاية المتاحة

تتضمن المجموعة أمثلة لجميع نقاط نهاية Venice API:

- توليد النص
- توليد الصور
- معلومات النموذج
- تحسين دقة الصور (Upscaling)
- تهيئة System Prompt

## أفضل الممارسات

- حافظ على أمان مفتاح API الخاص بك ولا تشاركه أبداً
- استخدم متغيرات البيئة للمعلومات الحساسة
- اختبر الاستجابات في وحدة تحكم Postman قبل التنفيذ
- راجع الاستجابات النموذجية لمعرفة هياكل البيانات المتوقعة

<Note>*ملاحظة: يتم تحديث مجموعة Postman بانتظام لتعكس أحدث تغييرات وميزات الـ API.*</Note>
