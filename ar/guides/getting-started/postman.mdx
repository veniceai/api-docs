## نظرة عامة

توفر Venice مجموعة Postman شاملة تتيح للمطورين استكشاف واختبار القدرات الكاملة لـ API لدينا. تتضمن هذه المجموعة طلبات مُكوَّنة مسبقًا، وأمثلة، ومتغيرات بيئة لمساعدتك على البدء بسرعة مع خدمات Venice للذكاء الاصطناعي.

## الوصول إلى المجموعة

مجموعة Postman الرسمية الخاصة بنا متاحة في Venice AI Workspace:

* [Venice AI Postman Workspace](https://www.postman.com/veniceai/workspace/venice-ai-workspace)
* [Venice AI Postman Examples](https://postman.venice.ai/)

## ميزات المجموعة

- **طلبات جاهزة للاستخدام**: استدعاءات API مُكوَّنة مسبقًا لجميع endpoints Venice
- **قوالب البيئة**: متغيرات بيئة مُهيكَلة بشكل صحيح
- **أمثلة الطلبات**: أمثلة استخدام من العالم الحقيقي لكل endpoint
- **عينات الاستجابة**: استجابات نموذجية لمساعدتك في فهم مخرجات API
- **التوثيق**: توثيق مدمج لكل طلب

## البدء

<Steps>
   <Step title="عمل Fork للمجموعة">
      - انتقل إلى Venice AI Workspace
      - انقر "Fork" لإنشاء نسختك الخاصة من المجموعة
      - اختر مساحة العمل الوجهة
   </Step>

   <Step title="إعداد البيئة">
      - أنشئ بيئة جديدة في Postman
      - أضف مفتاح Venice API الخاص بك
      - قم بتكوين base URL: `https://api.venice.ai/api/v1`
   </Step>

   <Step title="قم بأول طلب">
      - اختر أي طلب من المجموعة
      - تأكد من تحديد بيئتك
      - انقر "Send" لاختبار API
   </Step>
</Steps>

## Endpoints المتاحة

تتضمن المجموعة أمثلة لجميع endpoints Venice API:

- توليد النصوص
- توليد الصور
- معلومات النماذج
- تكبير الصور
- تكوين System Prompt

## أفضل الممارسات

- احتفظ بمفتاح API الخاص بك آمنًا ولا تشاركه أبدًا
- استخدم متغيرات البيئة للمعلومات الحساسة
- اختبر الاستجابات في console Postman قبل التنفيذ
- راجع استجابات الأمثلة لمعرفة هياكل البيانات المتوقعة

<Note>*ملاحظة: يتم تحديث مجموعة Postman بانتظام لتعكس أحدث التغييرات والميزات في API.*</Note>
