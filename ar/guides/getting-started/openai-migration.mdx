Venice AI هي **بديل مباشر** لـ OpenAI. نفس الـ SDK، نفس الكود — فقط غيّر سطرين. احصل على استدلال يركّز على الخصوصية، ونماذج بدون قيود، وأسعار تنافسية.

## الانتقال بسطرين

### Python

```python
# Before (OpenAI)
from openai import OpenAI
client = OpenAI()

# After (Venice) — change api_key and base_url
from openai import OpenAI
client = OpenAI(
    api_key="your-venice-api-key",          # ← Change 1
    base_url="https://api.venice.ai/api/v1"  # ← Change 2
)
```

### Node.js

```javascript
// Before (OpenAI)
import OpenAI from 'openai';
const client = new OpenAI();

// After (Venice)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: 'your-venice-api-key',
  baseURL: 'https://api.venice.ai/api/v1',
});
```

### cURL

```bash
# Before
curl https://api.openai.com/v1/chat/completions ...

# After — just change the URL and key
curl https://api.venice.ai/api/v1/chat/completions ...
```

### متغيرات البيئة

```bash
# Before
OPENAI_API_KEY=sk-...
OPENAI_BASE_URL=https://api.openai.com/v1

# After
OPENAI_API_KEY=your-venice-api-key
OPENAI_BASE_URL=https://api.venice.ai/api/v1
```

<Tip>
تقرأ العديد من المكتبات والأدوات `OPENAI_API_KEY` و `OPENAI_BASE_URL` تلقائيًا. قد يكون تحديث متغيرات البيئة هذه كل ما تحتاجه.
</Tip>

## ميزات Venice الحصرية

تقدم Venice قدرات لا تقدمها OpenAI:

### 1. بحث ويب مدمج
```python
response = client.chat.completions.create(
    model="venice-uncensored",
    messages=[{"role": "user", "content": "Latest AI news today"}],
    extra_body={
        "venice_parameters": {
            "enable_web_search": "auto"
        }
    }
)
```

### 2. استخراج محتوى الويب
```python
response = client.chat.completions.create(
    model="venice-uncensored",
    messages=[{"role": "user", "content": "Summarize https://example.com/article"}],
    extra_body={
        "venice_parameters": {
            "enable_web_scraping": True
        }
    }
)
```

### 3. الشخصيات (Characters)
```python
response = client.chat.completions.create(
    model="venice-uncensored",
    messages=[{"role": "user", "content": "Tell me about yourself"}],
    extra_body={
        "venice_parameters": {
            "character_slug": "venice-ai"
        }
    }
)
```

### 4. النماذج بدون قيود
نماذج Venice الخاصة لا تحتوي على تصفية محتوى، مما يجعلها مناسبة لـ:
- الكتابة الإبداعية بدون حواجز
- أبحاث الأمان واختبار الفريق الأحمر
- التحليل الصريح بدون أنماط الرفض
- المعلومات الطبية/القانونية بدون إخلاءات مسؤولية

### 5. توليد الفيديو
```python
# Queue a video generation job
import requests

response = requests.post(
    "https://api.venice.ai/api/v1/video/queue",
    headers={"Authorization": f"Bearer {api_key}", "Content-Type": "application/json"},
    json={
        "model": "wan-2.6-text-to-video",
        "prompt": "A serene lake at sunset with gentle waves",
        "resolution": "720p",
        "duration": 5,
    }
)
job_id = response.json()["id"]
```

## لماذا تنتقل؟

### الخصوصية
- **عدم الاحتفاظ بالبيانات** على النماذج الخاصة — لا يتم تخزين مطالباتك أبدًا
- **عدم التدريب على بياناتك** — أبدًا
- تحتفظ OpenAI بالبيانات لمدة 30 يومًا وقد تستخدمها لأبحاث السلامة

### التكلفة
- النماذج الخاصة **غالبًا أرخص** من مكافئات OpenAI
- `qwen3-4b` بسعر $0.05/1M input أرخص 10 مرات من gpt-4o-mini
- `venice-uncensored` بسعر $0.20/1M input مقابل gpt-4o بسعر $2.50/1M

### الحرية
- **لا تصفية محتوى** على النماذج بدون قيود
- لا توقف للحسابات لحالات الاستخدام المثيرة للجدل
- Web3-native مع خيارات الدفع بالعملات المشفّرة
- DIEM staking للاعتمادات اليومية

### تنوع النماذج
- الوصول إلى نماذج من موفّرين متعددين (Qwen و Llama و Mistral و Gemma و Claude و GPT و Grok وغيرها)
- التبديل بين النماذج الخاصة والمجهولة لكل طلب
- نماذج جديدة تُضاف بانتظام

## انتقال أطر العمل

تعمل معظم أطر الذكاء الاصطناعي مع Venice بتغيير base URL:

| الإطار | التغيير المطلوب |
|-----------|----------------|
| LangChain | `base_url` في `ChatOpenAI` |
| Vercel AI SDK | `baseURL` في `createOpenAI` |
| CrewAI | متغير بيئة `OPENAI_API_BASE` |
| LlamaIndex | `api_base` في `OpenAI` |
| AutoGen | `base_url` في التكوين |
| Haystack | `api_base_url` في `OpenAIGenerator` |
| Claude Code | علم `--api-base` أو متغير بيئة |
| Cursor | endpoint API مخصصة في الإعدادات |
| Continue.dev | `apiBase` في config.json |
