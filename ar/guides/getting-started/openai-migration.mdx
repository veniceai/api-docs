Venice AI **بديل جاهز للاستخدام** لـ OpenAI. نفس SDK، نفس الشيفرة — فقط غيّر سطرين. احصل على استدلال يعطي الأولوية للخصوصية، ونماذج بدون رقابة، وتسعير تنافسي.

## الهجرة بسطرين

### Python

```python
# Before (OpenAI)
from openai import OpenAI
client = OpenAI()

# After (Venice) — change api_key and base_url
from openai import OpenAI
client = OpenAI(
    api_key="your-venice-api-key",          # ← Change 1
    base_url="https://api.venice.ai/api/v1"  # ← Change 2
)
```

### Node.js

```javascript
// Before (OpenAI)
import OpenAI from 'openai';
const client = new OpenAI();

// After (Venice)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: 'your-venice-api-key',
  baseURL: 'https://api.venice.ai/api/v1',
});
```

### cURL

```bash
# Before
curl https://api.openai.com/v1/chat/completions ...

# After — just change the URL and key
curl https://api.venice.ai/api/v1/chat/completions ...
```

### متغيرات البيئة

```bash
# Before
OPENAI_API_KEY=sk-...
OPENAI_BASE_URL=https://api.openai.com/v1

# After
OPENAI_API_KEY=your-venice-api-key
OPENAI_BASE_URL=https://api.venice.ai/api/v1
```

<Tip>
تقرأ العديد من المكتبات والأدوات `OPENAI_API_KEY` و`OPENAI_BASE_URL` تلقائياً. مجرد تحديث متغيرات البيئة هذه قد يكون كل ما تحتاجه.
</Tip>

## ميزات حصرية لـ Venice

تقدم Venice قدرات لا تقدمها OpenAI:

### 1. بحث ويب مدمج
```python
response = client.chat.completions.create(
    model="venice-uncensored",
    messages=[{"role": "user", "content": "Latest AI news today"}],
    extra_body={
        "venice_parameters": {
            "enable_web_search": "auto"
        }
    }
)
```

### 2. كشط ويب (Web Scraping)
```python
response = client.chat.completions.create(
    model="venice-uncensored",
    messages=[{"role": "user", "content": "Summarize https://example.com/article"}],
    extra_body={
        "venice_parameters": {
            "enable_web_scraping": True
        }
    }
)
```

### 3. Characters (شخصيات ذكاء اصطناعي)
```python
response = client.chat.completions.create(
    model="venice-uncensored",
    messages=[{"role": "user", "content": "Tell me about yourself"}],
    extra_body={
        "venice_parameters": {
            "character_slug": "venice-ai"
        }
    }
)
```

### 4. نماذج بدون رقابة
نماذج Venice الخاصة ليس لديها أي تصفية محتوى، مما يجعلها مناسبة لـ:
- الكتابة الإبداعية بدون حواجز
- أبحاث الأمن والاختبار الهجومي (red teaming)
- التحليل الصادق بدون أنماط رفض
- المعلومات الطبية/القانونية بدون إخلاء مسؤولية

### 5. توليد الفيديو
```python
# Queue a video generation job
import requests

response = requests.post(
    "https://api.venice.ai/api/v1/video/queue",
    headers={"Authorization": f"Bearer {api_key}", "Content-Type": "application/json"},
    json={
        "model": "wan-2.6-text-to-video",
        "prompt": "A serene lake at sunset with gentle waves",
        "resolution": "720p",
        "duration": 5,
    }
)
job_id = response.json()["id"]
```

## لماذا الهجرة؟

### الخصوصية
- **عدم الاحتفاظ بالبيانات** على النماذج الخاصة — لا يتم تخزين طلباتك مطلقاً
- **عدم التدريب على بياناتك** — أبداً
- تحتفظ OpenAI بالبيانات لمدة 30 يوماً وقد تستخدمها لأبحاث الأمان

### التكلفة
- النماذج الخاصة **غالباً أرخص** من مكافئاتها في OpenAI
- `qwen3-4b` بسعر $0.05/1M للإدخال أرخص بـ 10 مرات من gpt-4o-mini
- `venice-uncensored` بسعر $0.20/1M للإدخال مقابل gpt-4o بسعر $2.50/1M

### الحرية
- **لا تصفية للمحتوى** على النماذج بدون رقابة
\u0001\u0001\u0001- لا تعليق للحسابات لحالات الاستخدام المثيرة للجدل
- متوافق أصلاً مع Web3 مع خيارات الدفع بالعملات الرقمية
- رهن DIEM للحصول على أرصدة يومية

### تنوع النماذج
- الوصول إلى نماذج من مزوّدين متعددين (Qwen، Llama، Mistral، Gemma، Claude، GPT، Grok، إلخ)
- التبديل بين النماذج الخاصة والمجهولة لكل طلب
- إضافة نماذج جديدة بانتظام

## هجرة الإطارات

تعمل معظم إطارات الذكاء الاصطناعي مع Venice بتغيير عنوان URL الأساسي:

| الإطار | التغيير المطلوب |
|-----------|----------------|
| LangChain | `base_url` في `ChatOpenAI` |
| Vercel AI SDK | `baseURL` في `createOpenAI` |
| CrewAI | متغير البيئة `OPENAI_API_BASE` |
| LlamaIndex | `api_base` في `OpenAI` |
| AutoGen | `base_url` في الإعدادات |
| Haystack | `api_base_url` في `OpenAIGenerator` |
| Claude Code | علم `--api-base` أو متغير بيئة |
| Cursor | نقطة نهاية API مخصصة في الإعدادات |
| Continue.dev | `apiBase` في `config.json` |
