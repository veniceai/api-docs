واجهة Venice API محمية عبر مفاتيح API. لبدء استخدام Venice API، ستحتاج أولاً إلى إنشاء مفتاح جديد. اتبع هذه الخطوات للبدء.

<Steps>
  <Step title="زر صفحة إعدادات API">
    للوصول إلى صفحة إعدادات API، زر https://venice.ai/settings/api. يمكن الوصول إلى هذه الصفحة بالنقر على "API" في شريط الأدوات الأيمن، أو بالنقر على "API" داخل إعدادات المستخدم.

    داخل لوحة التحكم هذه، يمكنك عرض أرصدة Diem وUSD، ومستوى API، واستخدام API، ومفاتيح API.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="نظرة عامة على API" />
    </Frame>
  </Step>

  <Step title="انقر على Generate New API Key">
    مرّر للأسفل في لوحة التحكم واختر "Generate New API Key". ستظهر لك قائمة بالخيارات.

    * **Description:** تُستخدم لتسمية مفتاح API.

    * **API Key Type:**

      * مفاتيح "Admin" لديها القدرة على حذف أو إنشاء مفاتيح API إضافية برمجياً.

      * مفاتيح "Inference Only" مسموح لها فقط بتشغيل الاستدلال.

    * **Expires at:** يمكنك اختيار تعيين تاريخ انتهاء صلاحية لمفتاح API يتوقف بعده عن العمل. افتراضياً، لن يتم تعيين تاريخ، وسيعمل المفتاح بشكل دائم.

    * **Epoch Consumption Limits:** يتيح لك ذلك إنشاء حدود لاستخدام API من مفتاح API الفردي. يمكنك اختيار تحديد مقدار Diem أو USD المسموح به ضمن حقبة معينة (24 ساعة).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="إنشاء مفتاح API جديد" />
    </Frame>
  </Step>

  <Step title="أنشئ المفتاح">
    النقر على Generate سيعرض لك مفتاح API.

    <Warning>
      **مهم:** يُعرض هذا المفتاح مرة واحدة فقط. تأكد من نسخه وتخزينه في مكان آمن. إذا فقدته، فستحتاج إلى حذفه وإنشاء واحد جديد.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="مفتاح API الخاص بك" />
    </Frame>
  </Step>
</Steps>
