تتم حماية Venice API عبر مفاتيح API. لبدء استخدام Venice API، ستحتاج أولًا إلى توليد مفتاح جديد. اتبع هذه الخطوات للبدء.

<Steps>
  <Step title="زر صفحة إعدادات API">
    للوصول إلى صفحة إعدادات API، قم بزيارة https://venice.ai/settings/api. يمكن الوصول إلى هذه الصفحة بالنقر على "API" في شريط الأدوات الأيسر، أو بالنقر على "API" ضمن إعدادات المستخدم الخاصة بك.

    داخل هذه اللوحة، يمكنك عرض أرصدتك من Diem والدولار الأمريكي، ومستوى API الخاص بك، واستخدامك لـ API، ومفاتيح API الخاصة بك.

    <Frame>
      <img src="/images/guides/API-Overview.png" alt="API Overview" />
    </Frame>
  </Step>

  <Step title="انقر على Generate New API Key">
    قم بالتمرير لأسفل في اللوحة واختر "Generate New API Key". ستُعرَض عليك قائمة من الخيارات.

    * **Description:** يُستخدم لتسمية مفتاح API الخاص بك

    * **API Key Type:**

      * مفاتيح "Admin" لها القدرة على حذف أو توليد مفاتيح API إضافية برمجيًا.

      * مفاتيح "Inference Only" مسموح لها فقط بتشغيل الاستدلال.

    * **Expires at:** يمكنك اختيار تعيين تاريخ انتهاء لمفتاح API وبعده سيتوقف عن العمل. افتراضيًا، لن يتم تعيين تاريخ، وسيعمل المفتاح إلى الأبد.

    * **Epoch Consumption Limits:** يسمح لك هذا بإنشاء حدود لاستخدام API من مفتاح API الفردي. يمكنك اختيار تقييد مبلغ Diem أو الدولار الأمريكي المسموح به ضمن epoch معينة (24 ساعة).

    <Frame>
      <img src="/images/guides/api-keys/create-key.png" alt="Generate New API Key" />
    </Frame>
  </Step>

  <Step title="ولِّد المفتاح">
    سيؤدي النقر على Generate إلى إظهار مفتاح API.

    <Warning>
      **مهم:** يتم عرض هذا المفتاح مرة واحدة فقط. تأكد من نسخه وتخزينه في مكان آمن. إذا فقدته، فستحتاج إلى حذفه وإنشاء واحد جديد.
    </Warning>

    <Frame>
      <img src="/images/guides/api-keys/result.png" alt="Your API Key" />
    </Frame>
  </Step>
</Steps>
