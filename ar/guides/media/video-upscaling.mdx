يتيح لك رفع دقة الفيديو تحسين الفيديوهات الموجودة إلى دقات أعلى مع تحسين الجودة البصرية. يستخدم نموذج **Topaz Video Upscale** رفع دقة مدعومًا بالذكاء الاصطناعي لزيادة الدقة بمعامل 2x أو 4x، أو لتطبيق تحسين الجودة عند الدقة الأصلية (1x).

## كيف يعمل

يستخدم رفع دقة الفيديو نفس نظام الطوابير غير المتزامن المستخدم لتوليد الفيديو:

1. **Queue** — أرسل فيديوك إلى `/video/queue` مع النموذج `topaz-video-upscale`
2. **Poll** — استعلم من `/video/retrieve` بـ `queue_id` المُعاد حتى تصبح الحالة `completed`
3. **Complete** — استدعِ `/video/complete` لإنهاء العملية والحصول على رابط الإخراج

يكتشف الخادم تلقائيًا مدة فيديو الإدخال ومعدّل إطاراته وأبعاده من الملف المرفوع. لست بحاجة إلى تقديم هذه القيم — تُحسب الفوترة من البيانات الوصفية الفعلية للفيديو.

## عوامل رفع الدقة

| `upscale_factor` | دقة الإخراج | حالة الاستخدام |
|---|---|---|
| `1` | نفس الإدخال | تحسين الجودة فقط (إزالة التشويش، التوضيح) |
| `2` (افتراضي) | ضعف أبعاد الإدخال | رفع دقة قياسي — يصبح إدخال 720p إخراج 1440p |
| `4` | أربعة أضعاف أبعاد الإدخال | أقصى رفع دقة — يصبح إدخال 480p إخراج 1920p |

<Note>
يحل المعامل `upscale_factor` محل `resolution` لنماذج رفع الدقة. سيؤدي تمرير `resolution` إلى إعادة خطأ. ذلك لأن دقة الإخراج تعتمد على أبعاد فيديو الإدخال — رفع دقة فيديو 720p بمعامل `2x` يُنتج نتيجة مختلفة عن رفع فيديو 480p بمعامل `2x`.
</Note>

## صيغ الإدخال المدعومة

- **الصيغ**: MP4, MOV, WebM
- **طرق الإدخال**: رابط HTTPS أو رابط بيانات `data:video/...;base64,...`
- **أقصى مدة**: 300 ثانية (5 دقائق)

## استخدام الـ API

### وضع مهمة رفع دقة في الطابور

<CodeGroup>
```bash cURL
curl https://api.venice.ai/api/v1/video/queue \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "topaz-video-upscale",
    "video_url": "https://example.com/input-video.mp4",
    "upscale_factor": 2
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.venice.ai/api/v1/video/queue",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "topaz-video-upscale",
        "video_url": "https://example.com/input-video.mp4",
        "upscale_factor": 2,
    },
)

data = response.json()
queue_id = data["queue_id"]
print(f"Queued: {queue_id}")
```

```javascript Node.js
const response = await fetch("https://api.venice.ai/api/v1/video/queue", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    model: "topaz-video-upscale",
    video_url: "https://example.com/input-video.mp4",
    upscale_factor: 2,
  }),
});

const { queue_id } = await response.json();
console.log(`Queued: ${queue_id}`);
```
</CodeGroup>

تتضمن الاستجابة `queue_id` لتتبع المهمة:

```json
{
  "model": "topaz-video-upscale",
  "queue_id": "abc123-def456-..."
}
```

### الاستعلام عن الاكتمال

<CodeGroup>
```bash cURL
curl https://api.venice.ai/api/v1/video/retrieve \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"queue_id": "abc123-def456-..."}'
```

```python Python
import time

while True:
    result = requests.post(
        "https://api.venice.ai/api/v1/video/retrieve",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        json={"queue_id": queue_id},
    )
    data = result.json()

    if data.get("status") == "completed":
        print(f"Video URL: {data['url']}")
        break

    time.sleep(5)
```

```javascript Node.js
const poll = async (queueId) => {
  while (true) {
    const res = await fetch("https://api.venice.ai/api/v1/video/retrieve", {
      method: "POST",
      headers: {
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json",
      },
      body: JSON.stringify({ queue_id: queueId }),
    });
    const data = await res.json();

    if (data.status === "completed") {
      console.log(`Video URL: ${data.url}`);
      return data;
    }

    await new Promise((r) => setTimeout(r, 5000));
  }
};

await poll(queue_id);
```
</CodeGroup>

## معاملات الـ API

| الحقل | النوع | مطلوب | الوصف |
|---|---|---|---|
| `model` | string | **نعم** | يجب أن يكون `topaz-video-upscale` |
| `video_url` | string | **نعم** | رابط فيديو الإدخال أو رابط بيانات. الصيغ المدعومة: MP4, MOV, WebM. |
| `upscale_factor` | number | لا | `1` أو `2` (افتراضي) أو `4`. يتحكم في مضاعِف رفع الدقة. |

### معاملات لا تُستخدم مع نماذج رفع الدقة

المعاملات التالية **غير مقبولة** مع `topaz-video-upscale` وسترجع خطأ إذا قُدّمت:

| الحقل | السبب |
|---|---|
| `resolution` | استخدم `upscale_factor` بدلًا من ذلك. تعتمد دقة الإخراج على أبعاد الإدخال. |
| `prompt` | لا يستخدم رفع الدقة موجّهات نصية. تُضبط سلسلة فارغة تلقائيًا. |

كما يُتجاهل المعامل `duration` — يكتشف الخادم المدة مباشرة من ملف الفيديو لدقة الفوترة.

## التسعير

يعتمد التسعير على **المدة** و**مستوى دقة الإخراج** و**معدّل الإطارات**. يُحدَّد مستوى دقة الإخراج بضرب ارتفاع فيديو الإدخال في عامل رفع الدقة.

### مستويات دقة الإخراج

| المستوى | ارتفاع الإخراج | معدّل لكل ثانية |
|---|---|---|
| 720p | ≤ 720px | ~$0.013 |
| 1080p | 721–1080px | ~$0.025 |
| 4K | > 1080px | ~$0.10 |

<Note>
الفيديوهات التي يزيد معدّل إطاراتها عن 48fps تكلّف ضِعف المعدّل لكل ثانية.
</Note>

### أمثلة تسعير

| الإدخال | عامل رفع الدقة | الإخراج | المدة | التكلفة المُقدَّرة |
|---|---|---|---|---|
| 480p, 30fps | 2x | 960p (مستوى 1080p) | 10 ث | ~$0.25 |
| 720p, 30fps | 2x | 1440p (مستوى 4K) | 10 ث | ~$1.00 |
| 1080p, 30fps | 2x | 2160p (مستوى 4K) | 30 ث | ~$3.00 |
| 360p, 24fps | 4x | 1440p (مستوى 4K) | 10 ث | ~$1.00 |
| 480p, 60fps | 2x | 960p (مستوى 1080p) | 10 ث | ~$0.50 |

استخدم [Video Quote API](/ar/api-reference/endpoint/video/quote) للحصول على تسعير دقيق قبل إرسال المهمة.

### الحصول على تسعيرة

```bash
curl https://api.venice.ai/api/v1/video/quote \
  -H "Authorization: Bearer $VENICE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "topaz-video-upscale",
    "duration": "10",
    "input_height": 720
  }'
```

تقبل نقطة نهاية التسعيرة `input_height` لتقدير مستوى دقة الإخراج. هذا اختياري — في حال حذفه، تفترض التسعيرة تقديرًا متحفّظًا.

## استكشاف الأخطاء

| المشكلة | السبب المحتمل | الإصلاح |
|---|---|---|
| `"Use upscale_factor instead of resolution"` | تم تمرير `resolution` في الطلب | احذف `resolution` واستخدم `upscale_factor` بدلًا منه |
| تكلفة أعلى من المتوقَّع | فيديو الإدخال بدقة عالية أو FPS عالٍ | تحقق من أبعاد الإدخال عبر نقطة نهاية التسعيرة. الإدخال 720p+ مع رفع دقة 2x يقع في مستوى تسعير 4K. |
| المهمة تستغرق وقتًا طويلًا | فيديو كبير أو طويل | رفع الدقة كثيف الحساب. الفيديوهات الأطول وعوامل رفع الدقة الأعلى تستغرق وقتًا أطول بالتناسب. |
| `"Insufficient balance"` | أرصدة الحساب منخفضة | أضف أرصدة على [venice.ai/settings/api](https://venice.ai/settings/api) |
