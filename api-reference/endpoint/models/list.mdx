## GPT Image 2 Pricing Metadata

For `gpt-image-2` and `gpt-image-2-edit`, `model_spec.pricing.gpt_image_tokens` exposes token-aware pricing details:

- `output_image_base_usd`: base image cost by `resolution` and `quality`
- `prompt`: prompt token estimation and input token rate
- `text_output`: fixed output token allowance and rate
- `input_images`: included image count and image-input token rate for edit models
- `final_price`: cent-rounding behavior

Other image and inpaint models keep their existing static pricing fields.
