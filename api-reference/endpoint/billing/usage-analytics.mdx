Get aggregated usage analytics for the authenticated user, with breakdowns by date, model, and API key. This endpoint provides summary views of your API usage data for building dashboards and monitoring consumption.

## Query Parameters

You can specify the time period for analytics using either:
- **lookback**: A relative period like "7d" (7 days), "30d" (30 days), up to "90d" (90 days)
- **startDate** and **endDate**: A custom date range in `YYYY-MM-DD` format

If no parameters are specified, the default lookback period is 7 days.

## Response Fields

### byDate
Daily usage totals showing total consumption in both USD and DIEM for each day in the period.

- **date**: The date in `YYYY-MM-DD` format
- **totalUsd**: Total usage in USD for that day
- **totalDiem**: Total usage in DIEM for that day

### byModel
Usage breakdown by model, showing aggregate consumption per model.

- **model**: The model identifier (e.g., "llama-3.3-70b")
- **totalUsd**: Total USD spent on this model
- **totalDiem**: Total DIEM spent on this model
- **promptTokens**: Total prompt tokens consumed
- **completionTokens**: Total completion tokens generated
- **totalTokens**: Combined total tokens

### byKey
Usage breakdown by API key, showing consumption per key.

- **keyId**: The API key identifier
- **keyName**: The display name of the API key
- **totalUsd**: Total USD spent via this key
- **totalDiem**: Total DIEM spent via this key

### byModelDaily
Time-series data for charting usage by model over time.

- **date**: The date in `YYYY-MM-DD` format
- **model**: The model identifier
- **totalUsd**: USD spent on this model for this day
- **totalDiem**: DIEM spent on this model for this day

### byKeyDaily
Time-series data for charting usage by API key over time.

- **date**: The date in `YYYY-MM-DD` format
- **keyId**: The API key identifier
- **keyName**: The display name of the API key
- **totalUsd**: USD spent via this key for this day
- **totalDiem**: DIEM spent via this key for this day

## Example Usage

```bash
# Get usage analytics for the past 7 days (default)
curl -X GET "https://api.venice.ai/api/v1/billing/usage-analytics" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Get usage analytics for the past 30 days
curl -X GET "https://api.venice.ai/api/v1/billing/usage-analytics?lookback=30d" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Get usage analytics for a specific date range
curl -X GET "https://api.venice.ai/api/v1/billing/usage-analytics?startDate=2026-02-01&endDate=2026-02-28" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
