Get current balance information for the authenticated user. Returns remaining DIEM/USD balances and total DIEM epoch allocation for calculating usage percentage.

## Response Fields

- **canConsume**: Whether the user has sufficient balance to make API requests
- **consumptionCurrency**: The currency that will be used for consumption (`DIEM`, `USD`, or `VCU`)
- **balances.diem**: Remaining DIEM balance for current epoch (null if not staking)
- **balances.usd**: Remaining USD balance (null if not available)
- **diemEpochAllocation**: Total DIEM allocation for the current epoch from staking

## Calculating Usage Percentage

To calculate your DIEM usage percentage for the current epoch:

```
usagePercentage = ((diemEpochAllocation - balances.diem) / diemEpochAllocation) * 100
```
