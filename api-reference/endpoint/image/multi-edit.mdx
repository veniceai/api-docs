<Info>
**Dynamic pricing:** Some multi-edit models (currently `gpt-image-2-edit`) use
request-based pricing. Pass the `quality` parameter (`low`, `medium`, `high`) to control
the fidelity/cost tradeoff. Final charges vary by quality, output resolution, prompt
length, and input image count (1–3). See the [Pricing overview](/overview/pricing) for
charge ranges.
</Info>
