## GPT Image 2 Edit Pricing

`gpt-image-2-edit` uses token-aware dynamic pricing for multi-edit requests. The final price depends on:

- `quality`: `low`, `medium`, or `high`
- `resolution`: `1K`, `2K`, or `4K`
- prompt length, estimated as `ceil(prompt characters / 3.2)` input tokens
- input images: the first input image is included; each additional input image adds image-input token cost

The final USD price is rounded up to the next cent after all billed components are summed. For exact formula fields, call the [Models endpoint](/api-reference/endpoint/models/list) and inspect `model_spec.pricing.gpt_image_tokens` for `gpt-image-2-edit`.
